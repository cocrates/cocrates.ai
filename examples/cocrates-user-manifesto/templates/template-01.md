# Template 01: Cover

## Derivation
- **Base:** none
- **Relationship:** standalone

## Background
- **Color:** #0a0a0a (near black), full-bleed

## Typography
- **Font family:** 굵은 고딕 계열 sans-serif (Noto Sans KR Bold, Pretendard Bold 등)
- **Title:** 매우 큼(48~60pt), 굵기 800~900, 흰색(#ffffff), 중앙 정렬, 자간 약간 좁게
- **Subtitle:** 중간(20~24pt), 굵기 400, 회색(#888888), 중앙 정렬
- **Closing quote:** 작게(14~16pt), 굵기 300, 옅은 회색(#666666), 중앙 정렬

## Color Palette
- **Background:** #0a0a0a
- **Primary text:** #ffffff
- **Secondary text:** #888888
- **Tertiary text:** #666666
- **Accent:** #ffffff (장식 최소화, 텍스트 자체가 디자인)
- **Divider line:** #333333

## Regions
### Title zone
- **Position:** center (full slide, vertical center)
- **Style:** title + subtitle 수직 배열. 둘 사이에 가는 구분선(#333333) 선택적
- **Max lines:** title 1~2줄, subtitle 1~2줄

### Quote zone *(optional)*
- **Position:** 하단, 여백
- **Use:** 마무리 인용문 — 작고 옅은 텍스트

## Supported content layouts
focal

## Layout Defaults
- **Content alignment:** center
- **Whitespace:** generous (넉넉한 여백)

## Intended Use
cover — opening slide only (Slide 01)
