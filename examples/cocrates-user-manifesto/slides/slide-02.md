**Section:** 도입
**Type:** 개념 설명
**Template:** template-02
**Content layout:** focal + stack
**Density:** normal
**Time:** ~2분

### Visual Design
- **Background:** #0a0a0a (template-02)
- **Layout:** 중앙 정렬 — 상-중-하 3단, 정의, 본질, 핵심 모두 중앙
  - 상단 (타이틀): "Cocrates란 무엇인가?" — 중앙 정렬
  - 중앙: Co + Socrates 해체 + 본질 설명 (2줄 내외) — 중앙 정렬
  - 하단: 핵심 메시지 — 굵고 선명하게, 중앙 정렬

### Content Elements
| 요소 | 내용 | 위치 / 비고 |
|------|------|------------|
| 타이틀 | Cocrates란 무엇인가? | 상단 중앙, 28~36pt, 굵기 800, #ffffff |

### Visual Elements Description
- 여백을 넉넉히 두어 각 요소가 숨 쉴 공간을 확보
- 'Co + Socrates'는 마치 브랜드 로고처럼 해체하여 표시 (예: Co | Socrates)
- 하단 핵심 문장은 인용 또는 박스 형태로 강조

### Transitions
- **In:** 타이틀 → 정의 → 본질 → 핵심 순차 등장
- **Out:** 핵심 문장을 남기고 fade
