# Slide Content Specification — Slide 03

## Slide 03: 문제 제기 — 도구만으로는 부족하다

**Section:** 도입
**Type:** contrast / 문제 제기
**Template:** template-02
**Content layout:** focal (대비 구조: 인정 → 전환 → 경고)
**Density:** normal
**Time:** ~2분

### Visual Design
- **Background:** #0a0a0a (template-02)
- **Layout:** 중앙 정렬 — 수직 3단, 인정, 반전, 경고로 이어지는 드라마
  - 상단 (타이틀): "도구만으로는 부족하다" — 중앙 정렬
  - 중앙 상단: "Cocrates는 훌륭한 도구입니다." (보통 톤) — 중앙 정렬
  - 중앙 (전환점): **"하지만..."** (압도적으로 크게) — 중앙 정렬
  - 중앙 하단: 경고 메시지 (빨간 악센트) — 중앙 정렬
  - 하단: 여백 — 경고의 여운

### Content Elements
| 요소 | 내용 | 위치 / 비고 |
|------|------|------------|
| 타이틀 | 도구만으로는 부족하다 | 상단 중앙, 28~36pt, 굵기 800, #ffffff |
| 인정 | "Cocrates는 훌륭한 도구입니다." | 중앙, 18pt, #cccccc |
| _(여백)_ | | 짧은 멈춤 |
| 전환 | **하지만...** | 중앙, 48~60pt, 굵기 900, #ffffff |
| 경고 | 도구가 아무리 좋아도, 제대로 사용하지 않으면 아무 소용없습니다. | 중앙, 16pt, #cccccc |
| 경고 핵심 | **당신은 여전히 AI의 노예입니다.** | 중앙, 22pt, 굵기 800, #ff3b30 |

### Visual Elements Description
- "하지만..."은 슬라이드에서 가장 큰 텍스트. 보통 텍스트에서 갑자기 큰 텍스트로 전환되면서 극적 반전 연출
- 마지막 경고문은 빨간색(#ff3b30), 이 슬라이드에서 유일한 컬러 포인트
- 전체가 마치 한 편의 미니 드라마처럼 흘러감

### Transitions
- **In:** 상단부터 순차적으로. "하지만..."은 잠시 멈춤 후 등장 (드라마틱하게)
- **Out:** 경고문을 화면에 남기고 서서히 fade
