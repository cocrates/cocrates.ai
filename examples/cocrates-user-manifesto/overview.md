# Presentation Definition — Cocrates User Manifesto

## Purpose
Cocrates User Manifesto의 7가지 계명을 **처음 접하는 개발자**에게 소개한다. AI 도구(Cocrates Harness)와 함께 일할 때 인간 아키텍트가 지켜야 할 원칙을 선언문 형식으로 전달하여, 단순한 도구 사용법을 넘어 **AI와 협업하는 자신만의 철학**을 정립하도록 돕는다.

구체적으로 다음을 다룬다:
1. **Cocrates의 3대 철학** — The unexamined code / I know nothing / Harness Ignorance
2. **사용자 선언 7계명** — 설계와 검증의 원칙부터 인지적 투쟁과 열린 공유까지
3. **도구(Harness)와 의지(Manifesto)의 결합** — 구조적 제어 도구 + 인간 의지 = 무한한 능력의 확장

## Audience
- **Background:** 소프트웨어 개발자(초급~중급), AI 도구(Copilot, ChatGPT 등)를 업무에 사용 중인 개발자
- **Knowledge level:** Cocrates를 처음 접함. AI 도구 사용 경험은 있으나, 체계적인 원칙 없이 사용 중
- **Interests/expectations:** AI 도구를 더 효과적으로 활용하는 방법, 단순한 꿀팁이 아닌 **원칙과 철학**을 기대

## Format & Duration
- **Time limit:** 20분
- **Slide count:** 13장 (~1분 30초/슬라이드)
- **Format:** 발표 세션 (in-person or online)
- **Tone/style:** 선언적이고 철학적인 어조. 다소 엄숙하지만 개발자의 공감을 이끌어내는 직설적인 화법.

## Visual Style
- **선 굵은 미니멀 스타일** — 굵은 타이포그래피, 넉넉한 여백, 장식 최소화
- **단색 위주의 강한 대비** — 흑백 또는 단일 악센트 컬러
- **무게감 있는 느낌** — Cocrates의 엄격한 철학을 시각적으로 반영
- **아이콘/도형은 최소화** — 텍스트 자체가 디자인 요소가 되도록

## Narrative Goals
1. 청중이 7계명 중 최소 3개 이상을 기억하고 설명할 수 있다
2. 청중이 "AI를 대하는 나의 원칙은 무엇인가?"라는 질문을 스스로에게 던지게 된다
3. 청중이 Cocrates에 대한 호기심을 갖고 실제로 사용해보고 싶게 만든다

## Scope / Out of Scope
- **In scope:** Cocrates User Manifesto의 7계명 상세 설명, 철학적 배경, 실천 방법
- **Out of scope:** Cocrates Harness의 설치/사용법 튜토리얼, 특정 기술 스택 가이드, 에피소드 시리즈와의 연결

## Validation Criteria
1. 각 계명 슬라이드가 하나의 명확한 메시지를 전달하는가?
2. 처음 접하는 개발자가 계명의 의미와 실천 방법을 이해할 수 있는가?
3. 전체 내러티브가 20분에 적합한 흐름인가?
4. 선 굵은 미니멀 스타일이 일관되게 적용되었는가?

## Constraints
- 20분 내외 (리허설 후 20분에 맞게 조정 권장)
- 스크립트 없이 슬라이드 자체로 메시지가 전달되어야 함 (on-slide copy가 충분해야 함)
- 각 계명은 영문 원문을 메인으로, 한국어 해설을 부제로 함께 제시
