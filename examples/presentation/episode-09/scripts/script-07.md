# Script: Slide 07 — ⚠️ 무지(Ignorance)의 기록 — 오개념 박제의 힘

**Time:** ~1분

**[SLIDE 07: Wrong Assumptions / Gaps 섹션 예시 — 전/후 비교 / Broken Expectation 이미지]**

Knowledge Capture의 꽃, `Wrong Assumptions / Gaps` 섹션을 더 깊이 들여다보겠습니다.

이 스킬의 철학은 이것입니다. "무엇을 새로 배웠는지"보다 **"어떤 오개념을 가지고 있었고 어떻게 틀렸었는지"** 를 박제하는 것이 장기 기억으로 가는 가장 강력한 도구다.

블룸 분류학을 예로 들어볼까요?

(슬라이드의 KB 예시를 가리키며)

```markdown
## Wrong Assumptions / Gaps
- (이전 오해) 블룸 분류학 = 기억→이해→...→창조의 순차적 피라미드
- (깨달음) 사실은 2차원 매트릭스 — 지식 차원 × 인지 과정의 교차점
```

왜 이렇게 하는 게 효과적일까요? 인지심리학에 따르면, 뇌는 새로운 정보보다 **깨진 예상, Broken Expectation**을 더 오래 기억합니다. "내가 틀렸다"는 사실이 기억을 강화하는 겁니다.

"아는 것"을 기록하는 KB는 흔합니다. 하지만 **"틀렸던 것"을 기록하는 KB, 이것이 Cocrates만의 차별점입니다.**
