# Slide 03: 🧗‍♂️ Education Skill — No Spoon-feeding과 Turn-based Mission

## Key Message
Education 스킬의 두 핵심 규칙 — No Spoon-feeding(제한)과 Turn-based Mission(생성) — 이 Cocrates 교육 대화의 근간이다.

## Content
### Headlines / text
- **규칙 1 — No Spoon-feeding (최상위 헌법):**
  - "절대 한 턴에 완전한 정답이나 전체 해결책을 떠먹여 주지 않는다"
  - 개념 설명은 최대 **1~3문장, 응답의 20% 이하**로 제한
  - **Incomplete State 원칙:** 대화를 항상 불완전한 상태로 멈춤 → 사용자가 채워야 함
- **규칙 2 — Turn-based Mission:**
  - 모든 응답은 정확히 **하나의 `🔥 [MISSION]` 블록**으로 끝나야 함
  - 사용자가 현재 MISSION에 답하기 전에 절대 다음 힌트를 스포일러하지 않음
  - MISSION은 단순 사실 확인이 아닌 **인지적 과제**여야 함
- **두 규칙의 관계:** No Spoon-feeding은 '제한', Turn-based Mission은 '생성' — **하나의 엔진**

### Visual elements
- 두 규칙을 좌우 2열 카드 배열
- 규칙 1: 🚫 숟가락 아이콘 (No Spoon-feeding)
- 규칙 2: 🎯 타겟 아이콘 (Mission)
- 하단: 두 규칙이 하나로 합쳐지는 이미지 또는 연결선

## Layout Notes
- 2열 배열로 시각적 대비
- 각 규칙 아래에 핵심 원칙을 짧은 글머리로
- 하단에 관계 설명 강조 박스
