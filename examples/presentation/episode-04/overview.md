# S1: Episode-04 발표 정의

## 제목

**소크라테스식 학습 활동 — 실습**

부제: "블룸의 분류학에 대해 알려줘"라고 물었더니 일어난 일

## 목적

시청자는 Cocrates의 **Learning 파이프라인 3단계**를 실제 대화 스크립트를 통해 생생하게 경험한다. 이 과정에서 자연스럽게 **블룸의 교육분류학(Bloom's Taxonomy)** 에 대한 개념적 이해를 얻게 된다.

## 청중

- **블룸 분류학에 대한 사전 지식:** 대부분 모름 (강사는 알고 있음)
- Ep3까지 시청한 개발자 (Cocrates Harness가 설치된 상태)
- AI에게 "블룸의 분류학에 대해 알려줘" 같은 질문을 해본 경험이 있는 사람
- AI가 단순히 정보를 제공하는 것에 익숙한 사람

## 예상 시간

약 20분 (12슬라이드 × ~1.5분)

## 핵심 메시지

> Cocrates는 답을 주는 자판기가 아니다. 미션과 Pull 전략으로 생각을 유도하고, 무지를 기록하게 하며, 진짜 이해를 확인한다.
>
> 블룸 분류학은 단순한 6단계 피라미드가 아니다. "내가 안다"는 것의 진짜 의미를 알면, 자신의 학습 수준을 진단하고 다음에 무엇을 공부할지 알 수 있다.
>
> 모르는 것을 무시하지 않고, AI에게 물어보는 것 — 그것이 "무지를 방치하지 마라"는 행동 강령의 실천이다.

## 성공 기준

시청자가 이 발표를 마친 후:
1. **블룸 분류학의 기본 개념**을 이해한다 — 6단계 피라미드, 2차원 매트릭스(Knowledge × Cognitive), Push/Pull 전략
2. **자신의 지식 수준**이 블룸 분류학의 어느 단계에 있는지 스스로 진단할 수 있다
3. Cocrates의 **Learning 파이프라인 3단계**(Education → Knowledge Capture → Reflection)를 설명할 수 있다
4. 각 단계에서 Cocrates가 구체적으로 어떻게 행동하는지 예시를 들 수 있다

## 에피소드 맥락

### 시리즈 내 위치

```

                 Ep2 (원칙 선언)
Part 2. 실습 ── Ep3 (설치) ← 지난 편
            ▶  Ep4 (소크라테스식 학습) ◀ 현재
                 Ep5 (구조 기반 생성)
                 Ep6 (워크플로우 생성)
Part 3. 구조 ── Ep7~12 (원리와 스킬)
Part 4. 결론 ── Ep13~14 (선언)
```

### 이전 에피소드와의 연결

- **Ep2 (원칙):** "무지를 방치하지 마라" — 이 에피소드는 그 원칙의 **실천**이다. 블룸 분류학을 모르면 AI에게 물어보고, 배운 내용을 검토(U→A→E→A)한다.
- **Ep3 (설치):** Cocrates가 설치된 환경에서 첫 번째 질문(블룸의 분류학)을 던지는 순간부터 시작

## 슬라이드 구성 (초안)

총 12슬라이드, 도입(2) + 본론(8) + 결론(2) 구조

### 도입 (~4분)
1. **타이틀 페이지** — Ep3 연결 (설치 완료! 이제 첫 질문)
2. **"블룸 분류학을 아시나요?"** — 모르면 AI에게 물어보는 것이 "무지를 방치하지 마라"의 실천. 학습 목표 4가지.

### 본론 — 3단계 파이프라인 경험 (~13분)
3. **시작: "블룸의 분류학에 대해 알려줘"** — 일반 AI vs Cocrates 응답 비교 + 블룸 분류학 6단계 피라미드 기본 설명
4. **🎯 미션 ① — 피라미드는 진도 순서가 아니다** (Pull 전략의 발견)
5. **🎯 미션 ② — 2차원 매트릭스** (Knowledge Dimension × Cognitive Process)
6. **🎯 미션 ③ — Push와 Pull의 밀당** (학습자 상태에 따른 전략 전환)
7. **🎯 파이널 미션 — Git 브랜치 전략 4시간 커리큘럼** (Create 수준)
8. **Knowledge Capture** — 요약이 아니라 '무지(Gap)'를 기록하다
9. **Reflection ①** — 면접관 모드: 2차원 매트릭스에 커리큘럼 매핑
10. **Reflection ②** — 명확한 성적표: ✅ 확실한 영역 vs ⚠️ 삐걱거리는 영역

### 결론 (~3분)
11. **핵심 요약 + 학습 확인** — 4-point 요약 + 자기 진단 질문
12. **다음 에피소드 연결** — Ep5: 구조 기반 산출물 생성 (코드 실습 예고)
