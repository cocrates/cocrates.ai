# Slide 11: 학습 목표 확인

## Key Message
시청자가 스스로 검토(U→A→E→A)를 실천하고 있는지 돌아보게 하는 소크라테스식 질문과 미션을 제시한다.

## Content
- **Headline:** 스스로에게 질문하라
- **질문 1:** "오늘 AI에게 받은 결과물 중, U→A→E→A를 모두 거친 것은 몇 개인가?"
- **질문 2:** "지금 이 결과물을, 내 말로 설명할 수 있는가?"
- **미션:** 🎯 영상을 멈추고, 가장 최근에 AI에게 받은 결과물을 U→A→E→A로 검토해보라
- **Visual Elements:**
  - 큰 물음표(?) 아이콘 2개
  - 질문 아래에 충분한 여백 (시청자가 생각할 시간)
  - 미션 영역은 다른 배경색으로 구분

## Layout Notes
- 질문과 미션을 명확히 구분 (질문 2개 → 미션)
- 여백을 넉넉히 주어 시청자가 생각할 시간을 시각적으로 암시
- 미션 영역은 강조 배경색 사용
