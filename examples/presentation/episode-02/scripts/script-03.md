# Script: Slide 03 — 선언 — "The Unexamined Code Is Not Worth Generating"

**Time:** ~1분

**[SLIDE 03: 화면 중앙에 선언문이 가장 크게, 격언/모토처럼 배치됨. 윗줄 "AI-native Engineer의 핵심 역량 = 검토". 중앙에 큰 따옴표와 함께 "The unexamined code is not worth generating." 아래에 한국어 번역]**

Ep1에서 우리는 질문을 던졌습니다.
"AI-native Engineer의 핵심 역량은 무엇인가?"

(잠시 멈추고, 단호하게)

그 질문에 대한 답은 바로 이것입니다.

**검토.**

그리고 이 답을 한 문장으로 선언합니다.
이 시리즈를 관통하는 단 하나의 원칙.

(천천히, 강조하며 읽는다)

**"The unexamined code is not worth generating."**
**"검토되지 않은 산출물은 생성할 가치가 없다."**

(청중을 바라보며)

AI에게 결과물을 받았을 때,
그냥 "아, 괜찮네" 하고 넘어가는 순간,
당신은 AI-assisted의 함정에 빠지고 있는 겁니다.

AI-native Engineer는 검토합니다.
이해하고, 분석하고, 평가하고, 승인합니다.
검토되지 않은 코드는 그냥 생성되지 않은 것과 같습니다.

→ [SLIDE 04: 소크라테스의 영감]
