# Slide 08: 🎓 Core Activity 2 — 학습 파이프라인

## Key Message
3단계 파이프라인: Education(질문-미션) → Knowledge Capture(핵심 기록) → Reflection(이해도 평가).

## Content
- **파이프라인:** Education → Knowledge Capture → Reflection
- **각 단계 역할:**
  - **Education:** 소크라테스식 질문-미션 기반 학습 ("직접 생각하게 한다")
  - **Knowledge Capture:** 배운 내용을 KB에 핵심과 Gap 중심으로 기록
  - **Reflection:** 실제 이해도를 평가 — ✅ 잘 아는 것 vs ⚠️ 제대로 모르는 것
- **연결:** Ep4 Bloom's taxonomy 미션 4개 — Mission 1~4, ✅ vs ⚠️ 평가
