# Script: Slide 08 — 🎓 Core Activity 2 — 학습 파이프라인

**Time:** ~1분

**[SLIDE 08: 파이프라인 흐름도 — Education → Knowledge Capture → Reflection. 각 단계 아이콘]**

두 번째 핵심 활동은 **학습**입니다. 이 파이프라인은 세 단계로 구성됩니다: Education, Knowledge Capture, Reflection.

**Education**은 소크라테스식 질문과 미션을 기반으로 한 학습입니다. 단순히 정보를 전달하지 않고, "직접 생각하게 합니다." Ep4에서 블룸 분류학을 배울 때 Cocrates가 정답을 바로 알려주지 않고 미션을 통해 스스로 깨닫게 했던 방식, 기억하시나요?

**Knowledge Capture**는 배운 내용을 KB, 지식 베이스에 핵심과 Gap 중심으로 기록합니다. "내가 틀렸던 가정"을 기록하는 것이 핵심입니다.

**Reflection**은 실제 이해도를 평가합니다. 단순 암기 확인이 아니라, ✅ 잘 알고 있는 것과 ⚠️ 제대로 알지 못했던 것을 구분해줍니다.

이 파이프라인을 담당하는 스킬들은 `education`, `knowledge-capture`, `reflection`입니다.

지금까지 우리는 Cocrates가 Agent + Skills 구조로 되어 있고, Agent는 6개 섹션, 그중에서도 Intent-To-Skill Routing이 핵심이며, 두 가지 Core Activity 파이프라인이 있다는 것을 배웠습니다.

그런데 이것으로 끝이 아닙니다. 이 구조를 이해하는 것의 진정한 의미를 다음 슬라이드에서 말씀드리겠습니다.
