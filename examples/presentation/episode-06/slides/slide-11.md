# Slide 11: 🔄 "보고서 써줘" 한 마디의 의미가 바뀌었다

## Key Message
스킬이 만들어지면 사용자의 역할이 '수동적 수용자'에서 '능동적 설계자'로 바뀐다. 같은 요청이 단계별 협업 프로세스로 변화한다.

## Content
- **스킬 전:** "블록체인 시장 조사 보고서 써줘"
  - AI가 웹 검색 내용을 짜깁기한 글을 던짐
  - 품질을 예측할 수 없음, 매번 새로운 검증 필요
- **스킬 후:** "블록체인 시장 조사 보고서 써줘"
  - 스킬의 룰에 따라 단계별로 진행
  - **[P1]** "타겟 독자와 목적부터 정하시죠. outline.md를 승인해 주세요."
  - **[P2]** "기술 분석 위주니 '비교/분석' 포맷이 좋겠습니다. 동의하십니까?"
  - **[P3]** "1번 섹션을 작성했습니다. 검토 의견을 주세요."
- **핵심:** 사용자 역할 변화 — 수동적 수용자 → 능동적 설계자

## Visual Elements
- Top: "Before" (with ❌ icon) — single chaotic output
- Bottom: "After" (with ✅ icon) — P1→P2→P3 structured flow
- Role change highlight box at bottom: "🙇 수동적 수용자 → 👑 능동적 설계자"

## Layout Notes

- Before side: messy, single box
