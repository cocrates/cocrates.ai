# Slide 13: 핵심 요약 + 학습 확인

## Key Message
4가지 핵심 포인트 요약과 스스로 이해를 확인하는 자기 질문 3가지.

## Content
- **4-Point 핵심 요약:**
  1. 🔄 스킬 = AI의 일하는 방식을 설계하는 것. 일회성 요청을 넘어 SOP를 정의. 한 번 만들면 영원히 재사용.
  2. 🔍 AI의 제안을 검토 없이 수용하지 마라. AI는 가장 쉽고 일반적인 결정을 내린다. 반드시 U→A→J→A 단계를 거쳐야 한다.
  3. ❄️ Snowflake는 공통 패턴이다. 학습, 생성, 스킬 모든 활동에 동일한 방법론이 적용된다.
  4. 👑 스킬이 만들어지면 사용자는 능동적 설계자가 된다.
- **학습 확인 (자기 질문):**
  - ❓ 일회성 요청과 스킬의 차이를 설명할 수 있는가?
  - ❓ 스킬 생성 파이프라인 6단계를 말할 수 있는가?
  - ❓ Snowflake 점진적 확장이 Ep4, Ep5, Ep6에 어떻게 각각 적용되는지 설명할 수 있는가?

## Visual Elements
- Top: 4-point summary in a clean numbered list
- Bottom: 3 self-check questions with ❓ icons
- Divider line between summary and questions

## Layout Notes
