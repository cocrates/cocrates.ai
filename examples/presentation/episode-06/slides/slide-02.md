# Slide 02: 학습 목표

## Key Message
오늘의 3가지 목표: 스킬 vs 일회성 요청 이해, 6단계 파이프라인 설명, Snowflake 실전 적용.

## Content
- **도입 질문:** "여러분, AI에게 같은 부탁을 여러 번 해보신 적 있나요? '보고서 써줘', '설명문 작성해줘' — 매번 비슷한 결과물을 받으면서도 만족스럽지 않았던 경험?"
- **핵심 전제:** 일회성 요청을 넘어, AI의 일하는 방식 자체를 설계할 수 있다
- **3가지 학습 목표:**
  1. 🔄 스킬(Skill)과 일회성 요청의 차이를 이해하고, 스킬이 왜 지속 가능한 자산인지 설명할 수 있다
  2. 🔧 스킬 생성 파이프라인 6단계(Kernel → Frame → Outline → Spec → Skill → Verification)를 설명할 수 있다
  3. ❄️ Snowflake 점진적 확장의 실전 적용 방법을 이해한다

## Visual Elements
- Three numbered items with icons (🔄, 🔧, ❄️)
- 도입 질문 in a speech bubble or highlighted box
- Bottom accent line or subtle progress indicator

## Layout Notes
- 도입 질문 at top in a callout box
- Three learning objectives listed with equal spacing
- Consistent with Ep2-Ep5 learning objectives slide template
