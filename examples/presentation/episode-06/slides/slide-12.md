# Slide 12: ❄️ Snowflake 점진적 확장의 힘

## Key Message
Snowflake는 Cocrates의 공통 패턴이다. 작게 시작해서 점진적으로 확장하는 방법론이 학습(Ep4), 생성(Ep5), 스킬(Ep6) 모든 활동에 적용된다.

## Content
- **Snowflake 원칙 (4가지):**
  1. 작게 시작한다 (Kernel 한 문장)
  2. 단계별로 확장한다 (Frame → Outline → Spec)
  3. 각 단계에서 검토와 승인을 받는다
  4. 중간 산출물을 파일로 저장한다 (채팅창 유실 방지)
- **Ep4-5-6의 공통 방법론:**
  - Ep4 (학습): Education → Knowledge Capture → Reflection
  - Ep5 (생성): ADR → Spec → Generation → Verification
  - Ep6 (스킬): Kernel → Frame → Outline → Spec → Skill → Verification
  - 모두 Snowflake: 작게 시작해서 점진적으로 확장
- **포인트:** 배움, 생성, 스킬 — 모든 활동에 동일한 방법론이 적용된다

## Visual Elements
- Top: 4 Snowflake principles in a grid
- Bottom: Three columns comparing Ep4, Ep5, Ep6 pipelines
- Each pipeline as a horizontal flow with arrows
- ❄️ Snowflake icon at top center
