# Script: Slide 11 — 🔄 "보고서 써줘" 한 마디의 의미가 바뀌었다

**Time:** ~1.5분

**[SLIDE 11: Before/After 비교 — 전: AI가 짜깁기한 글 / 후: 스킬의 P1→P2→P3 단계별 진행]**

이제 스킬이 완성되었습니다. 같은 요청이라도 전과 후가 어떻게 달라지는지 볼까요?

스킬이 만들어지기 전, "블록체인 시장 조사 보고서 써줘"라고 요청하면 어떻게 됐을까요? AI가 웹 검색 내용을 짜깁기한 글을 던져줍니다. 품질을 예측할 수 없고, 매번 새로운 검증이 필요합니다. 수동적 수용자가 되는 거죠.

그런데 스킬이 만들어진 후에는 완전히 달라집니다. 같은 요청을 해도, AI는 스킬의 룰에 따라 단계별로 진행합니다.

P1 단계: "타겟 독자와 목적부터 정하시죠. outline.md를 승인해 주세요."

그다음 P2: "기술 분석 위주니 '비교/분석' 포맷이 좋겠습니다. 동의하십니까?"

그다음 P3: "1번 섹션을 작성했습니다. 검토 의견을 주세요."

이것이 의미하는 바는 무엇일까요? **사용자의 역할이 근본적으로 바뀌었습니다.** 더 이상 AI가 주는 결과물을 수동적으로 기다리는 '수동적 수용자'가 아닙니다. 스스로 설계하고 통제하는 **'능동적 설계자'**가 된 것입니다.
