**VS Code + OpenChamber 확장**을 설치하면

**첫째,** 아직 VS Code가 없다면
code.visualstudio.com에 접속해서 다운로드하고 설치하세요.
GUI 설치 프로그램이라 Next만 누르면 끝입니다.

**둘째,** VS Code를 실행하고,
왼쪽 사이드바의 확장 아이콘(Ctrl+Shift+X)을 클릭한 다음
검색창에 **"OpenChamber"**를 입력하고 설치 버튼을 누르면 됩니다.

OpenChamber를 이용하면, 마치 VS Code의 Copilot이나 Cursor를 쓰는 것처럼,
VS Code 안에서 Cocrates를 사용할 수 있습니다.
