# Script: Slide 03 — 사용자의 진화 — 대화의 내면화

**Time:** ~1분

**[SLIDE 03: 좌측 — Cocrates 말풍선 / 우측 — 3가지 질문 카드 / 하단 — 개인 트레이너 아이콘]**

Cocrates를 처음 사용할 때, 많은 분들이 이런 생각을 하셨을 겁니다.

"아, 그냥 답만 주면 안 되나? 왜 자꾸 질문을 던지지?"

(청중을 보며, 공감하는 톤으로)

저도 처음엔 그랬습니다. 그런데 이 질문들은 단순한 대화 기법이 아닙니다.

이것은 파트너의 손짓입니다. 주도권을 여러분께 돌려드리기 위한, Cocrates만의 방식입니다.

(슬라이드 우측을 가리키며)

시간이 지나면, Cocrates가 외부에서 던지던 질문들이 어느 순간 — 여러분 자신의 안에서 울리기 시작합니다.

"이 결정의 트레이드오프는 무엇인가?"
"내가 놓치고 있는 제약 조건은 없는가?"
"이 구조를 타인에게 설명할 수 있는가?"

개인 트레이너와 운동하는 것을 상상해보세요. 처음에는 트레이너가 자세를 하나하나 교정해줍니다. 그런데 언젠가 트레이너가 없어도 — 올바른 자세로 운동할 수 있게 됩니다.

그게 바로 내면화입니다.

Cocrates의 궁극적 목표는 바로 이것입니다. 여러분이 스스로 질문할 수 있게 되는 것.
