# Slide Content Specification — Slide 9

## Slide 9: 3줄 요약 — 그리고 스스로에게 던지는 질문

**Section:** 결론
**Type:** 요약 + 참여 유도
**Time:** ~1분 30초

### Visual Design
- **Background:** 시리즈 공통 템플릿 (밝은 톤)
- **Layout:** 상/하 분할
  - 상단 (45%): 3줄 요약
  - 하단 (55%): 4개의 질문 말풍선

### Visual Elements Description
- 3줄 요약: 가로로 3개의 카드 배열 또는 세로 리스트
- 각 요약에 일관된 아이콘 + 굵은 키워드 + 설명문 구조
- 질문 말풍선: 4개를 2x2 격자 배열, 각각 다른 색상의 말풍선
- "정답이 아닌, 사고를 위한 질문"이라는 안내문이 질문 영역 상단에 배치

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 발표자가 질문을 하나씩 읽어줄 때 해당 말풍선이 강조되는 효과. 각 질문 후 잠시 멈춤(청중의 사고 시간).
