# Slide Content Specification — Slide 8

## Slide 8: 지속적 개선의 원칙 — The Unexamined Code

**Section:** 본론 3 — 오픈소스와 원칙
**Type:** 개념 설명 + 핵심 메시지 집약
**Time:** ~1분 30초

### Visual Design
- **Background:** 다소 어두운 톤 — 엄숙하지만 격려하는 분위기
- **Layout:** 3단 수직 분할
  - 상단: "The unexamined code" 인용문 강조
  - 중단: 3가지 원칙 카드
  - 하단: 핵심 메시지 요약 상자

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 📜 지속적 개선의 원칙 — **The Unexamined Code** | 상단 |
| 인용문 | **"The unexamined code is not worth generating."** — 검토되지 않은 산출물은 생성할 가치가 없다 | 중앙, 대형 인용문, 시리즈 전체 상징 |

| 포인트 | 완벽한 첫 시도는 없다. Cocrates Harness를 개선하고 진화시키는 과정 그 자체가 목적이다. Cocrates는 완성품이 아니라 **우리 팀과 함께 진화해 나가는 열린 설계도**다. | 최하단 |

### Visual Elements Description
- 인용문: 큰 따옴표 그래픽과 함께 대형 폰트로 표시
- 3원칙 카드: 동일한 크기와 스타일의 카드 3개를 가로로 배열
- 핵심 메시지 상자: 배경색이 다른 영역(골드/오렌지 계열), 두꺼운 테두리

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 인용문이 먼저 등장하고, 잠시 멈춘 후 나머지 내용이 나타나는 순차적 애니메이션. 발표자가 핵심 메시지를 전달할 때 슬라이드의 핵심 메시지 상자를 가리키며 강조.
