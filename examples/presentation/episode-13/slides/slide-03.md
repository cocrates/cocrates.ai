# Slide Content Specification — Slide 3

## Slide 3: 사용자의 진화 — 대화의 내면화

**Section:** 본론 1 — 사용자의 진화
**Type:** 개념 설명
**Time:** ~1분

### Visual Design
- **Background:** 시리즈 공통 템플릿
- **Layout:** 하프 앤 하프 (좌/우 분할)
  - 좌측 (50%): Cocrates와의 대화 이미지 (말풍선 형태)
  - 우측 (50%): 3가지 질문 카드
  - 하단: 개인 트레이너 비유 (아이콘 + 짧은 텍스트)

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 🌱 사용자의 진화 — 대화의 내면화 | 상단 |
| 좌측 영역 | Cocrates 말풍선: "그냥 답만 주면 안 되나?" → 성장의 첫 신호 | 말풍선 그래픽, 화살표로 "성장" 연결 |
| 좌측 설명 | "질문은 주도권을 사용자에게 돌려주기 위한 파트너의 손짓" | 말풍선 아래 작은 텍스트 |
| 우측 제목 | 🤔 질문의 내면화 — 세 가지 질문이 사고방식이 되는 순간 | 우측 상단 |
| 질문 카드 1 | "이 결정의 트레이드오프는 무엇인가?" | 카드 형태, 아이콘 포함 |
| 질문 카드 2 | "내가 놓치고 있는 제약 조건은 없는가?" | 카드 형태, 아이콘 포함 |

| 비유 영역 | 🏋️ 개인 트레이너 비유 — 트레이너가 없어도 올바른 자세로 운동할 수 있게 되는 것 | 하단, 작은 아이콘 + 한 줄 텍스트 |
| 포인트 | Cocrates의 궁극적 목표 = 자신의 질문을 내면화하여 스스로 성장하는 사용자 | 하단, 강조 상자 |

### Visual Elements Description
- 말풍선: Cocrates의 말풍선(좌측)에서 사용자의 말풍선(우측)으로 화살표 연결, 점차 사용자 내부로 질문이 이동하는 애니메이션 효과
- 질문 카드: 그림자 효과가 있는 카드 디자인, 각 카드에 해당 아이콘 (저울, 돋보기, 연결)

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 말풍선 애니메이션으로 질문이 외부→내부로 이동하는 과정 시각화.
