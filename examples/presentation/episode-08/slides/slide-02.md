# Slide 02: 학습 목표

## Key Message
이 발표를 통해 시청자가 얻을 수 있는 3가지 핵심 목표 — "알려줘"의 위험성 인식, 3대 교육 철학 이해, Learning 파이프라인 파악.

## Content
### Headlines / text
- **도입 질문:** "만약 누군가가 '블룸 분류학에 대해 알려줘'라고 요청했다면? 대부분의 AI는 즉시 위키피디아식 답변을 쏟아냅니다. 하지만 Cocrates는 다릅니다. 왜 그럴까요?"
- **3가지 학습 목표:**
  1. ⚠️ "알려줘"라는 요청이 왜 위험한지 세 가지 함정(이해의 착각, 블랙박스 수용, 무지의 눈덩이)을 설명할 수 있다
  2. 🏛️ Cocrates 학습 메커니즘의 3대 교육 철학(소크라테스 산파술, 블룸 분류학 2차원 매트릭스, ZPD)을 설명할 수 있다
  3. 🔄 Learning 파이프라인 3단계(Education → Knowledge Capture → Reflection)와 각 단계의 역할을 설명할 수 있다

### Visual elements
- 체크리스트 스타일의 3개 항목 — 각 항목 왼쪽에 아이콘(⚠️, 🏛️, 🔄)
- 각 항목은 서로 다른 색상 영역으로 구분
- 상단에 도입 질문을 인용문 스타일로 배치

## Layout Notes
- 도입 질문은 상단에 인용문 형태로
- 3개 목표는 하단에 수평 또는 수직 배열
- 각 목표는 아이콘 + 핵심어 + 설명 순서
