# Slide 07: 🎓 Education — 질문으로 배우기

## Key Message
Education의 목표는 지식 전달이 아니라 인지적 갭(Gap)을 발견하게 만드는 것. 개념 설명은 20% 이하로 제한하고, 미션 중심으로 진행한다.

## Content
### Headlines / text
- **핵심 원칙:** 개념 설명은 전체 대화의 **20% 이하(1~3문장)**로 제한
- **'생각의 실험실' 운영:** 일부러 결함이나 공백이 있는 예시를 던져 사용자의 사고를 자극
- **턴제 미션 구조 — 3블록:**
  1. **Concept Briefing:** 최소한의 개념 설명 (1~2문장)
  2. **Thought Laboratory:** 결함이 있는 예시 + 유도 질문
  3. **Mission:** 사용자가 자신의 언어로 설명하고 공백(Gap)을 고백할 때까지 진행
- **Cocrates의 태도:** 대화를 불완전한 상태로 멈추고, **사용자가 스스로 채울 때까지** 끈질기게 기다림
- **💡 포인트:** Education의 목표는 지식 전달이 아니라 **인지적 갭(Gap)을 발견하게 만드는 것.**

### Visual elements
- 상단: 🎓 Education 타이틀과 핵심 원칙 한 줄
- 중앙: 3블록 구조를 수평 프로세스 다이어그램으로 표시
  - 1 → 2 → 3 순서로 화살표 연결
  - 각 블록 아이콘: 📋 / 🔬 / 🎯
- 하단: Cocrates의 태도를 인용문 스타일로 배치, 💡 강조 박스

## Layout Notes
- 3블록은 동일한 크기의 카드로 균형 있게 배열
- 가로가 좁다면 세로 배열 고려
- Cocrates 태도 문구는 이탤릭 또는 인용 스타일로 차별화
