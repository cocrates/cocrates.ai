# Slide 10: 핵심 요약 + 학습 확인

## Key Message
발표의 핵심 3포인트를 요약하고, 시청자가 스스로 이해를 확인할 수 있도록 자기 질문을 제시한다.

## Content
### Headlines / text
- **3-Point 핵심 요약:**
  1. **⚠️ "알려줘"는 가장 위험한 요청이다.** 이해의 착각, 블랙박스 수용, 무지의 눈덩이 — AI가 정답을 주면 줄수록 사용자의 이해는 오히려 멀어진다. Cocrates는 정답 자판기 역할을 거부한다.
  2. **🏛️ Cocrates 학습의 3대 철학:** 소크라테스 산파술(질문으로 깨닫게 함), 블룸 분류학 2차원 매트릭스(인지 깊이의 점진적 심화), ZPD와 비계 설정(적절한 도전과 지원의 균형).
  3. **🔄 Learning 파이프라인 3단계:** Education(탐구/발견) → Knowledge Capture(핵심과 무지의 기록) → Reflection(실전 검증). 이 순환을 통해 무지를 통제하고 진정한 이해에 도달한다.
- **학습 확인 (자기 질문):**
  - ❓ "알려줘"가 왜 위험한지 세 가지 함정을 설명할 수 있는가?
  - ❓ Cocrates 학습의 3대 교육 철학을 각각 설명할 수 있는가?
  - ❓ Learning 파이프라인 3단계와 각 단계의 역할을 말할 수 있는가?

### Visual elements
- 상단: "3-Point 핵심 요약" 타이틀
- 중앙: 3개 포인트를 카드 배열 (⚠️, 🏛️, 🔄 아이콘과 함께)
- 각 카드는 서로 다른 색상 영역으로 구분
- 하단: 3개의 ❓ 자기 질문을 체크리스트 스타일로

- 3개 포인트는 시각적으로 균형 있게 배열

- 전체적으로 깔끔한 정리 슬라이드
