# Script: Slide 09 — Cocrates Harness — Co-Socrates의 의미

**Time:** ~1분

**[SLIDE 09: 중앙에 "Cocrates = Co + Socrates" 타이틀 아래 "AI-native Engineer + AI 팀원 리딩 역량 강화". 양방향 화살표 — 좌측 "👤 사용자(소크라테스) → 🤖 AI", 우측 "🤖 AI(소크라테스) → 👤 사용자". 하단에 "질문 → 검토 → 판단"]**

**Cocrates Harness**는 여러분이 AI-native Engineer가 되도록 도와주는 에이전트 하네스입니다.

Cocrates는 **Co + Socrates** 입니다. 상호 소크라테스 역할을 한다는 의미입니다.

**당신은 AI에게 소크라테스가 됩니다.**

AI는 확률적으로 가장 그럴듯한 답변을 생성할 뿐입니다. 당신이 질문하고, 검토하고, 판단해서 AI가 잘못된 것을 생성하지 않도록 막아야 합니다. 이것이 바로 AI-native Engineer의 핵심 역량입니다.

(잠시 멈춤)

**AI도 당신에게 소크라테스가 됩니다.**

당신이 AI에게 막연한 요청을 하거나 AI의 산출물을 이해하지 못한채 승인하지 않도록 합니다. 당신이 대충 알고 넘어가는 것을 제대로 이해하도록 합니다. 소크라테스처럼 당신의 무지를 알게하고, 당신 스스로 역량을 키우도록 합니다.

(강조하며)

Cocrates는 바로 이렇게 **"상호 소크라테스 관계"로 AI를 사용하게 하는 에이전트 하네스**입니다. 이것이 바로 AI-native Engineer로서 AI 팀원을 이끄는 도구이며, AI를 올바르게 사용하는 방법입니다.
