# Slide 03: "jsondb를 만들어줘" — 하지만 AI는 설계부터 권했다

## Key Message
같은 요청("jsondb 만들어줘")에 대해 일반 AI와 Cocrates가 완전히 다르게 반응한다. Cocrates는 설계 없는 생성을 거부한다.

## Content
- **Headlines / text:**

    - "jsondb? 알겠습니다!"
    - → 즉시 수백 줄 코드 생성
    - 검토 없이 쏟아내는 코드
  - **우측: 🦉 Cocrates**
    - "이 jsondb는 어떤 용도인가요? 아키텍처를 먼저 설계하고 검토받은 후 생성하겠습니다."
- **Key takeaway:** Cocrates는 **설계 없는 생성을 거부한다** (Ep2 원칙)
- **오늘 만들 것:** jsondb — 로컬 JSON 저장소. 간단해 보이지만 3번의 ADR 논쟁 필요
