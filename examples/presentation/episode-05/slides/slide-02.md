# Slide 02: 학습 목표

## Key Message
이 발표를 통해 시청자가 얻게 될 4가지 학습 목표를 명확히 제시한다. "AI의 제안을 무조건 믿지 말라"는 핵심 전제를 심어준다.

## Content
- **Headlines / text:**
  - **도입 질문:** "여러분, AI에게 '이것 좀 만들어줘'라고 요청한 후, 그대로 쓰신 적 있나요?"
  - **핵심 전제:** AI의 제안이 항상 내 요구와 맞는 것은 아니다.
- **4가지 학습 목표:**
  1. 🏗️ 구조 기반 생성 파이프라인 4단계(ADR → Spec → Generation → Verification)를 설명할 수 있다
  2. 🔍 AI의 제안이 왜 내 요구와 맞지 않는지 3가지 이유(일반적이다 / 쉽고 간단하다 / 품질 미반영)를 실제 사례로 설명할 수 있다
  3. 📄 ADR과 Spec이 함께 살아있는 문서(Living Document)라는 개념을 이해한다
  4. 🤫 Undocumented ASR(암묵적 설계 결정)의 개념을 이해하고 검증의 중요성을 설명할 수 있다

## Visual Elements
- 각 목표 앞에 아이콘 배치 (🏗️, 🔍, 📄, 🤫)
- 도입 질문은 별도 박스 또는 인용 스타일로 강조
- 발표 전반의 로드맵 표시 (14슬라이드 중 현재 위치: 2/14)

## Layout Notes
- 학습 목표 슬라이드이므로 텍스트 중심, 깔끔한 리스트 형식
- 각 목표는 간격을 두고 가독성 확보
