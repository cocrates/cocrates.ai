# Slide 06: ⚠️ 침묵의 기본값 (Silent Default)

## Key Message
사용자가 결정하지 않으면 AI가 대신 결정한다. 침묵은 AI에게 결정권을 위임하는 것임을 경고하라.

## Content
- **Section badge (상단):** ⚠️ 2.4 침묵의 기본값 (Silent Default)
- **Core principle (큰 글씨, 중앙):** 사용자가 결정하지 않으면 AI가 대신 결정한다
- **Sub-principle:** — 그리고 그 결정은 사용자의 의도와 다를 확률이 높다
- **🧳 여행 계획 비유 (좌측):**
  - 호텔 등급/식당 종류를 말하지 않음 → 플래너가 '기본값'으로 중간급 호텔 + 관광객용 식당 추천
  - 내가 원한 로컬 맛집? 내가 말하지 않았으니 플래너가 마음대로 결정
- **AI 생성 예시 (우측):**
  - "보고서 작성해줘" → 독자 미지정 → AI가 "일반 직장인" 기본값
  - "발표자료 만들어줘" → 시간 미지정 → AI가 "20분" 기본값
- **💡 Point box (warning):** 침묵은 AI에게 결정권을 위임하는 것이다. ASR을 명시적으로 말하라

## Visual Elements
- 경고 느낌의 강한 색상 (빨간색 계열 또는 주황)
- 말풍선에 "기본값" + 사용자 의도와 다른 방향 화살표
- 여행 계획 + AI 생성 두 컬럼 나란히
