# Slide Content Specification — Slide 5

## Slide 5: 7계명 (1/3) — 계명 1~2

**Section:** 본론 2 — 7계명
**Type:** 선언 (계명)
**Time:** ~2분

### Visual Design
- **Background:** 시리즈 공통 템플릿 (약간 어두운 톤, 엄숙한 분위기)
- **Layout:** 두 개의 카드 (좌우 분할)
  - 상단: 제목 — "📜 7계명 (1/3) — 계명 1~2"
  - 중앙 좌측: 계명 1 카드
  - 중앙 우측: 계명 2 카드 (내부에 3가지 이유 서브 불릿)
  - 하단: 💡 포인트

### Visual Elements Description
- 두 카드: 동일한 크기, 좌우 대칭. 각 카드 좌측 상단에 ❌ 아이콘 (빨간색).
- 계명 2 카드: 내부에 3가지 이유를 번호 리스트로 표시. 각 이유 옆에 작은 🔗 아이콘으로 Ep5 연결 표시.
- 하단 포인트: 붉은 계열 강조 배경.

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 가장 긴 슬라이드(2분). 계명 2의 3가지 이유를 설명할 때 잠시 멈추고 청중의 시선이 각 이유를 따라가도록 유도.
