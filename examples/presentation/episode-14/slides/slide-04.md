# Slide Content Specification — Slide 4

## Slide 4: 철학의 연결 — 시작과 끝

**Section:** 본론 1 — 3대 철학
**Type:** 개념 설명 (타임라인)
**Time:** ~1분

### Visual Design
- **Background:** 시리즈 공통 템플릿
- **Layout:** 수평 타임라인
  - 상단: 제목 — "🔗 철학의 연결 — 시작과 끝"
  - 중앙: 수평 타임라인 (좌: Ep2 → 우: Ep14)
  - 하단: "14개 에피소드 = 하나의 이야기" 메시지

### Visual Elements Description
- 수평선: 좌에서 우로 이어지는 굵은 선. 에피소드 마커는 선 위에 원형 아이콘.
- Ep14 마커: 다른 색상(금색 또는 강조색) + 더 큰 크기.
- 마커 간 간격: 실제 에피소드 번호 간격과 무관하게 균등 배치.

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 우측 Ep14로 시선이 집중되도록. 타임라인을 따라가며 빠르게 훑고, Ep14에서 멈춤.
