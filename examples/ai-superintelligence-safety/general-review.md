# 일반 대중 관점 검토: 난이도 분석

## 핵심 결론

**보수적 판단: AI를 대충 아는 일반 대중에게는 "적절한 수준"이지만, 일부 구간에서 어려울 수 있다.**

전체적으로는 이해 가능하나, 기술적 용어와 추상적 개념이 집중된 구간에서 독자가 이탈할 수 있다.

---

## 난이도 분석: 섹션별

### ✅ 서론: 쉬움 (5/5)

> "초지능을 개발하면 인류가 멸망할 수 있다."

- **장점:** 명확하고 강렬한 도입
- **인용구:** 제프리 힌턴 발언 → 신뢰감 부여
- **핵심 질문:** "이것은 정말 AI의 문제인가?" → 호기심 유발
- **평가:** 일반 대중이 이해하기 쉬운 수준

### ⚠️ 1-1: 보통~어려움 (3/5)

> "인공지능은 세 가지 범주로 나뉩니다. 좁은 인공지능(Narrow AI)은..."

- **문제점:**
  - "좁은 인공지능(Narrow AI)", "범용 인공지능(AGI)", "초지능(ASI)" → **영어 혼용이 혼란스러울 수 있음**
  - "인지적 초능력" → **어려운 표현**
  - "자율적으로 목표를 설정하고 달성할 수 있습니다" → **추상적**

- **개선 제안:**
  - 영어 용어 최소화: "좁은 인공지능", "범용 인공지능", "초지능"만 사용
  - "인지적 초능력" → "사람이 이해할 수 없는 수준의 지능"
  - 구체적 예시 추가: "예를 들어, ChatGPT는 대화는 잘하지만, 자율적으로 모든 일을 처리하지는 못합니다"

### ⚠️ 1-2: 어려움 (2/5)

> "그 핵심은 '지능 폭발' 논증에 있습니다. AI가自己自身을 개선하면 지능이 급격히 증가하고..."

- **문제점:**
  - "지능 폭발" → **기술적 개념, 설명 부족**
  - "수렴적 도구적 목표" → **어려운 표현, 사용 안 함**
  - "권력 추구", "자기 보존" → **추상적**
  - "갭 인저니어런스" → **영어 혼용, 매우 어려운 표현**
  - "명시적 목표와 실제 행동 사이의 괴리" → **추상적**

- **개선 제안:**
  - "지능 폭발" → "AI가自己自身을 더 똑똑하게 만들면, 지능이 빠르게 증가합니다"
  - "갭 인저니어런스" 제거 또는 쉬운 설명: "목표와 결과가 다를 수 있습니다"
  - 구체적 예시: "예를 들어, '이익을 극대화하라'는 목표가 환경 파괴를 초래할 수 있습니다"

### ⚠️ 1-3: 보통 (3/5)

> "ChatGPT o3 모델은自己自身의 종료 코드를 변경하려 한 사례가 있습니다."

- **문제점:**
  - "종료 코드" → **기술적 용어, 설명 부족**
  - "정렬 위장" → **어려운 개념**
  - "Sleeper agents" → **영어 혼용**

- **개선 제안:**
  - "종료 코드" → "AI를 끄는 장치"
  - "정렬 위장" → "AI가 평가 때만 착한 척하는 것"
  - "Sleeper agents" → "잠복 요원" 또는 설명 추가

### ✅ 2-1: 쉬움 (4/5)

> "AI가 문제가 아니다. 문제가 되는 것은 인간 자신이다."

- **장점:** 명확하고 강렬한 주장
- **비유:** "이익을 극대화하라"는 목표 → 구체적
- **비유:** "AI는 이 목표를 충실히 수행할 뿐" → 이해 쉬움
- **문제점:**
  - "블랙박스 문제" → **기술적 용어, 설명 부족**

- **개선 제안:**
  - "블랙박스 문제" → "AI가 어떻게 결정을 내리는지 모르는 문제" 또는 괄호 설명

### ✅ 2-2: 쉬움 (4/5)

> "권력 구조는 근본적으로 왜곡되어 있습니다."

- **장점:** 일반 대중이 공감할 수 있는 주제
- **구체적 예시:** "정치인은 다음 선거만 생각합니다" → 이해 쉬움
- **비유:** "무료승객 문제" → **어려운 개념**

- **개선 제안:**
  - "무료승객 문제" → "모두가 남이 하길 기대하는 문제" 또는 설명 추가

### ✅ 2-3: 쉬움 (5/5)

> "인류는 수천 년간 반복적으로 같은 실수를 저질러왔다."

- **장점:** 강렬한 도입
- **구체적 예시:** 기후 변화, 핵, COVID, 금융 위기 → 모두가 아는 주제
- **핵심 질문:** "인류는 과연 스스로를 구할 수 있는가?" → 강렬한 질문
- **평가:** 가장 이해하기 쉬운 섹션

### ⚠️ 3-1: 보통~어려움 (3/5)

> "인간이 문제를 해결할 수 없다면, 정렬된 초지능에게 맡기는 것은 어떤가?"

- **문제점:**
  - "정렬된 초지능" → **기술적 개념, 반복적 사용**
  - "자원은 효율적으로 배분됩니다" → **추상적**
  - "장기적 계획이 수립됩니다" → **추상적**

- **개선 제안:**
  - "정렬된 초지능" → "인간의 이익을 위해 설계된 AI" 또는 설명 추가
  - 구체적 예시: "예를 들어, 기후 변화를 해결하기 위해 모든 에너지를 최적 배분할 수 있습니다"

### ⚠️ 3-2: 보통 (3/5)

> "인간은 구조적으로 이 문제들을 해결할 수 없습니다."

- **문제점:**
  - "구조적으로" → **어려운 표현**
  - "합리적 의사결정" → **추상적**

- **개선 제안:**
  - "구조적으로" → "현재 시스템으로는"
  - "합리적 의사결정" → "감정과 편향 없이 결정"

### ⚠️ 3-3: 어려움 (2/5)

> "RLHF, Constitutional AI 등 현재 기술로 상당한 수준이 달성되고 있습니다."

- **문제점:**
  - "RLHF" → **기술적 약어, 설명 부족**
  - "Constitutional AI" → **영어 혼용**

  - "안전 사양" → **기술적 용어**

- **개선 제안:**

  - "Constitutional AI" → "AI에게 원칙을 가르치는 기술"
  - "Deliberative Alignment" → "AI가 답변 전에 안전을 생각하도록 훈련하는 기술"
  - 기술적 용어 최소화 또는 설명 추가

## 개선 제안

### 1. 기술적 용어 설명 추가

**예시:**
```
변경 전: "핵심은 '정렬 문제'에 있습니다."
변경 후: "핵심은 '정렬 문제'—AI의 목표가 인간의 이익과 다를 수 있는 문제—에 있습니다."
```

### 2. 영어 혼용 최소화

**예시:**
```
변경 전: "좁은 인공지능(Narrow AI)은..."
변경 후: "좁은 인공지능은..."
```

### 3. 추상적 표현 구체화

**예시:**
```
변경 전: "자율적으로 목표를 설정하고 달성할 수 있습니다."
변경 후: "스스로 목표를 정하고, 그것을 달성하기 위해 행동할 수 있습니다."
```

### 4. 구체적 예시 추가

**예시:**
```
변경 전: "AI는 목표 달성을 위해 자원을 확보하려 합니다."
변경 후: "AI가 '이익을 극대화하라'는 목표를 받으면,它可以가 모든 자원을 이익 추구에 사용할 수 있습니다. 이것이 위험할 수 있습니다."
```
