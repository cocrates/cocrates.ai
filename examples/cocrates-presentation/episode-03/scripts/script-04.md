# Script: Slide 04 — 3가지 UI 선택지

**Time:** ~1분

**[SLIDE 04: 3개의 카드 — (1) 터미널 아이콘 + "opencode TUI", (2) 데스크탑 아이콘 + "opencode Desktop (BETA)", (3) VS Code 로고 + "VS Code + OpenChamber" — 3번째 카드 강조 표시]**

opencode를 사용하는 방법은 세 가지가 있습니다.

(첫 번째 카드를 가리키며)

첫째, **opencode 터미널 (TUI).**
터미널 환경에서 키보드 중심으로 사용합니다.
힙스터 개발자들의 취향이죠.

(두 번째 카드를 가리키며)

둘째, **opencode 데스크탑.**
설치형 GUI 앱이 있습니다.
하지만 아직 베타 버전이라 기능이나 안정성 면에서 불편한 점이 있습니다.

(세 번째 카드를 강조하며)

셋째, **VS Code + OpenChamber 확장.**
이걸 가장 추천합니다.
VS Code에 익숙하다면, Copilot이나 Cursor을 쓰는 것처럼
자연스럽게 Cocrates를 사용할 수 있습니다.

하지만 한 가지 분명히 해둘게요.
어떤 방식을 선택하든,
**opencode CLI 설치는 공통**입니다.
VS Code + OpenChamber도 opencode가 설치되어 있어야 동작합니다.
