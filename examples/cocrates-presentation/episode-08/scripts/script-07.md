# Script: Slide 07 — 🎓 Education — 질문으로 배우기

**Time:** ~1분

**[SLIDE 07: Education 3블록 구조 — Concept Briefing → Thought Laboratory → Mission / 인포그래픽]**

첫 번째 단계, Education입니다. 이름은 Education이지만, 여러분이 생각하는 전통적인 "교육"과는 다릅니다.

핵심 원칙은 이것입니다. 개념 설명은 전체 대화의 **20% 이하, 길어야 1~3문장**으로 극도로 제한합니다.

대신 Cocrates는 **일부러 결함이나 공백이 있는 예시**를 던지는 '생각의 실험실'을 운영합니다. 그리고 3블록 구조의 미션을 제시합니다.

**Concept Briefing —** 최소한의 개념 설명. 1~2문장.

**Thought Laboratory —** 결함이 있는 예시와 유도 질문으로 사용자의 사고를 자극.

**Mission —** 사용자가 자신의 언어로 설명하고 공백(Gap)을 고백할 때까지 진행.

(잠시 멈춤)

Cocrates는 대화를 일부러 불완전한 상태로 멈춥니다. 사용자가 스스로 빈칸을 채울 때까지 끈질기게 기다리는 거죠.

Ep4에서 Cocrates가 블룸 분류학의 2차원 매트릭스를 직접 설명하지 않고, 4개의 미션을 통해 깨닫게 한 방식이 바로 이것입니다.

Education의 진짜 목표는 지식 전달이 아닙니다. **인지적 갭, 즉 "내가 모르는 것"을 발견하게 만드는 것입니다.**
