# Slide 05: 📈 블룸 분류학 + ZPD — 실제 대화에서의 구현

## Key Message
3대 교육 철학이 실제 Cocrates 대화에서 어떻게 구현되는지, 누구나 공감할 수 있는 "수요와 공급의 법칙" 예시로 이해한다.

## Content
### Headlines / text
- **시나리오 예시 — '수요와 공급의 법칙' 4개 미션:**
  - **Mission 1 (Apply — 적용):** "스마트폰이 출시된 지 1년 후 가격이 하락하는 현상을 수요와 공급의 법칙으로 설명해보세요" → 개념을 실제 현상에 적용
  - **Mission 2 (Analyze — 분석):** "수요 곡선은 우하향, 공급 곡선은 우상향합니다. 이 기울기가 말하는 사람들의 경제적 행동 원리를 분석해보세요" → 관계 분석
  - **Mission 3 (Evaluate — 평가):** "정부가 전월세 상한제를 도입하면 어떤 조건에서 수요와 공급의 법칙이 유지되고, 어떤 조건에서 깨지나요?" → 조건부 판단
  - **Mission 4 (Create — 창조):** "전기차 폐배터리 재활용 시장의 수요와 공급 균형을 예측하는 모델을 설계해보세요" → 새로운 구조 창출
- **ZPD 구현의 실제:**
  - 각 미션은 사용자의 현재 수준보다 **정확히 한 단계 위**
  - 힌트는 점진적으로 제거 (처음에는 예시 제공 → 나중에는 스스로 생각하도록)
- **💡 포인트:** Cocrates는 단순 정보 전달자가 아니라 **사용자의 인지 수준에 맞춰 도전 과제를 설계하는 트레이너**다.

### Visual elements
- 상단: 블룸 분류학 피라미드 + "수요와 공급의 법칙" 레이블
- 중앙: 4개 미션이 위에서 아래로 계단식 배열 (Apply → Analyze → Evaluate → Create)
- 각 미션 왼쪽에 Bloom 단계 아이콘
- ZPD 부분은 별도 영역에 비계 아이콘과 함께 배치
- 하단 💡 강조 박스

## Layout Notes
- 계단식 배열로 Bloom 단계의 상승을 시각화
- 실제 발표자(트레이너) 비유를 시각적으로 연결
- 경제학 예시이므로 그래프/차트 느낌의 아이콘 사용 가능
