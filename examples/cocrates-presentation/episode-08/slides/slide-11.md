# Slide 11: 다음 에피소드 연결

## Key Message
Learning 파이프라인의 원리 이해 완료. 다음 단계(Ep9)에서는 이 개념들이 실제 스킬 파일에서 어떻게 구현되는지 내부를 들여다본다. PT 비유로 마무리.

## Content
### Headlines / text
- **지금까지의 전환:** Ep7(전체 구조 이해) → **Ep8(Learning 파이프라인의 원리 이해) 완료** ✅
- **다음 편 (Ep9): 소크라테스식 학습 스킬**
  - Education, Knowledge Capture, Reflection — 이 개념들이 실제 `.opencode/skills/` 안에서 **구체적인 코드 구조와 대화 규칙**으로 어떻게 구현되어 있는지 내부를 들여다본다
  - Ep4의 실습과 Ep8의 원리가 실제 스킬 파일에서 어떻게 연결되는지 확인
- **마무리 비유 (헬스장 PT):**
  > "Cocrates가 아무리 훌륭한 트레이너라도, 실제로 땀을 흘리며 움직이는 것은 시청자 자신입니다. 질문하고, 기록하고, 스스로를 검증하는 주체가 되어야 진짜 성장이 일어납니다."
- 🦉
