# Slide 04: 🏛️ 능동적 뇌를 깨우는 3대 교육 철학

## Key Message
Cocrates의 학습 방식은 단순한 심술이 아니라, 수천 년간 검증된 3대 교육 철학을 프롬프트 아키텍처로 구현한 결과물이다.

## Content
### Headlines / text
- **도입 질문:** "Cocrates는 왜 이렇게 학습을 설계했을까? 단순히 불친절해서가 아니다."
- **1. 소크라테스 산파술 (Socratic Midwifery):**
  - 산파가 직접 아이를 낳아주는 것이 아니라 산모가 잘 출산하도록 돕는 것
  - Cocrates는 정답을 떠먹여 주지 않고 **정교한 질문**으로 사용자가 스스로 깨닫게 함
- **2. 블룸의 교육분류학 (Bloom's Taxonomy) 2차원 매트릭스:**
  - 기억→이해→적용→분석→평가→창조의 6단계
  - **지식 차원 × 인지 과정**의 2차원 매트릭스로 활용
  - 사용자의 인지 깊이를 점진적으로 심화
- **3. 근접발달영역(ZPD)과 비계 설정 (Scaffolding):**
  - 혼자서는 못 하지만 지지대(비계)가 있으면 도달 가능한 영역
  - 사용자의 현재 수준보다 **정확히 딱 한 단계 위**의 미션 제시
  - 성장함에 따라 힌트 서서히 철거 → 자립 유도
- **💡 포인트:** 단순한 아이디어가 아니다. 수천 년간의 교육 철학을 프롬프트 아키텍처로 구현한 결과물.

### Visual elements
- 세 철학을 3열 수평 카드 배열 — 각각 아이콘(🤱 산파, 📊 매트릭스, 🏗️ 비계) + 핵심어 + 한 줄 설명
- 중간에 작은 연결 화살표로 세 철학이 하나의 학습 철학으로 통합됨을 암시
- 하단 💡 강조 박스

## Layout Notes
- 상단: 도입 질문 (작은 폰트)
- 중앙: 3개 카드 배열 (같은 크기, 시각적 균형)
- 하단: 💡 포인트 강조 박스
