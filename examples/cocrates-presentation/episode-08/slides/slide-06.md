# Slide 06: 🔄 Learning 파이프라인 개관

## Key Message
Cocrates 학습의 핵심 엔진 — Education → Knowledge Capture → Reflection 3단계 파이프라인. 각 단계가 독립적으로 동작하면서도 하나의 선순환을 이룬다.

## Content
### Headlines / text
- **3단계 파이프라인:**
  ```
  [Education: 탐구/발견] ─→ [Knowledge Capture: 무지의 기록] ─→ [Reflection: 실전 검증]
        ▲                                                             │
        └─────────────────── (이해 보완이 필요할 시) ───────────────────┘
  ```
- **3단계 개요:**
  1. **Education:** Cocrates가 질문과 미션으로 사용자의 생각을 유도
  2. **Knowledge Capture:** 배운 내용의 핵심과 모르는 것을 KB에 기록
  3. **Reflection:** 진짜로 이해했는지 기습 검증
- **순환 구조:** Reflection에서 Gap 발견 시 Education 단계로 환류
- **💡 포인트:** 이 파이프라인이 Cocrates 학습의 핵심 엔진. 각 단계가 독립적으로 동작하면서도 하나의 선순환을 이룬다.

### Visual elements
- 중앙에 큰 파이프라인 순환 다이어그램
- 3개 박스를 화살표로 연결, 하단에 환류 화살표
- 각 박스에 아이콘: 🎓 Education / 💾 Knowledge Capture / 🔍 Reflection
- 하단 💡 강조 박스

## Layout Notes
- 다이어그램이 슬라이드의 중심을 차지
- 3단계는 왼쪽→오른쪽 수평 배열, 환류는 아래쪽으로 곡선 화살표
- 색상 그라데이션으로 단계 간 전환을 시각화
