# Slide Content Specification — Slide 2

## Slide 2: 학습 목표 — 오늘 알게 될 3+1가지

**Section:** 도입
**Type:** 학습 목표
**Time:** ~1분

### Visual Design
- **Background:** 시리즈 공통 템플릿 (밝은 톤 또는 중간 톤)
- **Layout:** 리스트형
  - 상단: 제목 "오늘 알게 될 3+1가지"
  - 중단: 4개 항목을 세로 리스트 형태로
  - 하단: 도입 질문 (작은 텍스트)

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 🎯 오늘 알게 될 3+1가지 | 상단 |
| 항목 1 | 🌱 **사용자가 진화해야 한다** — Cocrates의 질문이 내 사고방식이 되는 순간 | 아이콘 + 굵은 텍스트 |
| 항목 2 | 🔧 **Cocrates Harness도 진화해야 한다** — 스킬 확장, 프로세스 커스터마이징 | 아이콘 + 굵은 텍스트 |
| 항목 3 | 🔄 **함께 자라는 피드백 루프** — 사용 경험 → 개선 → 더 나은 경험 → 성장 | 아이콘 + 굵은 텍스트 |
| 항목 4 (강조) | 🎯 **Cocrates를 사용하는 것으로 충분하지 않다** — AI-native Engineer로서 우리 팀의 도구를 지속적으로 진화시켜야 한다 | 다른 색상/크기로 시각적 강조 |
| 도입 질문 | "Ep12에서 우리는 스킬을 만드는 방법을 배웠습니다. 그렇다면 만들어진 Cocrates를 어떻게 해야 할까요?" | 하단, 작은 텍스트 |

### Visual Hierarchy
- 항목 1-3: 동일한 크기와 스타일
- 항목 4: 더 큰 폰트, 다른 강조색 (예: 골드/오렌지), 배경 하이라이트 또는 테두리

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 항목 4는 발표자가 "특히 이 네 번째가 가장 중요합니다"라고 말할 때 시각적으로 강조 효과.
