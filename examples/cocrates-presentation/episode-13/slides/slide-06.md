# Slide Content Specification — Slide 6

## Slide 6: 함께 자라는 피드백 루프 — 지속적 진화의 엔진

**Section:** 본론 2 — Cocrates Harness의 진화
**Type:** 개념 설명 (순환 구조)
**Time:** ~1분 30초

### Visual Design
- **Background:** 시리즈 공통 템플릿
- **Layout:** 상/하 분할
  - 상단 (70%): 피드백 루프 원형 다이어그램 (중앙)
  - 하단 (30%): 사용자 참여도와 성능 관계 그래프

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 🔄 함께 자라는 피드백 루프 — **지속적 진화의 엔진** | 상단 |
| 피드백 루프 (원형) | **사용 경험** → **에이전트/스킬 개선** → **더 나은 경험** → **사용자 성장** → (다시 사용 경험으로) | 4단계 원형 화살표, 각 단계 아이콘 |
| 루프 설명 (AI-native) | • 나(팀장)는 AI 팀원과 함께 일한다<br>• AI 팀원이 더 잘 일하게 하려면 **도구를 개선**해야 한다<br>• 도구 개선 → 팀 생산성 향상 → 나의 검토 역량 성장 | 원형 다이어그램 옆, 작은 텍스트 |
| 강조 메시지 | ⭐ **진정한 AI-native Engineer = 팀의 도구를 지속적으로 진화시키는 팀장** | 중앙, 강조 상자 |
| "사용하는 것으로 충분하지 않다" | ✅ Cocrates는 우리 팀이 일을 하는 도구다<br>❌ 하지만 주어진 도구를 그저 사용하는 것에서 멈추면 안 된다 | 좌측, 체크/엑스 아이콘 |
| 그래프 | **사용자 참여도 ↔ 성능 = 정비례** — 상승 곡선 | 하단, 간단한 선 그래프 |

### Visual Elements Description
- 원형 다이어그램: 4개의 원을 화살표로 연결, 각 원 안에 아이콘 (경험: 👤, 개선: 🔧, 경험: ⭐, 성장: 📈)
- 화살표는 시계 방향으로 순환
- 그래프: X축=참여도, Y축=성능, 우상향 곡선

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 원형 다이어그램이 등장할 때 각 단계가 순차적으로 나타나는 애니메이션 효과.
