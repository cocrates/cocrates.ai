# Slide 07: 🏗️ Core Activity 1 — 산출물 생성 파이프라인

## Key Message
4단계 파이프라인: ADR(선택과 근거) → Spec(명세 통합) → Generation(Spec 기반 생성) → Verification(항목별 검증).

## Content
- **파이프라인:** ADR → Spec → Generation → Verification
- **각 단계 역할:**
  - **ADR:** 선택지 분석과 결정 ("왜 이렇게 할까?")
  - **Spec:** 결정된 모든 요구사항을 통합 ("무엇을 만들까?")
  - **Generation:** 승인된 Spec만을 근거로 생성
  - **Verification:** 결과물이 Spec과 일치하는지 항목별 검증
- **연결:** Ep5 jsondb 사례 — 3개 ADR, 72개 검증 항목

## Visual Elements
- 4단계 horizontal flow with arrows: ADR → Spec → Generation → Verification
- 각 단계 아래 아이콘 및 1줄 설명
- 하단: "담당 스킬: adr-writing, spec-writing, spec-driven-generation, spec-driven-verification"
