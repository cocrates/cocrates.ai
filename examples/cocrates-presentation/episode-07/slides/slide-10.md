# Slide 10: 핵심 요약 + 학습 확인

## Key Message
3-Point 요약 (만능 칼은 없다 / 6개 섹션 헌법 / Cocrates는 완성품이 아니다) + 3가지 자기 질문.

## Content
- **3-Point 핵심 요약:**
  1. **🍳 만능 칼은 없다.** 산출물 유형마다 구조적 접근이 다르다. Cocrates는 Agent(헌법)와 Skills(전문 부대)를 분리하여 해결.
  2. **📜 Cocrates Agent = 6개 섹션의 헌법.** Persona → Principle → Harness Architecture → Request Handling → Core Activities → Success Criteria. Intent-To-Skill Routing이 핵심.
  3. **🔄 Cocrates는 완성된 하네스가 아니다.** 구조 이해 = 커스터마이징과 진화의 첫걸음. 새 스킬 추가/수정으로 발전시켜야 한다.
- **학습 확인 (자기 질문):**
  - ❓ Cocrates가 Agent + Skills 구조를 채택한 이유를 설명할 수 있는가?
  - ❓ Cocrates Agent Prompt의 6개 섹션을 말할 수 있는가?
  - ❓ Intent-To-Skill Routing이 의도를 기반으로 어떻게 스킬을 선택하는지 설명할 수 있는가?

## Visual Elements
- 상단: 3-Point 요약 (각 포인트 아이콘 + 핵심 문장)
- 구분선
- 하단: 3개의 자기 질문 (❓ 아이콘)

## Layout Notes
- 요약과 학습 확인을 명확히 구분 (구분선)
- 각 요약 포인트는 2~3줄로 제한
- 자기 질문은 시청자가 잠시 멈추고 생각하도록 여백 확보
