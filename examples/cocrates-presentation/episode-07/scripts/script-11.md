# Script: Slide 11 — 다음 에피소드 연결

**Time:** ~1.25분

**[SLIDE 11: 여정 맵 — Part 3 (구조 편) 로드맵. Ep7(오늘: 전체 구조) → Ep8(학습 파이프라인 원리) 강조. 마지막 질문]**

오늘 우리는 Cocrates의 전체 구조를 이해했습니다. Agent + Skills 아키텍처, 6개 섹션, Intent-To-Skill Routing, 두 가지 Core Activity 파이프라인까지.

이제부터 Part 3에서는 이 구조 위에서 각 Core Activity가 어떤 원리로 작동하는지 하나씩 깊이 들어갑니다.

다음 편, Ep8의 주제는 **소크라테스식 학습 활동 - 원리**입니다. 오늘 배운 두 가지 Core Activity 중 첫 번째인 **학습 파이프라인의 원리**로 딥다이브합니다.

Ep4에서 여러분이 실습으로 경험한 그 학습 방식이 도대체 어떤 교육학적 원리로 설계된 것인지, 그 속을 낱낱이 파헤칠 겁니다. 소크라테스 산파술, 블룸 분류학, ZPD — 이 세 가지 철학이 어떻게 Cocrates의 학습 파이프라인에 구현되었는지 알아보겠습니다.

마지막으로 여러분께 한 가지 질문을 드립니다.

**"여러분은 Cocrates를 '사용'하는 것에서 멈출 것인가, '이해'하고 자신의 것으로 진화시킬 것인가?"**

Ep7에서 구조를 이해했다면, 이제 여러분의 선택입니다.
