# Script: Slide 02 — 학습 목표

**Time:** ~1분

**[SLIDE 02: 학습 목표 — 3가지 항목: 1. No Spoon-feeding + Turn-based Mission + 3블록 / 2. Ignorance 기록 + Merge / 3. Interviewer 페르소나 + Gap 처리]**

Ep8에서 Learning 파이프라인의 원리를 배웠습니다. 그런데 이 원칙들이 실제로 어떻게 '코드'로 굳어져 있을까요?

오늘 발표를 통해 여러분은 다음 세 가지를 설명할 수 있게 됩니다.

**첫째,** Education 스킬의 No Spoon-feeding 원칙 — 왜 Cocrates는 절대 정답을 바로 알려주지 않는지. 그리고 Turn-based Mission 구조 — 모든 응답이 왜 정확히 하나의 미션으로 끝나는지. 여기에 3블록 출력 포맷이 어떻게 작동하는지까지.

**둘째,** Knowledge Capture 스킬의 Ignorance 기록 철학 — 무엇을 배웠는지보다 무엇을 오해했는지를 기록하는 이유. 그리고 Merge 전략 — 지식이 어떻게 누적되는지.

**셋째,** Reflection 스킬의 Interviewer 페르소나 — 친절한 코치에서 엄격한 면접관으로의 전환. 그리고 Gap 발견 시 '가르치지 않는' 원칙.

자, 그럼 Education 스킬의 내부부터 들어가 보겠습니다.
