# Script: Slide 04 — 📊 Bloom's 2D Matrix + Push/Pull 전략

**Time:** ~1분

**[SLIDE 04: Bloom's 2D 매트릭스 (Y축 인지 과정 × X축 지식 차원) + Push/Pull 전환 다이어그램]**

그런데 Cocrates가 모든 사용자에게 똑같은 질문을 던질까요? 당연히 아닙니다. Education 스킬에는 사용자의 수준에 맞게 난이도를 동적으로 조절하는 두 가지 메커니즘이 있습니다.

**첫 번째, Bloom's 2D 매트릭스.** Y축은 인지 과정 — 기억에서 창조까지. X축은 지식 차원 — 사실적 지식에서 메타인지까지. 핵심 규칙은 '동시 격상 금지'입니다. 두 축을 동시에 올리지 않습니다. 같은 지식 차원 내에서 인지 깊이를 한 단계 높이거나, 같은 인지 과정에서 지식 차원을 넓히는 방식입니다.

**두 번째, Push/Pull 전략.** 기본값은 Pull입니다. 높은 수준의 도전을 먼저 던져서 사용자가 스스로 지식을 끌어당기게 만듭니다. 그런데 사용자에게 인지적 붕괴나 혼란이 감지되면? 즉시 Push로 전환합니다. 단계를 낮춰서 징검다리를 놓아주는 거죠.

(잠시 멈춤)

단순한 퀴즈가 아닙니다. 사용자의 실시간 반응에 적응하는 지능형 난이도 조절 엔진입니다. Cocrates가 Ep4에서 사용자의 답변에 따라 다음 미션을 조절했던 경험, 기억나시나요? 그게 바로 이 메커니즘입니다.
