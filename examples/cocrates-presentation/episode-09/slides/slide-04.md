# Slide 04: 📊 Bloom's 2D Matrix + Push/Pull 전략

## Key Message
Education 스킬은 Bloom's 2D 매트릭스와 Push/Pull 전략으로 사용자의 실시간 반응에 적응하는 지능형 난이도 조절을 수행한다.

## Content
### Headlines / text
- **Bloom's 2D Matrix 활용:**
  - Y축(인지 과정): Remember → Understand → Apply → Analyze → Evaluate → Create
  - X축(지식 차원): Factual → Conceptual → Procedural → Metacognitive
  - **핵심 규칙 — 동시 격상 금지:** 두 축을 동시에 올리지 않음
- **Push/Pull 전략:**
  - **Pull (기본값):** 높은 수준 도전 → 사용자가 지식을 끌어당김
  - **Push:** 인지적 붕괴 시 단계 낮춰 징검다리 제공
  - **판단 규칙:** 기본 Pull, 혼란 시 Push로 전환 후 한 축만 낮춤
- **💡 포인트:** 사용자의 실시간 반응에 적응하는 지능형 난이도 조절 엔진

### Visual elements
- 중앙: Bloom's 2D 매트릭스 다이어그램 (Y축 × X축)
- 우측: Push/Pull 전환 화살표 다이어그램
- 하단: 💡 강조 박스

## Layout Notes

- Push/Pull은 우측에 별도 영역
