# Slide 03: 선언 — "The Unexamined Code Is Not Worth Generating"

## Key Message
AI-native Engineer의 핵심 역량은 검토이며, 이를 한 문장으로 선언한다: 검토되지 않은 산출물은 생성할 가치가 없다.

- **Headline (작게, 상단):** AI-native Engineer의 핵심 역량 = 검토
- **선언문 (가장 크게, 중앙):** "The unexamined code is not worth generating."
- **번역 (아래):** "검토되지 않은 산출물은 생성할 가치가 없다"
- **Visual Elements:**

  - 따옴표 장식으로 강조
  - 배경은 최소화하여 텍스트에 집중

## Layout Notes
