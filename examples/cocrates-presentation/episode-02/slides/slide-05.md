# Slide 05: 'Code'의 확장

## Key Message
Cocrates가 말하는 'Code'는 소스 코드만이 아니다. AI를 사용해 만드는 모든 최종 산출물이 Code다.

## Content
- **Headline:** Cocrates의 Code = AI로 만드는 모든 최종 산출물
- **5가지 예시 (아이콘 카드, 좌→우):**
  - 💻 소스 코드
  - 📝 보고서와 문서
  - 📊 발표 자료와 슬라이드
  - 📚 블로그 시리즈
  - 🧠 학습 노트와 요약본
- **하단 멘트:** "당신이 AI에게 만들어 달라고 요청한 모든 것 = Code"
- **Visual Elements:**
  - 5개 아이콘을 카드 형식으로 가로 배열
  - 각 카드 아래에 텍스트 레이블
  - 아이콘은 흑백/단색으로 통일하여 일관감

## Layout Notes
- 5개 카드를 균등 간격으로 배치
- 1줄로 배치 어려울 경우 2줄 (3+2) 가능
- 카드 디자인: 둥근 모서리, 얇은 테두리, 그림자
