# Slide 04: 소크라테스의 영감

## Key Message
Cocrates의 선언은 소크라테스의 "성찰하지 않는 삶은 살 가치가 없다"에서 영감을 받았다. 철학과 개발의 연결을 통해 원칙의 깊이를 전달한다.

## Content
- **좌측 — 소크라테스:** 아이콘 + "The unexamined life is not worth living" / "성찰하지 않는 삶은 살 가치가 없다"
- **가운데:** 화살표(→) 또는 연결선
- **우측 — Cocrates:** 로고 + "The unexamined code is not worth generating" / "검토되지 않은 산출물은 생성할 가치가 없다"
- **하단:** "끊임없는 질문과 성찰 — 철학에서 개발로"
- **Visual Elements:**
  - 좌측에 소크라테스 얼굴 아이콘 (실루엣 또는 간단한 일러스트)
  - 우측에 Cocrates 로고 (올빼미 또는 모자 아이콘)
  - 중앙 연결 화살표

## Layout Notes
- 3분할 레이아웃 (좌: 소크라테스 / 중: 연결 / 우: Cocrates)
- 텍스트는 각 아이콘 아래 배치
- 소크라테스와 Cocrates 모두 같은 수평선상에 배치하여 대칭감
