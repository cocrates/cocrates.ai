# Slide 12: 다음 에피소드 연결

## Key Message
원칙("무엇")을 알았으니 이제 실천("어떻게")할 차례. 다음 편에서는 Cocrates Harness 설치 가이드를 제공한다.

## Content
- **좌측 — 오늘:** "무엇"이 옳은 일인지 알게 되었다 (원칙)
- **우측 — 다음:** "어떻게" 실천할지 배운다 (설치와 첫 대화)
- **중앙 화살표:** 오늘 → 다음
- **하단:** "Cocrates Harness 설치 — 생각보다 정말 간단합니다"
- **마무리:** "진짜 여정의 첫걸음, 다음 편에서 함께 내딛자"
- **Visual Elements:**
  - 시리즈 맵 (전체 14개 에피소드 중 현재 Ep2 위치 표시, 다음 Ep3 위치 강조)
  - 또는 간단한 "오늘 → 다음" 진행형 아이콘

## Layout Notes
