# Slide 02: 학습 목표

## Key Message
이 에피소드를 마치면 세 가지를 할 수 있게 된다: (1) 핵심 원칙 이해 (2) Code의 확장된 의미 이해 (3) 검토(U→A→E→A)의 진짜 의미 설명

## Content
- **Headline:** 오늘의 학습 목표
- **Checklist (빈 체크박스 3개):**

  - [ ] 'Code'의 확장된 의미를 이해한다 (소스 코드만이 아니다)

- **Visual Elements:**
