# Script: Slide 04 — 소크라테스의 영감

**Time:** ~1분

**[SLIDE 04: 좌측에 소크라테스 아이콘과 "The unexamined life is not worth living". 우측에 Cocrates 로고와 "The unexamined code is not worth generating". 가운데 화살표 연결. 하단에 "끊임없는 질문과 성찰" 텍스트]**

그런데, 혹시 이 선언이 어디서 영감을 받았는지 아시나요?

(소크라테스 쪽을 가리키며)

소크라테스는 말했습니다.
**"The unexamined life is not worth living."**
"성찰하지 않는 삶은 살 가치가 없다."

끊임없이 질문하고, 자신의 무지를 인정하며,
진리를 향해 나아가는 것.
그것이 소크라테스가 말한 철학자의 삶이었습니다.

(Cocrates 쪽으로 시선을 옮기며)

Cocrates는 이 철학을 개발의 영역으로 가져옵니다.
**"The unexamined code is not worth generating."**
"내가 이해하고 검토하지 않은 AI의 결과물은 가치가 없다."

단순한 말장난이 아닙니다.
소크라테스가 삶의 태도에 대해 질문했다면,
Cocrates는 AI 시대 엔지니어의 태도에 대해 질문하는 겁니다.

"당신은 지금, 검토하지 않은 결과물을 그냥 쓰고 있지 않은가?"

→ [SLIDE 05: 'Code'의 확장]
