# Script: Slide 05 — 'Code'의 확장

**Time:** ~1분

**[SLIDE 05: 5개 아이콘을 카드 형식으로 나란히 배치. 💻 소스 코드 | 📝 보고서 | 📊 슬라이드 | 📚 블로그 | 🧠 학습 노트. 상단에 "Cocrates의 Code = AI로 만드는 모든 최종 산출물"]**

(밝은 어조로)

여기서 한 가지 짚고 넘어갈 점이 있어요.

"어? 나는 개발자가 아닌데... Code라니?
내가 무슨 코드를 검토해?"

(고개를 저으며)

아닙니다. Cocrates가 말하는 **Code**는 훨씬 더 넓은 의미입니다.

(아이콘을 하나씩 가리키며)

💻 소스 코드는 당연히 포함됩니다.
하지만 그뿐만이 아닙니다.
📝 AI가 작성해준 보고서와 문서,
📊 발표 자료와 슬라이드,
📚 블로그 시리즈,
🧠 그리고 당신의 학습 노트와 요약본까지.

**당신이 AI에게 "만들어 줘"라고 요청한 모든 것.**
그것이 바로 Cocrates가 말하는 Code입니다.

이 에피소드를 보고 계신 당신,
지금 이 슬라이드도 AI가 만들었습니다.
그렇다면 이것 역시 Code입니다.
그리고 저는 이것을 충분히 검토하고 생성했습니다.
