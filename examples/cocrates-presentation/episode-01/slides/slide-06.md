# Slide 06: AI-assisted vs AI-native — 명확한 구분 & "가능할까?"

## Key Message
AI-assisted와 AI-native는 명확히 구분된다. AI-native에서는 AI가 일을 수행하고 사용자가 leader로 이끈다. 하지만 많은 엔지니어들이 "이게 진짜 가능할까?"라고 의문을 가진다.

## Content
- **Headlines / text:**
  - 좌측: **AI-assisted Engineer** — AI가 나를 도와주는 구조. AI = 보조 도구
  - 우측: **AI-native Engineer** — AI가 일을 수행하고, 내가 leader로 AI 팀원을 이끄는 구조. AI = 팀원
  - AI-native의 핵심: **지시 → AI가 수행 → 내가 검토(이해 → 분석 → 평가 → 승인)**
  - 하단 물음표: "근데... 이렇게 하는 것이 진짜 가능할까?"
- **Data / figures:** 없음
- **Visual elements:**
  - 좌우 비교 다이어그램
  - 좌측: 화살표 흐름 "나 → AI(도구) → 나"
  - 우측: 화살표 흐름 "나(Leader) → AI(팀원) → 수행 → 검토(U→A→E→A) → 승인"
  - 하단: 큰 물음표 아이콘
