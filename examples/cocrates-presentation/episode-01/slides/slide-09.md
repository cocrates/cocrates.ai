# Slide 09: Cocrates Harness — Co-Socrates의 의미

## Key Message
Cocrates = Co + Socrates = 상호 소크라테스 관계. 사용자는 AI에게 소크라테스가 되어 검토·판단하고, AI는 사용자에게 소크라테스가 되어 막연한 요청/무지한 승인을 방지한다.

## Content
- **Headlines / text:**
  - **Cocrates = Co(상호) + Socrates(소크라테스)**
  - 부제: "AI-native Engineer + AI 팀원 리딩 역량 강화"
  - **첫째, 당신이 AI에게 소크라테스가 됩니다:**
    - 질문하고, 검토하고, 판단하여 잘못된 생성을 방지
    - 이것이 AI-native Engineer의 핵심 역량
  - **둘째, AI도 당신에게 소크라테스가 됩니다:**
    - 막연한 요청이나 이해 없는 승인을 방지
    - 대충 넘어가는 것을 제대로 이해하도록
    - 무지를 알게 하고 스스로 역량을 키우도록
  - 결론: **"상호 소크라테스 관계"로 AI를 사용하게 하는 에이전트 하네스**
- **Data / figures:** 없음
- **Visual elements:**
  - 중앙 상단: "Cocrates = Co + Socrates" 타이틀
  - 양방향 화살표 아이콘
    - 좌측: "👤 사용자(소크라테스) → 🤖 AI"
    - 우측: "🤖 AI(소크라테스) → 👤 사용자"
  - 하단: "질문 → 검토 → 판단"
