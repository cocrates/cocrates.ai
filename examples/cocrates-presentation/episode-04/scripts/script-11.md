# Script: Slide 11 — 핵심 요약 + 학습 확인 (자기 진단)

**Time:** ~1.5분

**[SLIDE 11: 4개 요약 카드 (아이콘 + 핵심 문장). 하단에 자기 진단 질문 박스 (거울 아이콘 + 질문 4개)]**

자, 오늘 배운 내용을 정리해볼게요.
네 가지만 기억하세요.

(하나씩 짚으며)

**하나, 블룸 분류학.**
6단계 피라미드 — 기억, 이해, 적용, 분석, 평가, 창조.
진짜는 2차원 매트릭스 — Knowledge × Cognitive.
그리고 Push와 Pull 전략으로 학습을 설계할 수 있습니다.

**둘, Education.**
Cocrates는 답을 주는 자판기가 아닙니다.
미션과 Pull 전략으로 여러분이 스스로 생각하게 만듭니다.

**셋, Knowledge Capture.**
'내가 틀렸던 가정(Gap)'을 기록하는 것이
핵심 요약보다 강력합니다.
무지의 기록이 나중에 깨달음을 생생하게 재생합니다.

**넷, Reflection.**
도전적인 질문을 통해
진짜 아는 영역(✅)과 어설프게 아는 영역(⚠️)의
경계를 획정합니다.

(시청자를 바라보며, 진지하게)

자, 이제 여러분 스스로에게 질문할 시간입니다.

**질문 1:** 블룸 분류학의 6단계를 말할 수 있는가?

**질문 2:** 2차원 매트릭스(Knowledge × Cognitive)가
무엇인지 설명할 수 있는가?

**질문 3:** Push와 Pull 전략의 차이와
각각 언제 사용하는지 설명할 수 있는가?

**질문 4:** **지금 여러분은 블룸 분류학의
어느 단계에 있는가?**

기억하는 수준인가요?
이해하는 수준인가요?
아니면 직접 적용할 수 있는 수준인가요?

이 질문에 스스로 답할 수 있다면,
오늘 에피소드는 성공입니다.
