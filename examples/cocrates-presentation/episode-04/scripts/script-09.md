# Script: Slide 09 — 🔍 Reflection ① — 면접관 등장

**Time:** ~1.5분

**[SLIDE 09: Cocrates 아이콘이 🦉에서 🎙️(면접관)으로 변신. 큰 말풍선 질문. 유저의 당황한 반응 (💦). 하단에 "이해 ≠ 적용" 비교]**

Knowledge Capture까지 마쳤습니다.
이제 마지막 단계입니다.

유저가 말합니다.
"내가 제대로 이해했는지 검증해 줘."

(아이콘 변화를 가리키며)

그러자 Cocrates가 변신합니다.
🦉 코치에서 🎙️ **면접관(Interviewer)** 으로.

(면접관 목소리로)

"좋습니다. 그렇다면 방금 짜신 Git 커리큘럼의
세션들을 2차원 매트릭스의
**어느 셀**에 각각 배치할 수 있는지
매핑해 보세요."

(잠시 멈추며, 고개를 저으며)

유저의 반응은 이러했습니다.
"말문이 턱 막혔습니다."

왜일까요?

개념을 말로 설명하는 수준(이해)과,
실제 매트릭스 셀에 자로 잰 듯 배치하는 수준(분석, 적용).
이 둘 사이에는 상당한 격차가 있습니다.

(💡 표시)

여기서 Reflection의 진짜 가치가 드러납니다.

**내 이해의 경계를 스스로 발견하는 것.**

"아, 나는 이 부분을 설명할 수는 있는데,
실제로 적용하려니까 헷갈리네."

이렇게 내 경계를 아는 것,
그것이 Reflection의 힘입니다.
