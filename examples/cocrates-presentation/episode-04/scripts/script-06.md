# Script: Slide 06 — 🎯 미션 3 — "Push와 Pull, 언제 무엇을 써야 할까?" (세 번째 깨달음)

**Time:** ~1.5분

**[SLIDE 06: Push 아이콘 (밀기, →) ↔ Pull 아이콘 (당기기, ←). 사이에 학습자 상태 막대 (초보자 ↔ 숙련자). 중간에 인지적 붕괴 지점 전환 스위치]**

두 번의 깨달음을 통해
블룸 분류학의 진짜 모습이 드러났습니다.

이제 Cocrates가 자신의 전략을
메타인지적으로 설명하기 시작합니다.

(연기하듯)

Cocrates가 묻습니다.

"지금까지 제가 당신에게 질문을 던진 방식과,
당신이 헤맬 때 개념을 설명해준 방식.
이 두 전략의 차이는 무엇이고,
각각 언제 효과적일까요?"

(Push를 가리키며)

**Push 전략.** 순차적 주입입니다.
낮은 단계 → 높은 단계로 차근차근 가르치는 방식.
초보자에게, 또는 인지 과부하 상태의 학습자에게 적합합니다.
전통적인 교육이 주로 이 방식을 쓰죠.

(Pull을 가리키며)

**Pull 전략.** 미션형 인출입니다.
다짜고짜 높은 단계의 과제를 던져서,
필요에 의해 아래 단계를 학습하게 하는 방식.
기본 지식이 있는 학습자에게, 동기부여가 필요할 때 최고입니다.

(전환 스위치를 가리키며)

Cocrates는 기본적으로 **Pull 전략**을 사용합니다.
떡하니 결함 있는 미션을 주고 생각하게 만들죠.
그러다 유저가 뇌정지, 즉 인지적 붕괴를 일으키면,
싹 상냥하게 **Push 전략**으로 전환해 개념을 짚어줍니다.

(잠시 멈추며)

"가르치는 사람은 정원사와 같습니다.
때로는 뿌리에 물을 주고(Push),
때로는 잎에 물을 줍니다(Pull)."

이것이 세 번째 깨달음입니다.
