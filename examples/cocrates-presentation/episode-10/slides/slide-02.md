# Slide 02: 학습 목표

## Key Message
오늘 발표의 3가지 목표를 제시한다: 문제 인식 → ASR 이해 → 파이프라인 습득.

## Content
- **도입 질문 (italic, 상단):** "AI가 이렇게 잘 생성해주는데, 왜 굳이 먼저 구조를 설계해야 할까요?"
- **Card 1 (🏗️):** "그냥 작성해줘"의 문제 — 논리 붕괴 · 설명 책임 상실 · 블랙박스화
- **Card 2 (🔍):** ASR과 침묵의 기본값 — 구조적으로 중요한 요구사항을 식별하지 않을 때의 위험
- **Card 3 (🏡):** 4단계 파이프라인 — Concern/ASR → ADR → Spec → Generation & Verification

## Visual Elements
- 3개의 동일 크기 카드, 가로 배치
- 각 카드: 아이콘(상단) + 제목(중앙) + 설명(하단)

## Layout Notes
- Three-card 레이아웃, 흰 배경, 얇은 카드 테두리
- 상단 도입 질문은 이탤릭 처리
