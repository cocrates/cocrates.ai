# Slide 12: 📝 핵심 요약 + 학습 확인

## Key Message
3-Point 핵심 요약을 기억하고, 스스로 학습 확인 질문에 답할 수 있어야 한다.

## Content
- **Section title (상단):** 핵심 요약 및 학습 확인
- **Point 1 (🏗️ 카드):** "그냥 써줘" = "아무렇게나 써줘" — 논리 붕괴, 책임 상실, 블랙박스화
- **Point 2 (🔍 카드):** 침묵의 기본값을 깨부숴라 — ASR을 명시하지 않으면 AI가 대신 결정
- **Point 3 (🏡 카드):** 4단계 파이프라인 — Concern/ASR → ADR → Spec → Gen&Ver
- **자기 질문 (하단, 체크리스트):**
  - ❓ "그냥 작성해줘"가 낳는 세 가지 문제는 무엇인가?
  - ❓ ASR은 무엇이고, 왜 중요한가?
  - ❓ 4단계 파이프라인의 각 단계를 설명할 수 있는가?

## Visual Elements
