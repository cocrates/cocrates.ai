# Slide 07: 🏡 전원주택 건축으로 보는 4단계 파이프라인 개관

## Key Message
4단계 파이프라인(Concern/ASR → ADR → Spec → Gen&Ver)이 Cocrates 구조 기반 생성의 핵심 엔진임을 한눈에 이해하라.

## Content
- **Section badge (상단):** 🏡 2.5 4단계 파이프라인 개관
- **Pipeline diagram (수평, 4개 상자 + 화살표):**
  - **① Concern/ASR 식별:** 중요한 질문들 도출
  - **→ ② ADR:** 선택지 비교 → 사용자 결정
  - **→ ③ Spec:** 결정사항 통합 설계도
  - **→ ④ Generation & Verification:** 설계도대로 시공 및 검수
- **Analogy row (파이프라인 아래):**
  - **1단계:** "지붕을 할까, 옥상을 할까, 옥탑방을 할까?" — 질문 도출
  - **2단계:** "지붕 vs 옥상 vs 옥탑방? 각각의 장단점은?" — 선택지 비교
  - **3단계:** 결정된 내용을 하나의 설계도로 통합
  - **4단계:** 설계도대로 시공하고 검수
- **💡 Point box:** 이 파이프라인이 Cocrates 구조 기반 생성의 핵심 엔진이다
