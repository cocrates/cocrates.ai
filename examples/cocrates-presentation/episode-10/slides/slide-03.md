# Slide 03: 🏗️ "그냥 작성해줘"의 실체 — 아무렇게나 작성해줘

## Key Message
"그냥 작성해줘"는 AI에게 "아무렇게나 작성해줘"라고 방임하는 것과 같다. 세 가지 문제를 인식하라.

## Content
- **Section badge (상단):** 🏗️ 2.1 "그냥 작성해줘"의 문제
- **비유 박스 (callout):** 집을 지어달라는 요청에 건축가가 아무것도 묻지 않고 벽돌부터 쌓는 상황
- **Problem 1 (❌):** 논리와 일관성의 붕괴 — 문맥상 그럴듯한 문장 나열, 앞뒤 충돌
- **Problem 2 (❌):** 설명 책임(Accountability)의 상실 — "왜 이 흐름이죠?"에 답 불가
- **Problem 3 (❌):** 검토 불가능한 블랙박스화 — 설계도(기준) 없음 → 검증 자체 불가
- **💡 Point box:** "그냥 써줘"는 AI에게 "아무렇게나 써줘"라고 방임하는 것과 같다

## Visual Elements
- 세 문제는 각각 아이콘 + 짧은 설명의 카드 형태 (빨간색 악센트)
- "그냥 써줘" → "아무렇게나 써줘" 화살표 변환 시각화

## Layout Notes
- Left/Upper: 비유 + 문제 카드
- Right/Lower: 포인트 박스 (강조)
