# Slide 04: 🎯 Kernel — 핵심 목적을 단 한 문장으로 압축

## Key Message
Kernel은 스킬의 헌법이다. Cocrates의 질문 3종 세트를 통해 모호한 요청을 단 한 문장으로 압축한다.

## Content
- **Cocrates의 질문 3종 세트:**
  1. "어떤 종류의 보고서를 주로 쓰실 건가요?"
  2. "주요 독자는 누구인가요?"
  3. "산출물은 어떤 파일 형태로 저장되나요?"
- **사용자의 응답:** "일반적인 문서, 누구나 독자, .md 파일"
- **Kernel 수립 (스킬의 헌법):**
  > "이 스킬은 Markdown 형식의 설명문/보고서를 검토 가능한 단계를 거쳐 생성하도록 돕는다."
- **포인트:** 이 한 문장이 모든 결정의 기준이 된다

## Visual Elements
- Top: Three question bubbles with arrows pointing down
- Middle: User response box
- Bottom: Kernel statement highlighted in a box with "⚖️ 헌법" label
- Flow diagram: Question → Answer → Kernel

## Layout Notes
- Vertical flow: questions → response → kernel
- Kernel box should be most prominent (highlighted/bordered)
- Use distinct colors for AI questions vs user responses
