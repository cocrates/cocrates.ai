# Slide Content Specification — Slide 4

## Slide 4: 🔍 산출물 5대 프리즘 — 소설 창작으로 이해하기

**Section:** 본론 1 — 메타-스킬의 정체성
**Type:** 개념/프레임워크
**Time:** ~1분 30초

### Visual Design
- **Background:** 시리즈 공통 템플릿
- **Layout:** 5개 카드 레이아웃
  - 상단: 제목 "산출물 5대 프리즘" (32pt)
  - 중단: 5개 카드를 가로 또는 2+3 배열로 배치
  - 각 카드: 아이콘 + 구성 요소명 + 소설 예시 설명
  - 하단: "같은 프리즘, 다른 산출물" 연결 화살표 + 💡 포인트

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 🔍 산출물 5대 프리즘 (Artifact Components) | 상단 |
| 카드 1 | 📋 **과업과 제약** — 장르(판타지/미스터리), 분량, 마감일, 출판사 가이드라인 | 좌상단 |
| 카드 2 | 🎨 **맥락과 규칙** — 세계관(중세/현대/미래), 매직 시스템, 톤, 언어 스타일 | 우상단 |
| 카드 3 | 🧩 **개체 (등장인물)** — 주인공, 조연, 악당, 장면, 플롯 포인트, 테마 | 중앙/좌하단 |
| 카드 4 | 📍 **공간과 배치** — 챕터 구성, 1/2/3막 구조, 페이지 레이아웃 | 중앙/중단 |
| 카드 5 | 🔗 **구조와 흐름** — 발단→전개→위기→절정→결말 (5막 구조), 서브플롯 | 중앙/우하단 |
| 연결부 | "같은 프리즘, 다른 산출물" — 블로그 시리즈 예시 연결 | 하단 |
| 💡 포인트 | "소설이든 보고서든 게임이든 — 이 5대 프리즘으로 모든 산출물 유형을 분석할 수 있다" | 최하단 박스 |

### Layout Notes
- 5개 카드는 각각 고유 아이콘과 색상으로 구분.
- 카드 크기는 균일하게. 각 카드 내에서 구성 요소명은 굵게, 소설 예시는 작은 글씨로.
- 하단 연결부는 블로그 시리즈의 5대 프리즘 매핑을 간략히 표시.
