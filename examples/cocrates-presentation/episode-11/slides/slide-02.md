# Slide Content Specification — Slide 2

## Slide 2: 학습 목표

**Section:** 도입
**Type:** 학습 목표
**Time:** ~1분 30초

### Visual Design
- **Background:** 동일 템플릿 (밝은 톤으로 전환 가능)
- **Layout:** 3열 그리드
  - 상단: "오늘 알게 될 3가지" 제목
  - 3개 컬럼, 각각 아이콘 + 목표

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 🎯 오늘 알게 될 3가지 | 상단 중앙 |
| 목표 1 | 🔄 Cocrates가 요청을 받으면 어떤 스킬을 쓸지 **라우팅**하는가 | 좌측 컬럼 |
| 목표 2 | 🚦 **spec-driven-generation**이 spec이 없을 때 어떻게 **spec 작성을 유도**하는가 | 중앙 컬럼 |
| 목표 3 | 🛠️ 4개 스킬 각각의 **핵심 원칙과 절대 금지 사항** | 우측 컬럼 |
| 도입 질문 | "Ep10의 파이프라인 원리 — 실제 코드로는 어떻게 구현되어 있을까?" | 발표자 구두 전달 |

### Visual Notes
- 각 컬럼 하단에 작은 아이콘 또는 일러스트레이션 추가 가능
- 목표 3은 특히 강조 (나머지 3개 슬라이드의 프리뷰 역할)

### Transitions
- **In:** 콘텐츠가 위에서 아래로 페이드 인
- **Out:** 다음 슬라이드로 전환
- **Notes:** 1분 30초. 각 목표당 20초, 도입 질문 30초.
