# Presentation Definition — Episode 14

## Topic
Cocrates 사용자 선언문 — AI 주권을 선언하는 7가지 계명

## Purpose
드디어 시리즈의 마지막 에피소드. 지금까지 13개 에피소드를 통해 우리는 Cocrates Harness의 설치, 실습, 구조, 그리고 진화까지 모든 것을 배웠다.

이 에피소드에서는 그 여정을 **7계명의 사용자 선언문**으로 집약한다. 이는 시스템을 향한 명령어가 아니라, **AI의 범람 속에서 인간 지성의 영토를 지키겠다는 사용자 자신과의 엄격한 약속**이다.

구체적으로 다음을 다룬다:
1. **Cocrates 프레임워크를 관통하는 3대 철학** — The unexamined code / I know nothing / Harness Ignorance
2. **사용자 선언 7계명** — 설계와 검증의 원칙부터 인지적 투쟁과 열린 공유까지
3. **도구(Harness)와 의지(Manifesto)의 결합** — 구조적 제어 도구 + 인간 의지 = 무한한 능력의 확장

## Audience
- **Background:** 소프트웨어 개발자(초급~중급), AI 도구를 업무에 활용하는 지식 노동자
- **Knowledge level:** Ep1~13 전체 시리즈를 통해 Cocrates의 모든 것을 학습한 상태
- **Interests/expectations:** 시리즈 전체를 관통하는 원칙을 선언문 형태로 마무리하여, 단순한 지식 습득을 넘어 **자신의 AI 사용 철학**을 정립하고자 함

## Logistics
- **Time limit:** 13분 (13슬라이드, ~1분/슬라이드)
- **Format:** 시리즈 마지막 발표 세션 (in-person)
- **Tone/style:** Part 4(결론) — 격려와 다짐의 톤, 선언적 어조. 시리즈 전체를 마무리하는 엄숙하면서도 힘찬 분위기.

## Success Criteria (발표자 관점 — "전달에 성공했는가?")
1. **3대 철학 전달** — 청중이 Cocrates를 관통하는 3대 철학(The unexamined code / I know nothing / Harness Ignorance)을 자신의 말로 설명할 수 있다
2. **7계명 체화** — 청중이 7계명 중 최소 3개 이상을 기억하고, 이를 자신의 업무 상황에 적용할 수 있다
3. **실천 선언 형성** — 청중이 발표 후 스스로에게 "AI를 대하는 나의 원칙은 무엇인가?"라는 질문을 던지게 된다

## Constraints & Taboos
- 7계명을 단순 나열하지 않고, 시리즈 전체 여정과 연결지어 의미를 부여할 것
- Part 4의 스타일 — 격려와 다짐의 톤, 선언적 어조 유지
- 시리즈 전체를 마무리하는 에피소드이므로, 새로운 개념이나 기술적 내용은 금지
- 마지막 인용문("The unexamined code is not worth generating")으로 시리즈 전체를 원형적으로 마무리
