# Slide Content Specification — Slide 3

## Slide 3: Cocrates를 관통하는 3대 철학

**Section:** 본론 1 — 3대 철학
**Type:** 개념 설명
**Time:** ~1분 30초

### Visual Design
- **Background:** 시리즈 공통 템플릿
- **Layout:** 3개의 수평 카드 배열
  - 상단: 제목 — "🏛️ Cocrates를 관통하는 3대 철학"
  - 중앙: 3개 카드 가로 배열 (카드 간 연결선 또는 화살표로 "하나의 프레임워크" 표현)
  - 하단: 💡 포인트 요약

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 제목 | 🏛️ Cocrates를 관통하는 3대 철학 | 상단 |
| 카드 1 🔍 | **"The unexamined code is not worth generating."** — 검토되지 않은 산출물은 생성할 가치가 없다 | 첫 번째 카드. 돋보기 아이콘. |
| 카드 2 🤔 | **"I know I know nothing."** — 나는 내가 아무것도 모른다는 것을 안다 | 두 번째 카드. 물음표 아이콘. |
| 카드 3 🛠️ | **"Harness Ignorance."** — 당신의 무지를 통제하고 제거하라 | 세 번째 카드. 공구 아이콘. |
| 포인트 | **검토하라. 질문하라. 개선하라.** — 이것이 Cocrates의 3대 철학이자 원칙이다. | 하단 강조 상자 |

### Visual Elements Description
- 3개 카드: 동일한 크기와 스타일. 각 카드 상단에 큰 아이콘, 중앙에 철학 문장(큰 따옴표), 하단에 설명.
- 카드 간 연결선: 점선 또는 실선 화살표로 연결되어 "하나의 흐름"임을 표현.
- 하단 포인트: 밝은 강조색 배경 상자.

### Transitions
- **In:** 이전 슬라이드에서 푸시
- **Out:** 다음 슬라이드로 푸시
- **Notes:** 각 카드를 하나씩 가리키며 설명. 철학의 무게감을 살리기 위해 약간의 멈춤.
