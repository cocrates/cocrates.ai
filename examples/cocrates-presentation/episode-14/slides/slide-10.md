# Slide Content Specification — Slide 10

## Slide 10: 제6계명 — 언제나 최적의 구조를 갈망한다

**Section:** 본론 2 — 7계명
**Type:** 선언 (계명)
**Time:** ~1분 10초

### Visual Design
- **Background:** 시리즈 공통 템플릿 (밝고 진취적인 톤)
- **Layout:** 단일 계명 집중형
  - 상단: 계명 번호 — "📜 Cocrates User Manifesto (6/7)"
  - 중앙: 계명 문장 (큰 텍스트) + 영문
  - 중앙 하단: 3개 칼럼 (왜 중요한가 / 실천 방법 / 검증 기준)
  - 하단: 💡 포인트

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 계명 번호 | 📜 Cocrates User Manifesto (6/7) | 상단, 작게 |
| 계명 문장 | **"언제나 최적의 구조를 갈망한다"** ✅ | 중앙, 대형 텍스트 |
| 영문 | I always strive for the optimal. | 계명 아래, 이탤릭 |
| 왜 중요한가 | • 첫 결과물에 안주하는 순간 성장은 멈춘다 | 좌측 칼럼 |

| 검증 기준 | • 최적의 결단만이 헌법(Spec)으로 안착할 자격을 얻는다 | 우측 칼럼 |

### Visual Elements Description
- 계명 문장: ✅ 아이콘 초록 계열.
- ADR 트레이드오프 저울 이미지. "첫 결과물 → 최적 구조" 발전 화살표.
- 하단 포인트: 초록 계열 강조 배경.
