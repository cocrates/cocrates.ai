# Slide Content Specification — Slide 5

## Slide 5: 제1계명 — 설계 없는 요구를 거부한다

**Section:** 본론 2 — 7계명
**Type:** 선언 (계명)
**Time:** ~1분 10초

### Visual Design
- **Background:** 시리즈 공통 템플릿 (약간 어두운 톤)
- **Layout:** 단일 계명 집중형
  - 상단: 계명 번호 — "📜 Cocrates User Manifesto (1/7)"
  - 중앙: 계명 문장 (큰 텍스트) + 영문
  - 중앙 하단: 3개 칼럼 (왜 중요한가 / 실천 방법 / 검증 기준)
  - 하단: 💡 포인트

### Content Elements
| 요소 | 내용 | 비고 |
|------|------|------|
| 계명 번호 | 📜 Cocrates User Manifesto (1/7) | 상단, 작게 |
| 계명 문장 | **"설계 없는 요구를 거부한다"** ❌ | 중앙, 대형 텍스트 |
| 영문 | I do not blindly request. | 계명 아래, 이탤릭 |
| 왜 중요한가 | • 목적과 구조 없는 요청은 인지적 나태함의 산물 | 좌측 칼럼 |
| 실천 방법 | • 요청 전 3가지 자문: 목적? 제약? 구조? | 중앙 칼럼 |
| 검증 기준 | • 산출물을 아키텍트로서 설명 가능해야 함 | 우측 칼럼 |
| 포인트 | **좋은 설계와 좋은 질문만이 가치 있는 답변을 만든다** | 하단 강조 상자 |

### Visual Elements Description
- 계명 문장: 큰 글씨, 중앙 정렬. ❌ 아이콘 빨간색 강조.
- 3개 칼럼: 동일 너비 가로 배열.
- 하단 포인트: 강조색 배경 상자.
