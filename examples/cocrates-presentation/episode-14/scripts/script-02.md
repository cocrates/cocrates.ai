# Script: Slide 02 — 학습 목표

**Time:** ~1분

**[SLIDE 02: 세로 리스트 — 🏛️📜⚖️ / 하단: Ep1→Ep14 시리즈 여정]**

오늘 우리가 함께 알아갈 세 가지 목표입니다.

(슬라이드를 가리키며)

**첫째.** Cocrates Harness를 관통하는 3대 철학.
"The unexamined code is not worth generating." — "I know I know nothing." — "Harness Ignorance."
이 세 문장이 어디서 왔고, 왜 중요한지 되짚어보겠습니다.

**둘째.** 사용자 선언 7계명.
설계와 검증의 원칙부터 인지적 투쟁과 열린 공유까지. 7가지 원칙을 하나씩 선언합니다.

**셋째.** 도구와 의지의 결합.
Cocrates Harness라는 도구, 그리고 오늘 선언하는 7계명이라는 의지.
이 둘이 만날 때 어떤 일이 일어나는지 — 그것이 오늘의 최종 메시지입니다.

(청중을 보며)

주의할 점이 하나 있습니다. 오늘은 '새로운 것을 배우는' 에피소드가 아닙니다.

오늘은 **'지금까지 배운 모든 것을 하나로 묶는'** 에피소드입니다.

14개의 에피소드가 하나의 선언문으로 수렴하는 순간을, 함께 지켜봐 주십시오.
