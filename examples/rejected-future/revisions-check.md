## 사용법

- 각 체크 항목의 `[ ]`를 `[x]`로 변경하여 완료 표시
- **Layer별로** 검증 — Layer 1 완료 후 Layer 2, Layer 2 완료 후 Layer 3
- **파일 경로**를 클릭하여 해당 파일에서 실제 내용을 확인
- **"확인 방법"** 열의 지시에 따라 검증
