# 한국대학병원 (Korea University Hospital)

> 서울특별시 **종로구**에 위치한 종합병원. 이야기의 거의 모든 장면이 이곳에서 펼쳐진다.
> 한국 최고 병원은 **ACE의료원**이며, 한국대학병원은 그에 뒤쳐진 2군 병원이다.
> 외과 수술 로봇 1대(SR-03)에 의존해야 하는 열세 속에서, 병원장 김현수는 추가 로봇 배정을 위해 정부와 로비전을 벌이고 있다.

---

## 건물 개요

| 항목 | 내용 |
|------|------|
| **위치** | 서울특별시 종로구 (종로경찰서 관할) |
| **규모** | 본관 6층(지상 4층 + 지하 2층) + 별관 2층 + VIP 센터 |
| **병상 수** | 약 800병상 |
| **로봇 인력** | 인간형 로봇 SR-03(외과), RE-07(정비), 비인간형 로봇 다수 |
| **기술팀** | 박동훈(팀장), 나유진, 조민석, 김보람, RE-07 |
| **병원 서열** | 한국 의료계 2~3위권. **1위는 ACE의료원**(외과 수술 로봇 5대 보유) |
| **병원장 숙원** | 추가 외과 수술 로봇 배정 — 매년 국회 로비 중 |

## 1. 옥상 (Rooftop)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 본관 최상부 (4F → 옥상 출입문) |
| **접근성** | 의료진 전용. 환자 출입 제한 (일부 회복기 환자 허용) |
| **분위기** | 개방적이고 적막함. 바람 소리와 도시 소음이 섞임 |

### 특징

- **좁지만 의미 있는 공간**: 건평의 절반가량만 데크로 되어 있고, 나머지는 기계실·냉각탑이 차지.
- **벤치**: 두 개의 낡은 나무 벤치 — 한쪽은 흡연 구역 (사실상 흡연이 금지되었지만 묵인), 다른 쪽은 그냥 앉아 있기.
- **난간**: 이중 난간 — 안전용과 기대어 서기용. 난간 너머로 서울 도심이 펼쳐짐.
- **조명**: 형광등 대신 주변 건물의 불빛과 간판 빛만이 유일한 조명.

### 주요 장면

- 강재민이 수술 전후 혼자 서서 생각하는 장면
- 이성모 과장이 재민을 불러 "로봇을 믿지 마라"고 조언하는 장면
- 재민과 유진이 병원장 결정에 대해 이야기하는 장면 (첫 진지한 대화)
- 밤, 재민이 SR-03의 수술 데이터를 보고 난 후 담배를 피우며 생각하는 장면

### 관련 캐릭터

| 캐릭터 | 관계 |
|--------|------|
| 🟦 강재민 | **단골 이용자**. 수술 전후·중요한 결정 전에 항상 오름 |
| 👴 이성모 과장 | **원조 단골**. 40년째 이 공간을 사색의 장소로 씀. 재민과 이 곳에서 만남 |
| 🟩 나유진 | **간헐적 방문**. 재민을 만나러 올라옴. 자신의 공간은 별관 |
| 🤖 SR-03 서준 | **방문하지 않음**. 하지만 재민이 여기 있을 때를 데이터로 알고 있음 |

### 동선

```
옥상 ←→ 4F 병동 ← 계단/엘리베이터 → 각 층
```

## 2. 병동 — 4F (General Ward)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 본관 4F 전면 |
| **병상** | 약 120병상 (2인실·4인실 혼합. 소수 1인실) |
| **분위기** | 조용하고 차분함. 그러나 밤이 되면 요동침 — 환자의 통증, 보호자의 불안 |

### 특징

- **간호사 스테이션**: 병동 중앙. 한미영 수간호사의 지휘 아래 24시간 가동.
- **회진 동선**: 매일 아침 재민(또는 담당의)이 전실 순회 — 인간 의사와 SR-03이 함께 도는 날이 증가하는 추세.
- **환자 면담실**: 병동 끝 유리칸막이 공간 — 보호자와의 면담, 사망 통보 등 무거운 장면이 주로 이곳에서.
- **자동 약품 관리 시스템**: AI가 약품을 관리하지만, 최종 투약은 인간 간호사의 확인 필요.

### 주요 장면

- 재민의 아침 회진 — 각 병실의 환자 상태 체크
- SR-03이 데이터 기반으로 환자 상태 변화를 예측해 재민에게 보고
- 테러리스트에 의한 로봇 피해 환자가 이곳에 입원
- 한미영 수간호사가 회진 후 재민에게 조언

### 관련 캐릭터

| 캐릭터 | 관계 |
|--------|------|
| 🟦 강재민 | **회진 주체** |
| 🤖 SR-03 서준 | **회진 보조** — 함께 도는 날 인간보다 데이터를 더 많이 읽음 |
| 👩 한미영 수간호사 | **이 층의 실질적 지배자** |
| 👴 이성모 과장 | 가끔 회진 — 옛 제자들을 만나러 |

## 3. 중환자실 — 3F (ICU)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 본관 3F, 수술동과 직접 연결 |
| **병상** | 약 20병상 (1인용 격리) |
| **분위기** | 기계음이 끊이지 않는 공간. 모니터 소리, 인공호흡기 소리, 알람 — 생명 유지의 소음 |

### 특징

- **수술동과 직통**: 수술이 끝난 중증 환자는 곧바로 이곳으로 이송.
- **24시간 상주 의료진**: 전담 중환자 전문의 + 전담 간호사.
- **방문 제한**: 1일 1회, 30분. 유리문 너머로만 면회 가능.
- **로봇 접근**: SR-03의 데이터 분석이 중환자 상태 예측에 결정적 역할을 하는 공간. 그러나 인간 의사의 최종 판단 필수.

### 주요 장면

- 재민이 집도한 수술 후 환자가 이곳으로 — 재민이 밤새 지켜보는 장면
- SR-03의 예측 데이터와 인간 의사의 판단이 충돌하는 장면
- 중환자실 앞 복도에서 보호자와 의사의 면담

### 관련 캐릭터

| 캐릭터 | 관계 |
|--------|------|
| 🟦 강재민 | **수술 후 환자 상태 가장 걱정하는 사람** |
| 🤖 SR-03 서준 | **데이터 예측 제공** |
| 👩 한미영 수간호사 | ICU와 수술동 사이를 오가며 조율 |

## 4. 수술동 — 3F (Operating Room)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 본관 3F, 중환자실과 인접 |
| **수술실 수** | 8실 (인간 의사 전용 4실, AI 보조 2실, 로봇 공동 2실) |
| **분위기** | 냉랭하고 위생적. 시간이 다르게 흐르는 공간 |

### 특징

- **수술실 구분**:
  - 인간 의사 전용실: 전통적인 방식의 수술. 재민이 가장 선호.
  - AI 보조실: AI가 영상 분석·데이터 제공. 인간 의사가 집도.
  - 로봇 공동 수술실: SR-03과 인간 의사가 함께. 재민이 가장 꺼리지만 통계적으로 성공률이 가장 높은 공간.
- **수술 전 브리핑룸**: 각 수술 전 팀 미팅 — 재민과 SR-03의 의견 차이가 가장 선명하게 드러나는 공간.
- **수술 후 회복실**: 마취에서 깨어나는 환자. 간호사 1:1 상주.
- **수술 기록실**: 모든 수술 데이터가 기록·보관됨. SR-03의 데이터와 재민의 필기 노트가 함께 보관됨.

### 주요 장면

- 에피소드 오프닝 — 재민과 SR-03의 첫 수술 호흡
- 재민이 로봇 공동 수술실을 배정받고 불만을 표시하는 장면
- SR-03이 수술 중 예상치 못한 변수를 데이터로 예측해 재민을 도움
- 수술 실패 — 재민이 수술실에 홀로 남아 오랜 시간을 보내는 장면

### 관련 캐릭터

| 캐릭터 | 관계 |
|--------|------|
| 🟦 강재민 | **주 사용자**. 이 공간이 그의 전장 |
| 🤖 SR-03 서준 | **공동 사용자**. 수술실에서는 재민과 동등함 (데이터로) |
| 👴 이성모 과장 | **이전 주인**. 은퇴 직전이라 이 공간을 점점 덜 방문 |
| 👩 장서영 전공의 | **견습**. 수술 참관 |
| 👨 오세영 전공의 | **혼란**. 이 공간에서 갈등이 최대화됨 |
| 👩 한미영 수간호사 | **운영자**. 수술실의 모든 것을 통제 |

### 동선

```
수술동 ←→ 중환자실 (직통)
    │
    ↓
병동 (회복 후 이송)
```

## 6. 로비·접수·응급실 — 1F (Lobby / ER)

### 기본 정보

| 항목 | 내용 |
|------|------|
| **위치** | 본관 1F 전면 + 우측 별도 응급구역 |
| **분위기** | 24시간 불이 꺼지지 않는 전쟁터 |

### 특징

- **접수처**: AI 키오스크와 인간 직원 공존. 키오스크가 대부분의 단순 접수를 처리하지만, 복잡한 민원·응급은 인간 직원이 대응.
- **로비**: 높은 천장, 중앙 벤치. 병원의 모든 사람이 반드시 지나가는 허브.
- **응급실**: 모든 극적 장면의 시작점. 구급차 도착, 응급 분류, 초동 처치.
- **로봇 안내 도우미**: 비인간형 안내 로봇 — 질문에 답하고 목적지까지 안내. 그러나 한계가 명확해 인간 직원의 도움을 필요로 하는 상황이 자주 발생.
- **편의점·카페**: 로비 한쪽. 직원과 보호자의 유일한 위로.

### 주요 장면

- 응급실로 실려 들어오는 환자 — 재민이 급히 내려와 수술 준비
- 재민과 유진이 로비에서 우연히 마주치고 어색한 인사
- 병원장 김현수가 로비에서 사업가를 맞이

### 관련 캐릭터

| 캐릭터 | 관계 |
|--------|------|
| 🟦 강재민 | **통과 공간** — 응급 호출 시 가장 먼저 달려감 |
| 🟩 나유진 | **통과 공간** — 별관으로 가기 위해 반드시 지나감 |
| 🤖 SR-03 서준 | **통과 공간** — 로비는 수술실과 별관 사이의 경로 |
| 👨 김현수 병원장 | **접견 장소** — 외부 인사를 이곳에서 맞음 |
| 🧨 테러리스트 | **침투 지점** |

## 8. 서버실 — 지하 2F (Server Room)

### 기본 정보

|------|------|
| **위치** | 본관 지하 2F 최심부 |

### 특징

## 9. 로봇 센터 — 별관 1F (Robot Center)

> 자세한 명세는 별도 파일 참조: `locations/로봇-센터.md`
>
> **감각적 시그니처** (다른 에피소드에서 일관되게 사용):
> - **냄새:** 오존 + 윤활유 + 금속 + 온풍 — 처음 방문하는 사람은 코를 찌르는 이 냄새에 눈살을 찌푸리지만, 유진에게는 '집' 냄새
> - **소리:** 저주파 윙윙거림(충전 스테이션) + 간헐적 공구 소리 + 데이터 키보드 타이핑

> - **조명:** 청색 LED 패널 — 유진의 작업대만 유일하게 노란 책상 램프
> - **온도:** 연중 20~22°C — 항상 약간 서늘함. 유진은 항상 가디건을 걸침

- **로봇 배치**: VIP 환자에게는 SR-03의 진료가 우선 배정됨 — 예산과 정치의 영역.

- 사업가가 병원장을 만나 로비하는 장면

- VIP 환자의 수술을 두고 재민과 병원장 사이의 갈등

## 전체 맥락 요약

한국대학병원은 인간과 로봇의 공존이 가장 집약적으로 나타나는 공간이다. 각 층과 공간은 캐릭터들의 관계와 갈등을 물리적으로 구현한다:

- **상층부(옥상~4F)**: 인간 의사의 전통적 영역. 사색과 회복.
- **중층(3F~2F)**: 인간과 로봇의 경계. 가장 치열한 경쟁과 협력의 현장.
- **하층(1F~지하)**: 혼란과 시스템의 영역. 모든 것이 시작되고 연결되는 곳.
- **별관**: 로봇의 영역. 인간과 로봇이 가장 평화롭게 공존하는 공간.
- **VIP 센터**: 권력과 자본의 영역. 병원의 가장 어두운 단면.
