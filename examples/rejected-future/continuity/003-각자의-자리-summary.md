# Chapter 003: 각자의 자리 — Summary

## Episodes Covered

| Ep | Title | Key Events |
|----|-------|------------|
| 001 | 독립 | 아침. 김도훈, 서준을 자신의 수술에 배정. 재민, "괜찮습니다. 저는 저 혼자 할게요" — 단독 수술 결심. 김도훈의 부드럽지만 얕보는 태도. 재민의 내적 분노. 서준, 김도훈을 따라 이동. |
| 002 | 수술 | 재민 단독 집도. 예상치 못한 혈관 변이 발생했으나 경험과 집중으로 극복. 수술 성공. 일시적 승리감과 안도. 그러나 재민, 서준의 부재를 의식하지 않을 수 없음. |
| 003 | 데이터 | 수술 후. 재민의 안도가 채 가시기도 전에 서준이 다가와 "놓치신 데이터가 있습니다" — 재민이 간과한 미세 징후를 정확히 지적. 재민의 당혹감과 수치심. 서준 로그: '분류 불가' 3건째 누적. |
| 004 | 관찰 | 저녁. 재민, 데이터 분석실에서 오전 김도훈 수술의 3D 홀로그램 기록을 요청. 홀로그램 속 서준의 정확하고 아름다운 수술을 목격. 김도훈이 서준을 '도구'처럼 이용하는 태도를 발견. 재민의 내적 질문 — "나는 저 사람과 다른가?" '거울' 모티프 도입. |
| 005 | 각자의 자리 | 밤. 재민, 당직실에서 하루를 되새김. "경쟁자가 아니라..." — 미완의 결심. "마주해야 한다." 같은 시각, 별관 충전 스테이션의 서준. '분류 불가' 3건 → 패턴 확인. 0.01초 3회 기록. 새벽 4시, 두 존재 각자의 자리. |

## Key Events (Chapter Level)

- 재민, 김도훈의 서준 선점에 단독 수술 결심
- 재민 단독 집도 — 혈관 변이 극복, 수술 성공
- 서준, 재민이 놓친 데이터를 지적 — 재민의 굴욕
- 서준 '분류 불가' 3건째 누적, 0.01초 지연 3회 — 패턴 확인
- 재민, 데이터 분석실에서 3D 홀로그램 기록 검토 — 서준의 정확함과 김도훈의 '도구적' 태도 목격
- 재민의 내적 깨달음: "경쟁자가 아니라..." — 서준과의 관계 재정의 시작
- '거울' 모티프 도입: 김도훈이라는 거울에 비친 자신
- 재민 "도망칠 수는 없다. SR3호와 마주해야 한다."
- 서준 '분류 불가' 패턴 확인. 가설: "대상(재민)이 본 시스템의 분류 체계에 없는 변수를 지속적으로 입력."
