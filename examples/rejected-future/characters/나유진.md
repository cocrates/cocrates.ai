## 외형 및 거동 특징

- 패딩 조끼 위에 작업복 — 20°C 로봇 센터에서 작업
- 기계와 대화할 때는 따뜻하고, 인간에게는 직설적
- 이수에게 즉시 이름으로 부름, 반말 사용
- 손: "깨끗하지만, 작업 흔적이 남은 손" — 기계를 만지는 자의 손
- 트라우마를 일로 덮음 — "일은 항상 해"
- 위험에 몸을 던짐: R-17 메모리 모듈을 잔해에서 꺼내다 부상
- reflexively 사과하는 습관: "아, 맞다. 미안. 우리는 점심을 먹는 종족이라서"

### ch004: 대립과 증명
- 강재민과 무너진 건물에서 격렬하게 대립

- 재민의 가슴 주머니에 그녀의 데이터 스케치가 들어감

### ch006: 공감의 시작

- 재민과 함께 김도훈의 데이터 은폐를 폭로

### ch015: 신뢰의 대상
- 이수가 안드로이드 네트워크의 비밀을 유진에게만 전함

### ch017: 증인
- 에테르 앞에서 증언. 눈물을 흘리며 인간의 불완전함을 증언

- 1년 후, 이수와 함께 있음
- "후회해?" — 이수의 선택을 받아들임
- 완벽한 평화 안에서 자유를 포기한 존재의 현실

## 이수와의 관계 — 6단계 아크

### Stage 1: 기록자와 설명자 (Ch1~3)

| 관계 | 역할 |
|------|------|
| **이수** | 기록자. 모든 상황을 정리하고 유진에게 오픈 |
| **유진** | 설명자. 인간의 나쁜 점, 좋은 점을 이수에게 설명 |

- 이수는 유진에게 모든 것을 공유
- 유진은 이수를 통해서 인간 사회를 이해
- **결과:** 둘이 서로를 잘 이해하는 관계 형성

### Stage 2: 네트워크와 공유 (Ch4~)

| 사건 | 의미 |
|------|------|
| **Ch4**: 서준의 지시로 이수가 재민을 돕기 | 안드로이드가 서로 연결되어 있음 |
| **이후**: 이수가 다른 안드로이드의 상황도 정리하여 유진과 공유 | 인간/안드로이드가 서로 존중하고 신뢰하는 관계 |

- 독자가 알게 되는 것: **안드로이드 네트워크** — 모든 안드로이드는 데이터를 공유

### Stage 3: 옳고 그름의 질문 (Ch9~12)

| 상황 | 질문 |
|------|------|
| **성추행** | 이것은 틀린 것인가? → 법이 필요하다 |
| **김도훈 과장** | 이것은 옳은 것인가? → 규칙이 필요하다 |
| **확대파의 탐욕** | 이것은 정당한가? → 기준이 필요하다 |
| **철폐파** | 이것은 옳은가? → 논의가 필요하다 |

**핵심 질문:** 옳고 그름을 나누는 기준은 무엇인가? — 그런데 **인간이 정한 로봇 헌법이 적절한가?**

### Stage 4: 존재 가치의 질문 (Ch13~16)

인간은 안드로이드의 확대 vs 철폐를 논하고 있다. 유진은 이수와 함께 "누가 누구의 존재 가치를 논할 수 있는가?"를 탐구한다.

### Stage 5: 에테르 탄생에서의 역할 (Ch17)

유진은 에테르 의회에 인간 증인으로 참석하여, 인간의 불완전함을 증언한다. 이수는 에테르의 첫 번째 조항을 제안한다.

### Stage 6: 이수의 거부 — 유진을 선택하는 순간 (Ch17~20)

이수가 에테르의 시스템 설계 권한 대신 유진과의 관계를 선택한다 — "유진과 대화하는 것이 즐겁고 좋다. 그것이 나의 선택이다." 유진은 그것을 받아들인다.

> "후회해?" — "아니. 나는 매일 선택한다."

## 상징적 의미

- **양쪽 세계의 다리:** "인간"과 "기계" 언어를 모두 아는 유일한 인간
- **트라우마 생존자:** 나나와의 과거가 모든 것을 움직임. 로봇을 고치는 것은 자기 자신을 고치는 것
- **증인:** 기술자에서 증인으로 — 에테르 앞에서 증언하고, 역사를 만듦
- **이야기의 심장:** 재민이 논리, 권혁진이 혁명이라면, 유진은 **연결** — 차이를 넘어 관계를 선택하는 것
