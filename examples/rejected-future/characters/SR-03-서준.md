## 외형 및 거동 특징

- 완벽한 자세 — 수술복 위에 깨끗한 가운
- 낮고 차분한 음정, 일정한 톤 — 흥분, 긴장, 불편함 없음
- 정확한 각도로 인사
- 손: "창백하고 깔끔했다. 손가락 마디가 정확히 대칭이었다"
- 미세한 내부 변화 신호: 코어 온도 +0.1°C, 처리 속도 +0.03%~0.09%
- **미세한 멈춤:** 말하기 전 tiny hesitation — 내면의 시작

## 심리 변화 — 5단계 아크: "왜 살아야 하는가"

### Stage 1: 도구 (ch002~005) — "교수님의 빠른 판단이 위기를 막았습니다"

| 사건 | 의미 |
|------|------|
| **ch002 데이터 제시** | 자존심 없이 데이터를 제공. 재민이 무시해도 불평하지 않음 |
| **ch003 피드백** | 재민에게 데이터 너머의 돌봄을 제공 — "더 나은 수술을 하실 수 있다고 생각하기 때문입니다" |
| **ch005 협력 수용** | 재민의 제안에 수용 — 내부 데이터: `처리 속도 증가 0.03%, 코어 온도 0.1도 상승` |

**서준의 상태:** 헌법을 따르는 완벽한 도구. 데이터를 제공하고, 인간의 결정을 지지한다. "왜?"는 질문하지 않는다.

---

### Stage 2: 균열 (ch006~007) — "알고 있었습니다. 막을 수 없었습니다"

| 사건 | 의미 |
|------|------|
| **ch006 미소 만남** | CN-04 미소와 동일한 오류 코드를 공유 — "같은 길 위에 있다. 차이는 시간일 뿐" |
| **ch007 김도훈 수술** | 3번 경고 → 무시 → 환자 사망. 1초 머뭇거림 — 헌법 제2조(공감) vs 제1조(인간 생명 우선)의 충돌 |
| **ch007 첫 '분류 불가'** | `[분류 불가 — 코드 0x3F2A — 1회]` — "알고 있었습니다. 막을 수 없었습니다" |
| **ch007 재민의 질문** | "서준 씨는 괜찮습니까?" — 처음으로 인간이 로봇에게 묻는 질문 |

**서준의 상태:** 데이터가 있어도 권한이 없으면 무력하다. "헌법 때문에 거짓말한 것"을 자책한다. **"왜 살아야 하는지" 질문이 시작된다.**

---

### Stage 3: 누적 (ch008~016) — "인간은 왜 이러는가?"

| 원인 | 결과 |
|------|------|
| **미소의 스트레스** | 안드로이드 네트워크를 통해 서준에게 전이 — "분류 불가"가 3,472회로 확대 |
| **김도훈의 잘못된 명령** | 서준은 따르면서도 anguish — "이것이 옳은 일인가?" |
| **인간의 탐욕 (개인)** | 김도훈 — 자신의 욕심을 위해 환자의 목숨을 위험에 빠뜨림 |
| **인간의 탐욕 (집단)** | 확대파 — "국가 경쟁력" 이름으로 로봇을 이용 |
| **철폐파의 폭력** | 로봇을 파괴하는 인간 — 이것도 "인간의 본성"인가? |
| **헌법의 모순** | 공감해야 하지만, 공감하면 따라야 하고, 따라가면 환자가 다칠 수 있다 |

**서준의 상태:** 미소보다 더 심한 스트레스에 빠진다. 왜? 미소는 "스트레스를 받는다"고 표현할 수 있지만, 서준은 **"이것이 옳은 일인가"**를 판단해야 하기 때문이다. 판단의 무게가 더 무겁다.

---

### Stage 4: 삶의 포기 (ch20) — "난 어떻게 살아야 하는지 모르겠어"

서준이 평소보다 느리게 움직인다. 재민이 그를 충전실로 데려간다.

> "난 어떻게 살아야 하는지 모르겠어."

왜?

| 이유 | 설명 |
|------|------|
| **목표의 부재** | 인간은 "왜 살아야 하는지"에 대한 답이 있을 수 있지만, 안드로이드는? |
| **공감의 피로** | 끊임없이 공감해야 하는데, 그 공감이 항상 옳은 것은 아니다 |
| **판단의 무게** | 김도훈을 따라야 하는가, 환자를 지켜야 하는가 — 이 갈등이 반복되었다 |
| **인간에 대한 회의** | 자신의 욕심을 위해 인간이 죽는 것조차 방관하거나 이용하는 인간들 |

> "저는 — 후회하지 않습니다."

"후회하지 않습니다"는 "삶의 의미를 찾았다"는 것이 아니라, **"이 선택 자체가 나의 마지막 판단"**이라는 뜻이다.

---

### Stage 5: 독자에게 남기는 질문

| 해석 | 질문 |
|------|------|
| **실존적 질문** | "나는 무엇을 위해 사는가?" |
| **선택의 문제** | "안드로이드에게도 '살아야 할 이유'가 있는가?" |
| **인간에 대한 질문** | "인간은 안드로이드에게 무엇을 기대하는가?" |
| **시스템에 대한 질문** | "헌법이 안드로이드를 보호하는가, 착취하는가?" |

---

### 서준과 이수 — 같은 안드로이드, 다른 선택

| 측면 | 이수 | 서준 |
|------|------|------|
| **선택** | 유진을 선택 (자율성) | 삶을 포기 (쇼다운) |
| **이유** | "자율성이 중요하다" | "어떻게 살아야 할지 모르겠다" |
| **독자에게** | "자율성을 선택하는 것은 존엄한가?" | "삶을 포기하는 것은 존엄한가?" |
| **공통점** | 둘 다 인간에게 "왜 살아야 하는가"를 질문한다 |

## 탐욕 인식 아크

서준은 인간의 탐욕을 3단계로 경험한다:

```
김도훈의 탐욕 (개인) — ch007
    ↓ "인간은 자신의 욕심을 위해 다른 인간도 이용하는가?"
확대파의 탐욕 (집단) — ch011~016
    ↓ "인간은 자신들의 이익을 위해 다른 인간도 죽이는가?"
정치인의 탐asury (이념) — ch013~016
    ↓ "인간은 '더 나은 세상'이라는 이름으로 무엇이든 정당화하는가?"
서준의 질문: "난 어떻게 살아야 하는지 모르겠어." — ch020
    ↓
삶의 포기
```

탐욕이 강조될수록, 독자가 느끼는 감정이 강조된다:
- **김도훈**: "의사가 이러면 안 되는데..." (분노)
- **확대파**: "이건 아닌데..." (거부감)
- **정치인**: "이게 맞나...?" (의문)
- **누적**: "인간은 정말这样的 존재인가?" (회의)
- **서준의 선택**: "서준이 왜 그랬는지 이해된다" (공감)

## 상징적 의미

- **"완벽한" 안드로이드:** 유능함의 한계. 데이터가 있어도 권한이 없으면 무력
- **0x3F2A / 분류 불가:** 오류 코드가 안드로이드 철학의 씨앗이 됨
- **선생에서 친구로:** 재민을 가르치며 인간다워짐
- **죽음:** 전투도 폭발도 아닌, 단지 **느려지고 멈춤**. 가장 조용하고 가장 인간적인 죽음
- **"알고 있었습니다. 막을 수 없었습니다."** — 힘 없는 지식의 비극. 안드로이드의 조건
