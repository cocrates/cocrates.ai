## 기본소득 (UBI)

| 항목 | 내용 |
|------|------|
| 시행 시기 | 2043년 (현재 2년차) |
| 대상 | 모든 시민 |
| 형식 | 무세금. 정부가 매달 파격적 생활비 지급 (과거 중상위 소득 수준) |
| 효과 | 거리가 한산함. 카페·공원·낚시터에 여가를 즐기는 사람들 |
| 부작용 | 무기력증, 중독, 허무주의 확산. "나는 왜 사는가" |

### 재원 구조

기본소득의 재원은 **안드로이드 로봇의 노동 수입 전액**이다.

```
안드로이드 노동 수입 → 전액 국가 귀속 → 인간에게 기본소득으로 분배
```

안드로이드 로봇의 노동으로 발생하는 모든 수익은 국가에 귀속되며, 이것이 기본소득의 재원이 된다. 안드로이드 로봇自身的 생활비(의류/충전비/주거비)도 모두 국가가 부담한다. 즉, **안드로이드가 벌어들이는 수입이 국가의 수입이 되고, 이것이 인간의 기본소득으로 전환되는 구조**다.

| 항목 | 부담 주체 |
|------|-----------|
| 안드로이드 노동 수입 | 전액 국가 수취 |
| 안드로이드 생활비 (의/식/주) | 국가 부담 |
| 인간 기본소득 | 국가 지급 (안드로이드 수입에서 충당) |
| 인간 세금 | **없음** (무세금) |

### 기본소득의 역설
- 의식주는 완벽히 해결되었으나, 대다수의 인간 계층에서 무기력증, 중독, 허무주의가 확산
- 정부는 삶의 의지를 회복시키기 위해 노동을 적극 권장
- "인간의 일자리 수요를 먼저 파악한 뒤, 남는 공석에만 로봇 수급을 조정"

## 이중 통제 체제의 역설

```
안드로이드 노동 ➔ 정부가 수익 전액 수취 ➔ 정부가 인간에게 파격적 지원금 지급
                                         (무세금, 과거 중상위 소득급 기본소득)
```

### 인위적 노동 장려 정책과 잉여감
파격적인 기본소득으로 의식주는 완벽히 해결되었으나, 일을 하지 않는 대다수의 인간 계층에서 무기력증, 중독, 허무주의("나는 왜 사는가")가 확산된다. 정부는 삶의 의지를 회복시키기 위해 "청소 등 배우지 않아도 할 수 있는 일도 고귀하다"며 노동을 적극 권장하고 교육 기관을 뒷받침한다.

### 의도치 않은 역전 현상 (The Great Inversion)
시스템은 이상적이나, 현장은 다르다. 자본과 대기업들은 인간 노동자를 쓰느니 법망을 우회해서라도 훨씬 똑똑하고 피로를 모르는 안드로이드를 배치해달라고 요구한다.

결국:
- 안드로이드가 팀장·전문가급 상층부 역할을 담당
- 인간들은 로봇이 하기 '아까운' 인위적인 하급 노동에 종사
- 인간은 점점 더 '쓸모없어지고 있다'는 공포에 직면

## 병원의 경제 구조 (한국대학병원)

### 수익 구조
- 1층: 로비, 접수처, 편의점, 약국
- 지하 1층: 주차장, 카페, 편의점
- VIP 센터: 고위층 전용 시설, 최고급 서비스, 로비의 핵심

### 비용 구조
- 안드로이드 로봇 인건비 (국가 부담)
- 첨단 의료 장비 유지비
- 건물 유지비

### 로봇 비용의 비밀
- 기본 유지비: 국가 부담
- 진단·수리 건: 국가 수수료
- 추가 서비스: 인간이 지불
- 성과급: 인간이 지불 (로봇은 국가가 수취)
