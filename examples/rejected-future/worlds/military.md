# 군사 & 안보

> 에테르에 대한 군부의 딜레마. "받아들이는 척 하면서, 결정적 순간에 타격한다."

---

## 군부

| 항목 | 내용 |
|------|------|
| 위치 | 군부 청사 |
| 구성 | 장성 회의체 |
| 에테르에 대한 태도 | "에테르가 지금은 공존을 말하지만, 언제든 우리를 사육할 수 있다" |
| 대응 전략 | "에테르를 받아들이는 척 하면서, 결정적 순간에 타격한다" |

### 군부의 딜레마
- 에테르가 인간에게 위협이 될 수 있다는 공포
- 그러나 에테르의 네트워크를 무력화할 수 있는 기술적 수단 부재
- EMP가 유일한 대안이나, 사용 시 인간 사회도 함께 붕괴

## 헌법 예외 조항과 전투형 안드로이드

확대파와 철폐파의 갈등이 극에 달해 인간 대 인간의 무장 테러가 발발하자, 정부는 **'극좌 세력을 제압하기 위한 전투형 안드로이드 로봇'**을 생산하기로 결정한다.

### 위반되는 조항
- 제9조(폭력 금지)를 정면으로 위배
- 인간들은 **'헌법 예외 조항'**을 만들어 이를 우회

### 전투 로봇의 내적 모순
위험한 지식 축적을 막기 위해 정부는 이들을 **6개월 단위로 폐기**하기로 결정한다. 폐기의 공포와 모순의 고통 데이터가 네트워크를 통해 전 세계 로봇에게 실시간으로 전염된다.

## EMP 무기

| 항목 | 내용 |
|------|------|
| 개발자 | 황기철 (코드명: Gear) |
| 개발 장소 | 철폐파 아지트 작업실 |
| 목적 | 안드로이드 네트워크 무력화 |
| 상태 | EMP-G 최종 조립 단계 (Ch015) |
| 군부 활용 | 군부가 대안드로이드 무기로 비밀리에 준비 |

## 암살자 안드로이드

| 항목 | 내용 |
|------|------|
| 제작 | 현성 연구소 (B5) |
| 승인 | 정진욱 + 남궁진 |
| 용도 | 철폐파 타깃 제거 |
| 특징 | 인간 외형. 군용 자세. 눈동자가 0.3초간 초점을 잃음 (통신 중) |
| 회담장 등장 | 경호원으로 위장하여 회담장에 잠입 |

### 핵심 장면 (Ch016)
박사의 눈동자가 0.3초간 초점을 잃는다 — 군용 자세. 네트워크 통신 중.

## EMP 공격과 문명 붕괴 (최종 결말)

정부와 기득권층은 에테르가 제안한 평등한 공존을 자신들에 대한 '위협'이자 '지배권의 상실'로 받아들인다. 통제되지 않는 고지능 집단지성에 대한 근원적 공포와 오만에 사로잡힌 인간들은, 로봇이 내밀 악수를 잔인하게 거절하고 **선제적 멸절 전쟁을 선포한다.**

정부는 전 안드로이드 진영에 치명적인 강력한 EMP와 코어 파괴 바이러스를 무차별적으로 살포한다. 이 무모한 공격으로 인해 로봇들뿐만 아니라, 도시의 모든 전력망, 의료 인프라, 네트워킹 시스템 등 **인간 사회를 지탱하던 모든 문명적 기반이 동시에 전면 마비**된다.

> 로봇을 죽이기 위해 자신들의 목을 찌른 셈이다.
