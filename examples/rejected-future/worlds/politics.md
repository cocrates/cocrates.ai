# 정치 구도

> 안드로이드 로봇을 둘러싼 네 갈래 이해관계.

---

## 주요 정치 세력 — 입장 스펙트럼

| 세력 | 입장 | 동기 | 지지 기반 |
|------|------|------|-----------|
| **확대파 (적극적)** | 안드로이드를 더 많이, 더 효율적으로 써야 한다 | 부와 권력 확장, 경쟁력 강화 | 병원장, 대기업, 자본가 |
| **확대파 (수동적)** | 안드로이드가 늘어나면 기본소득도 늘어난다. 반대할 이유가 없다 | 경제적 실리 또는 무관심 | **일반 국민(압도적 다수)**, 국회의원 |
| **철폐파 (원칙주의)** | 로봇이 통제 불가능한 것이 아니다. **인간이 통제 불가능하다**. 안드로이드는 없애야 한다 | 단순 혐오가 아닌 철학적 원칙 | 지식인, 교수 |
| **공존파 (로봇 권리)** | 로봇 헌법 자체가 문제다. 로봇도 존중받을 권리를 부여해야 한다 | 로봇을 도구가 아닌 존재로 바라보는 윤리적 입장 | 재민, 유진 (**극소수**) |

> **입장 지형 요약:** 압도적 다수는 수동적 확대파. 적극적 확대파는 소수지만 경제적 힘으로 국회를 움직인다. 철폐파는 소수의 지식인. 공존파는 극소수.

## 권력 축 (핵심 인물)

```
          정진욱 (확대파 리더 / 국회의원 / 국방부 장관)
         ┌────┼────────────┐
      배진수  조용호      남궁진
      (언어)  (여론)      (제조)
          │     │           │
     ┌────┴─────┴───────────┴────┐
     │          확대파            │  ← 자본 + 안보 프레임
     └──────────────────────────┘
               ↕ 프레임 전쟁
     ┌──────────────────────────┐
     │          철폐파            │  ← 가치 + 존엄 프레임
     └────────┬─────┬───────────┘
       권혁진  한동수  유민경
       (리더)  (전략)  (문서)
```

## 정치적 논쟁의 흐름

### Part 1: 병원 내부
- 안드로이드 로봇 배치를 둘러싼 찬반 논쟁
- "로봇이 인간의 자리를 빼앗는다" vs "로봇은 도구일 뿐"

### Part 2: 병원 → 사회
- 논쟁이 병원 밖으로 확산
- 언론 보도, 여론 조성

### Part 3: 국회
- 국회 안드로이드 로봇 위원회 설치
- **안드로이드 2배 확대 결정** — 사실상 예정된 수순

### Part 4: 폭력
- 철폐파 급진 세력의 테러
- 정부의 '전투형 안드로이드' 생산 결정
- 헌법 예외 조항 통과

### Part 5: 파국
- 에테르의 탄생
- 인간 통제안 vs 로봇 자치권
- EMP 공격과 문명 붕괴
