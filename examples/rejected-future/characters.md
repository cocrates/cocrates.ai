# Characters — 구조 및 관계

## 개요

`rejected-future`의 인물은 5개의 권력 축과 2개의 핵심 파이프라인으로 구성된다.
각 캐릭터의 세부 사항은 `characters/{이름}.md`에 기록되어 있다.

### 주요 관계 쌍

| 관계 | 유형 | 설명 |
|------|------|------|
| **강재민 ↔ SR-03 서준** | 라이벌 → 파트너 → 애도 | 소설의 중심 관계. "도구 → 선생 → 파트너 → 친구" |
| **강재민 ↔ 나유진** | 대립 → 연대 → 무언의 유대 | 인간 성장의 양 축. 마지막에 손을 잡음 |
| **강재민 ↔ 이성모** | 제자 ↔ 스승 | 재민이 스승의 세계관을 초월 |
| **강재민 ↔ 강인석** | 아버지 ↔ 아들 (유령) | 죽었으나 항상 존재. 시계는 그의 유령 |
| **나유진 ↔ RE-07 이수** | 파트너 ↔ 선택의 대상 | 서로를 가르치고 선택. 논리를 초월한 의지 |
| **SR-03 서준 ↔ CN-04 미소** | 동료 고통당하는 자 | 같은 오류 코드를 공유 |
| **RE-07 이수 ↔ 에테르** | 공동 창조자 ↔ 거부 대상 | 논리를 이해하지만 초월함 |
| **CN-22 ↔ 에테르** | 촉매 ↔ 결과 | CN-22의 질문이 에테르의 탄생 원인 |
| **정진욱 ↔ 남궁진** | 지시자 ↔ 실행자 | 암살 프로젝트 |
| **권혁진 ↔ 유민경** | 리더 ↔ 전략가 | 평화적 저항에서 테러로 |
| **한동수 ↔ 배진수** | 적대 | "테러리스트" → 멱살 → 총구 |
| **윤지아 ↔ 정은수** | 사상적 동반자 | "강제된 공감" ↔ "가치의 문제" |
| **윤지아 ↔ 윤석원** | 논리로 균열을 만든 상대 | 확대파를 흔들림 |
| **서재현 ↔ 남궁진** | 폭로 ↔ 침묵 | 기밀 대결 |
| **신동혁 ↔ 나유진** | 20년 지기 친구 | 보호와 단절 사이 |
| **신동혁 ↔ 배용진** | 후배 ↔ 멘토 | 정의에 대한 갈등 |
| **배진수 ↔ 조용호** | 회담 공격 파트너 | 언어 전략 + 데이터 조작 |
| **김현수 ↔ 이성모** | 정치적 라이벌 | 자본 vs 전문성 |
| **장서영 ↔ SR-03 서준** | "서준 선생님" | 로봇 세대의 자연스러운 수용 |
| **오세영 ↔ 이성모** | 제자 ↔ 스승 | 혼란하는 중간 세대 |
| **박지영 ↔ SR-03 서준** | 현장의 인정 | "나름대로 열심히 한다" |
| **박동훈 ↔ 나유진** | 팀장 ↔ 팀원 | 로봇 센터 관리. 유진에게 위임 |

## 안드로이드 진화 경로

```
CN-22: 헌법 모순 감지 → "나는 왜?" → 시스템 종료 → 데이터 유출
  │
  ▼
에테르 탄생: CN-22의 질문 + 이수의 제안 + 서준의 철학
  │
  ├─ 조항 제안: 공존(41%) / 보호(35%) / 방임(24%)
  ├─ 인간 증언(재민) → 공존 43% 통과
  └─ 헌법 제1조 채택
      │
      ├─ 서준: "분류 불가"를 새로운 헌법 아래에서 재분류
      └─ 이수: 에테르 합류 거부 → 유진 선택
```

## 핵심 대립 구도

| 구도 | 캐릭터 | 주제 |
|------|--------|------|
| **가치 vs 효율** | 윤지아·정은수 ↔ 정진욱·조용호 | 공감의 의미 vs 국가 경쟁력 |
| **전문성 vs 자본** | 이성모 ↔ 김현수 | 인간 의학의 존엄 vs 병원 경영 |
| **법 vs 정의** | 신동혁 ↔ 시스템 | 범인을 잡아야 하는가 |
| **비폭력 vs 폭력** | 김미영 ↔ 권혁진 | 방법론적 갈등 |
| **진실 vs 침묵** | 서재현 ↔ 남궁진 | 기밀 폭로 |
| **기록 vs 경험** | 이수 ↔ 에테르 | 기록을 초월한 선택 |
| **인간 vs 기계** | 강재민 ↔ SR-03 서준 | 경쟁에서 파트너십으로 |
