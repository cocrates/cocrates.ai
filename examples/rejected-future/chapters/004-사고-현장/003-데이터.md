# Ch 004 Ep 003: 데이터

**POV 주도:** 유진/이수 → 재민 → 이수
**타임라인:** 현장 ~ 3시간 후 ~ 의료소 ~ 수술 결정
**길이:** 약 1,300~1,500자

---

## Purpose
Ep 002에서 준비된 데이터 복구의 완결. 유진과 이수, R-17의 메모리 모듈을 성공적으로 분리하고 데이터를 추출한다. 데이터 속에는 붕괴 당시 부상자의 위치·자세·압박 범위가 기록되어 있음. 유진, 이 데이터를 들고 재민에게 향한다. 재민, Ep 002에서 이수의 데이터를 이미 경험한 터라 이번에는 더 빠르게 수용한다. 데이터가 결정적 단서가 되어 재민의 수술 계획이 수립된다. 충돌에서 협력으로의 전환 — 말이 아닌 행동으로 증명됨.

## Story Summary
유진과 이수, R-17의 메모리 모듈을 성공적으로 분리한다. 데이터 추출 — 건물 붕괴 당시의 센서 기록이 고스란히 살아 있음. 부상자의 위치, 자세, 압박 범위, 시간 경과에 따른 변화까지. 유진, 이 데이터를 들고 의료소로 향한다. "이 데이터가 필요할 겁니다." 재민, 이번에는 데이터 패드를 받아든다 — Ep 002에서 이수의 데이터를 확인한 이후, 두 번째. 데이터를 확인한 재민의 표정이 바뀐다. "이 데이터라면... 수술이 가능하다." 유진과 재민, 말없이 고개를 끄덕인다. 이수, 이 장면을 지켜본다. 충돌이 협력으로 전환되는 순간. 이수 로그 — "강재민 교수 — 로봇 데이터 신뢰도 상승. 충돌에서 협력으로 전환 확인."

## Scene Breakdown

### Scene 1: 데이터 복구
**POV:** 유진/이수 (3인칭)

**Location:** R-17 매몰 지점 — 내부

**Time:** Ep 002 직후 ~ 30분

**Content:**
- 유진, 좁은 잔해 틈 속에서 메모리 모듈 분리 도구 사용. 이수, 외부에서 안정성 모니터링 및 데이터 링크 유지.
- 유진의 손 — 파편이 박힌 손바닥이 데이터 케이블을 연결함. 피가 묻지만, 손은 흔들리지 않음.
- "접촉했어. 메모리 모듈, 살아 있어."
- 이수, 유진의 생체 신호 급변 감지: "심박수 상승. 호르몬 변화 — 기쁨? 안도? 분류 불가."
- 메모리 모듈 분리 성공. 유진, 모듈을 안전하게 꺼냄.
- 데이터 리더기에 연결. 데이터 스트림 — 건물 붕괴 당시의 센서 기록.
- 부상자의 위치, 자세, 압박 범위, 시간 경과에 따른 데이터 변화.
- 유진, 데이터를 훑으며 표정이 바뀜. "여기에... 전부 기록되어 있어. 구조 당시의 모든 데이터."
- 이수, 데이터 검증. "데이터 무결성: 97.8퍼센트. 복구 성공."
- 유진, 데이터 패드를 집어 든다. 재민에게 가야 한다는 것을 앎.
- 이수, 유진의 결정을 확인. "강재민 교수에게 전송?"
- 유진: "내가 직접 가야겠어."

**Key lines/detail:**
- "살아 있어. 아직 데이터가 살아 있어." — 유진, 메모리 모듈을 꺼내며. '데이터'를 살아 있는 것으로 표현하는 언어 선택.
- 이수 — 유진의 언어 선택 기록: "'살아 있다' — R-17의 데이터에 생명체 속성 부여. 유진의 로봇관: R-17은 단순 장비 이상의 존재."

### Scene 2: 데이터 전달
**POV:** 재민 (3인칭) → 유진 (3인칭)

**Location:** 의료소

**Time:** 현장 ~ 3시간 30분 후

**Content:**
- 의료소. 재민, 중상자 상태 확인 중. 후송 준비 — 하지만 환자의 상태가 악화. 시간이 없다.
- 이때 유진이 나타난다. 데이터 패드를 들고.
- 유진, 아무 말 없이 데이터 패드를 내민다. 재민, 받아든다.
- 두 번째 대면. 첫 번째(충돌)와는 다른 분위기. 그러나 말은 없다.
- 재민, 데이터 확인. 데이터 패드 위에 R-17의 센서 기록 — 3D 맵, 부상자 위치와 자세, 압박 데이터.
- 재민의 표정이 바뀐다. "...이 데이터는..."
- 유진: "저 로봇이 기록한 거야. 붕괴 당시의."
- 재민, 데이터를 스크롤. 부상자의 자세, 압박 지점, 예상 손상 범위. CT도 X-ray도 없이 — 이 데이터가 부족한 진단을 보완한다.
- "...이 데이터라면 수술이 가능하다."
- 유진, 고개를 끄덕인다. 재민, 고개를 끄덕인다. 말없이.
- 충돌에서 협력으로의 전환 — 말로 선언되지 않음. 두 사람의 고갯짓이 전부.
- 재민, 데이터를 바탕으로 수술 계획을 수정. 이수의 구조 데이터 + R-17의 충돌 당시 데이터 → 부상자의 상태가 명확해짐.

**Key lines/detail:**
- 재민 — "...이 데이터라면 수술이 가능하다." (말로 함)
- **재민의 내적 독백 (말하지 않음): *…살렸어. 저 로봇의 데이터가.** *
- 유진과 재민의 고갯짓 — 충돌의 해소가 아닌, 결과에 대한 상호 인정.
- 재민, 데이터를 바탕으로 수술 계획을 수정하며 — 이 순간 유진의 말(Ep 001)을 떠올림. "저 로봇의 데이터가 없으면 당신도 사람을 구할 수 없습니다." — 그 말이 맞았음을 인정함. 그러나 말로 하지 않음.

### Scene 3: 변화의 기록
**POV:** 이수 (3인칭)

**Location:** 의료소 외부

**Time:** 현장 ~ 4시간 후

**Content:**
- 이수, 의료소 외부에서 장면을 관찰. 유진이 데이터를 전달하고, 재민이 수용하는 전 과정.
- 데이터 기록: "강재민 — 데이터 수용 시간: 2.3초. 이전(Ep 002) 대비 1.2초 단축. 사실 기록."
- 이수 — 재민의 데이터 수용 패턴 관찰. "강재민 — SR-03 데이터: 지연 수용. R-17 데이터: 즉시 수용. 데이터 출처에 따른 반응 차이 기록."
- 이수의 두 번째 관찰: 유진과 재민의 관계 변화. "유진-재민 — 충돌 후 데이터 교환. 관계 변화 관찰: 충돌 → 협력. 패턴 등록."
- 이수, 유진과 재민의 고갯짓을 기록. 사진처럼.
- "RE-07 로그: R-17 메모리 복구 완료. 유진-재민 데이터 교환 완료. 관계 상태: 충돌 → 협력."
- 에피소드 클로징: 유진과 재민이 각자의 자리로 돌아간다. 이수, 유진에게 다가간다. 유진의 손 — 피가 굳어 있다. 이수: "유진. 당신의 손. 치료가 필요합니다."

**Key lines/detail:**
- 이수 로그: "데이터가 인간-인간 관계의 변화를 촉발. 흥미로운 패턴. — RE-07."
- 이수 — 유진에게: "당신의 손. 치료가 필요합니다."
- 에피소드의 마지막 이미지: 유진의 손. 피와 먼지. 그러나 데이터는 살아 있다.

## Narrative Voice & POV Notes
- **유진 POV:** 데이터 복구의 성공에서 오는 안도감과 재민과의 연결에서 오는 미묘한 감정 변화. 그러나 말로 설명하지 않음. 행동과 표정으로만.
- **재민 POV:** 데이터를 확인하는 순간의 표정 변화 — 이전(Ep 002)보다 빠르게 수용함. 유진의 말이 맞았음을 인정하지만, 말하지 않음. 고갯짓 하나로 전달. **내적 독백 '*…살렸어. 저 로봇의 데이터가.*'는 대사로 처리 금지 — 독자만 알게 할 것.**
- **이수 POV:** 관찰자로서의 거리 유지. 그러나 '흥미로운 패턴'이라는 표현에서 이수의 내부 변화가 극도로 미묘하게 드러남. 이수에게 이 장면은 단순한 데이터 포인트가 아니라, 인간-로봇-인간 관계의 새로운 가능성을 보여주는 사례.

## Seeds
- 유진의 데이터 전달 — 유진과 재민의 관계 변화 시작. (→ Ep 004~005로 이어짐)
- 이수, 재민의 데이터 수용 시간 단축 관찰. (→ 재민의 내적 변화 지표)
- 유진의 손 부상 — 치료 필요. (→ Ep 004에서 이수의 치료 장면으로 연결)
- R-17의 데이터가 재민의 수술에 활용됨 — Ch 005에서 이 데이터가 재민의 수술 기록으로 남게 됨. (→ Ch 005 연결)
