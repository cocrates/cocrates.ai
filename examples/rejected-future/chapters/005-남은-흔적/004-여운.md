## Purpose
Ch 005의 마무리. 유진, R-17의 데이터 백업을 완료하고 복구 작업을 마무리한다. 이수와의 대화 — **"유진, 당신은 왜 R-17의 데이터를 살리려 했나?"** 유진, 답하지 못한다. 또는 답하지 않는다. 저녁이 되었다. 유진, 저장한 재민의 프레임을 다시 한 번 본다. Ch 007(병원 재회)로의 연결을 암시하는 마지막 장면 — 유진, 다음 주 병원 일정을 확인한다. R-17의 구조 데이터 제출 건으로. **그러나 그 결정에 재민이 얼마나 영향을 미쳤는지 — 유진 자신도 인식하지 못한다.**

## Reader Engagement
### Personal Stake
- **유진:** 하루 동안의 복구 작업 마무리. R-17의 데이터는 백업되었다. 그러나 그녀에게 남은 것은 데이터 이상의 것 — 재민의 프레임, 나나의 기억, 그리고 스스로 설명할 수 없는 다음 주 병원 방문 결정.
- **이수:** 로봇으로서 인간에게 '왜'를 묻는 순간. 유진의 답변 부재 — 그 답변 부재 자체가 데이터가 됨. 유진의 병원 방문 결정을 '재민 교수 관련 가능성 67.3%'로 예측.

### Reader Questions
- 유진은 왜 R-17의 데이터를 살리려 했는가? 그녀 자신도 대답할 수 없는 이 질문의 의미는?
- 유진의 '다음 주 병원' 결정 — R-17 데이터 제출이 공식적 이유이지만, 진짜 이유는?
- 이수의 67.3% — 이 숫자가 의미하는 바는? 유진보다 이수가 먼저 유진의 마음을 데이터로 읽기 시작함.

### Stakes
- **Ch 005의 종결:** R-17 복구 작업 완료. 재민에 대한 유진의 인식 변화 완료. Ch 007로의 연결 설정.
- **관계 전환점:** 유진이 수동적 관찰자에서 능동적 행동(병원 방문 결정)으로 나아감. 그러나 그 의도를 스스로 인식하지 못함. 이수가 먼저 인지.
- **캐릭터 발전:** 유진에게 R-17의 의미 — '장비' 이상의 존재. 이 질문에 답하지 못하는 것이 그 증거.

### Reader-Discovered Meaning
**What reader should conclude:**
유진이 '다음 주 병원'을 언급한 순간, 독자는 그 결정이 단순한 업무 이상임을 알게 된다. R-17의 구조 데이터는 이메일로도 제출 가능하다. 유진이 직접 방문을 선택한 것 — 그 선택에 재민의 프레임이 영향을 미쳤음을 독자는 알고 있다. 그러나 유진은 그 사실을 인식하지 못한다. 이수의 67.3%는 이 사실을 객관적으로 증명한다. Ch 005의 마지막 — 유진은 아직 재민을 '만나지 않았지만', 이미 그녀의 결정은 재민을 향해 움직이기 시작했다.

**Hold — do NOT write in manuscript:**
- 유진이 '다음 주 병원' 결정의 진짜 이유를 합리화하지 않음. "사실 데이터 제출이 핑계였어" 금지.
- 이수의 67.3%가 유진에게 전달되지 않음 — 이수는 내부 로그에만 기록.
- 유진의 감정을 노골적으로 서술하지 않음 — 독자는 그녀의 행동(일정 확인, 프레임 다시 열기)을 통해 유추.

**Closing image:**
저녁. 작업실의 불이 꺼졌다. 유진, 주머니 속 시리얼 플레이트를 만지며 로봇 센터를 나선다. 이수, 그 뒤를 따른다. 내일 — 그리고 다음 주. 유진은 아직 모르지만, 그녀의 발걸음은 이미 재민을 향해 있다.

## Key Events

### Scene 1 — 백업 완료 / 작업 마무리
**POV:** 유진 (3인칭) | **Location:** 로봇 센터 — 작업실

- **Situation:** 저녁. R-17 데이터 백업이 완료됨. 유진, 작업실을 정리하기 시작. 하루 종일의 분석이 끝남.
- **Beat:** 모니터에 'Backup Complete — R-17 Memory Module / 2045-11-23' 메시지. 유진, 그 메시지를 바라봄. R-17의 모든 데이터가 안전하게 백업되었다. 그러나 그녀의 표정은 완료의 안도가 아님 — 뭔가 끝나지 않은 듯한 표정. 유진, R-17의 시리얼 플레이트를 주머니에서 꺼내 잠시 바라봄. 'R-17 / 2042-03-21'. 다시 주머니에 넣음.
- **Turn:** 이수, 조용히 다가옴. "유진, 당신은 왜 R-17의 데이터를 살리려 했나?" 유진의 손이 멈춤. 그녀, 잠시 동안 대답하지 못함. R-17의 데이터를 살린 이유 — 기술적 필요? 업무상 의무? 정서적 애착? 그중 어느 것도 완전한 대답이 될 수 없음을 앎. "…글쎄." 그리고 덧붙임: "그냥… 그래야 했어." 이수, 더 묻지 않음.
- **Function:** Ch 004 Ep 004에서 이수가 던진 질문의 연장. 유진의 대답 부재 — R-17이 그녀에게 단순 장비 이상이었음을 증명. 이 질문은 Ch 005의 여운을 결정지음.

### Scene 2 — 저장된 프레임 / 다시 보는 재민
**POV:** 유진 (3인칭) | **Location:** 유진 작업실

- **Situation:** 유진, 퇴근 준비를 하려다 저장한 재민의 프레임을 다시 엶. (→ 이전 장면의 질문과 연결되는 유진의 무의식적 행동.)
- **Beat:** 유진, 작업실 정리를 마치고 가방을 챙기려다 손이 마우스로 감. '개인' 폴더 — frame_0174.jpg. 더블클릭. 재민의 얼굴이 다시 모니터에. 유진, 이번에는 오래 바라보지 않음. 그러나 완전히 무시하지도 않음. 그냥… 확인. 이수, 그 화면을 바라봄. "강재민 교수입니다." 유진: "…응." 유진, 프레임을 닫음. 이번에는 의식적으로. "됐다."
- **Turn:** 유진, 자리에서 일어남. 그리고 말함 — 계획된 일정처럼. "다음 주에 병원에 데이터 제출하러 가야겠네. R-17의 구조 데이터. 원본 제출 요청이 있었어." — 사실 그런 요청이 있었는지 불분명함. 그러나 유진은 그렇게 말함. 이수: "…그렇겠네요." 유진, 가방을 챙김. 그리고 아주 약간 — 미소일까? 아닐까? — 표정이 흐려짐. 유진 자신도 모르는 표정. 이수, 기록한다.
- **Function:** Ch 007 병원 재회로의 연결. 유진의 '다음 주 병원' 결정 — 겉으로는 업무상 필요이나, 그 결정의 진짜 동기는 재민의 프레임. 유진 자신도 이 연결을 인식하지 못함. 이수만이 데이터로 확인.

### Scene 3 — 저녁 / Ch 005 종료
**POV:** 유진 (3인칭) — 이수 내부 로그 보조 | **Location:** 로봇 센터 — 로비 / 퇴근

- **Beat:** 유진, 로봇 센터 건물 밖으로 나서며 깊은 숨. 하루 종일 데이터 속에 있었음. 신선한 공기. 그리고 생각 — R-17, 재민, 나나. 모든 것이 하루 동안 그녀의 앞에 펼쳐짐. 아직 모든 것을 이해하지 못함. 이수, 그녀의 옆에 서 있음. 유진: "…오늘 고생했어, 이수." 이수: "유진도요." 유진, 짧게 웃음. 그러나 웃음이 아님.
- **Turn:** (POV Insert — 이수 내부 로그) 이수, 기록: "유진 — '다음 주' 일정 언급. 생체 신호 변이 감지 — 심박수/호흡 패턴 변화. 원인 분석: 강재민 교수 관련 가능성 67.3%. 추가 데이터 수집 필요." 유진, 이수의 기록을 모름. 그저 걸어감. 주머니 속 시리얼 플레이트가 발걸음에 맞춰 흔들림.

## Ch 005 통합 설계 노트 (에피소드 상세 설계 완료 후)

### 에피소드 간 흐름 요약

```
Ep 001 잔해 (새벽, 현장)
  └─ 침묵·고요 — R-17 물리적 종말 — 시리얼 플레이트 보관
     └─ R-17 메모리 모듈 → 로봇 센터 이동
Ep 002 기록 (오전~오후, 로봇 센터)
  └─ 데이터 분석 — R-17의 마지막 재구성 — 재민 수술 데이터 발견
     └─ "이 의사, 실력이 상당한데" — 첫 평가
        └─ 시각 센서 로그 복구 예고
Ep 003 프레임 (오후, 로봇 센터)
  └─ 시각 센서 이미지 복구 — 재민의 얼굴 발견
     └─ 23초 — 유진의 시선 — 나나 기억 스침
        └─ '개인' 폴더에 프레임 저장
Ep 004 여운 (저녁, 로봇 센터 → 퇴근)
  └─ 백업 완료 — "왜 R-17의 데이터를 살리려 했나?"
     └─ 프레임 다시 열기 — "다음 주 병원"
        └─ 이수 기록: 강재민 관련 가능성 67.3%
```

### 설계 의도 충족 확인

| 원칙 | 구현 |
|------|------|
| 유진 단독 POV — 재민 직접 등장 없음 | ✓ 유진 POV만 유지. 재민은 Ep 002(수술 데이터)·003(프레임 속)에서만 간접적으로 존재. |
| Ch 004의 빠른 템포 → Ch 005의 느린 템포 | ✓ Ep 001 침묵 → Ep 002 집중 → Ep 003 정적 → Ep 004 대화와 결정. Ch 004와 확연히 다른 템포 곡선. |
| 데이터 분석 = 유진이 재민을 알게 되는 과정 | ✓ Ep 002(수술 데이터 → 전문성 인정) → Ep 003(얼굴 프레임 → 개인적 관심) → Ep 004(병원 방문 결정 → 행동). 재민에 대한 유진의 인식이 점진적으로 변화. |
| 나나 — 극도로 가볍게 스침 | ✓ Ep 003에서만 한 번 등장. 설명 없음. 유진도 의미를 모름. |
| R-17을 '장비' 이상으로 대하는 유진의 태도 | ✓ Ep 001(시리얼 플레이트 보관) → Ep 004("왜 살리려 했나?"에 답하지 못함). 행동과 침묵으로 증명. |
| 이수 — 유진 관찰 지속 | ✓ 23초 기록(003), 67.3% 예측(004). 유진보다 먼저 유진의 변화를 데이터로 포착. |

### Ch 004 연계성 검증

| Ch 004 스레드 | Ch 005 처리 |
|---------------|-------------|
| 유진 손 부상 → 이수 치료 | Ep 001 — 붕대. Ch 004의 신체적 흔적. Ep 002~004에서도 간헐적으로 언급 가능. |
| R-17 메모리 모듈 복구 → 분석 | Ep 001(물리적 잔해 수습) → Ep 002(데이터 분석) → Ep 003(시각 데이터 복구) → Ep 004(백업 완료). 전 에피소드에 걸친 주요 플롯. |
| 유진-재민 헬기장 1초 | Ep 003 — 프레임 속 재민을 보며 1초가 떠오름. 그러나 유진이 의식적으로 연결하지 않음. |
| 이수, 재민 '분류 중' 등록 | Ep 002 — 재민 수술 데이터 분석. Ep 003 — 유진 반응 기록. Ep 004 — 강재민 관련 가능성 67.3%. 이수의 재민 데이터베이스 지속적 확장. |
| 재민, 유진-이수 관계 관찰 | 유진 POV이므로 직접 다루지 않음. 그러나 독자는 Ch 004의 재민 POV를 기억하며 유진-이수 관계를 새롭게 볼 기회를 가짐. |
