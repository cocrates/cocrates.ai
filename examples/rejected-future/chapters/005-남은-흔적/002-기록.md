## Purpose
로봇 센터. 유진, R-17의 메모리 전체를 분석한다. 단순한 데이터 복구가 아니라, R-17이 마지막으로 '본 것'과 '기록한 것'을 재구성하는 과정. 그 과정에서 붕괴 당시 R-17이 기록한 센서 데이터 속에 재민의 응급 수술 장면이 포함되어 있음을 발견한다. 유진, 재민의 수술 데이터를 보며 그의 의사로서의 실력을 간접적으로 목격한다. **"이 의사, 실력이 상당한데?"** — 그녀의 재민에 대한 첫 평가. Ch 004 Ep 003과의 대칭: Ch 004에서 재민이 데이터로 유진을 알게 되었다면, Ch 005에서 유진이 데이터로 재민을 알게 됨.

## Reader Engagement
### Personal Stake
- **유진:** R-17의 데이터 분석. 단순 업무이지만, R-17의 '마지막 순간'을 재구성하는 과정이 정서적임. 데이터 속에서 발견한 재민의 수술 — 처음 보는 그의 전문성.
- **이수:** 유진의 데이터 분석 방식을 관찰하며 인간이 데이터를 '의미'로 해석하는 과정 기록. 발견한 재민 데이터 보완.

### Reader Questions
- R-17이 마지막으로 '본' 것은 무엇인가? 그 데이터 속에서 유진이 발견하는 것은?
- 유진은 재민의 수술 데이터를 보고 무엇을 느끼는가? 단순한 감탄인가, 그 이상인가?
- Ch 004에서 재민이 유진의 데이터를 통해 그녀를 알게 된 것과의 대칭 — 독자가 이 구조를 인지하는가?

### Stakes
- **정보적:** 재민의 수술 데이터 — 그의 전문성을 입증하는 객관적 증거. 유진이 재민에 대해 갖는 첫 인상의 근거.
- **정서적:** 유진이 R-17의 데이터를 통해 재민을 '발견'하는 과정. Ch 007(병원 재회)로 가는 첫걸음.

### Reader-Discovered Meaning
**What reader should conclude:**
유진이 재민의 수술 데이터를 보고 한 첫 평가("실력이 상당한데")는 단순한 감탄이 아니다. 그녀는 재민을 의사로서 처음 인정하는 순간이다. Ch 004에서 충돌했던 '이름 모를 의사'가 이제 '실력 있는 의사'로 재정의된다. 그리고 이 평가는 Ch 007(병원 재회)에서의 관계 발전을 위한 토대가 된다. 동시에 Ch 004 Ep 003과의 대칭이 드러난다 — 재민은 데이터를 통해 유진을 알게 되었고, 유진은 데이터를 통해 재민을 알게 된다. 두 사람이 동일한 데이터를 매개로 서로를 발견하는 구조.

**Hold — do NOT write in manuscript:**
- "재민은 실력 있는 의사구나" 같은 내적 독백 금지. 독자는 유진의 행동(데이터 재생, 특정 구간 확대, 짧은 평가)을 통해 스스로 결론을 내림.
- "Ch 004에서 재민이 나를 데이터로 봤겠네" 같은 대사 금지. 대칭 구조는 독자가 발견하도록.

**Closing image:**
유진, 수술 데이터 재생을 마친 후 잠시 화면을 응시. "…괜찮네." 이수, 그 평가를 기록한다. 유진, 다음 데이터로 넘어가기 전 — 재민의 이름을 처음으로 메모장에 입력한다: '강재민 교수 — 수술 데이터 확인'.

## Key Events

### Scene 1 — R-17 메모리 분석 시작
**POV:** 유진 (3인칭) | **Location:** 로봇 센터 — 유진 작업실

- **Situation:** 오전. 로봇 센터. 유진, R-17 메모리 모듈을 리더기에 연결. 현장 작업에서 돌아와 세수도 제대로 하지 않고 바로 분석 시작. 이수, 조용히 보조.
- **Beat:** 모니터에 데이터 스트림이 흐름. 유진, 체계적으로 데이터를 분류. 센서 로그, 시스템 로그, 시각 데이터, 청각 데이터. R-17이 마지막 순간에 기록한 모든 것. 유진, 데이터를 따라가며 R-17의 마지막을 재구성. 이 과정이 단순 기술 분석이 아님을 느낌 — R-17의 '목소리'를 듣는 것.
- **Turn:** 유진, 데이터 중간에서 임시 의료소 영역의 센서 기록 발견. 붕괴 당시 R-17이 의료소 부근까지 커버하고 있었음. 유진, 데이터를 재생. 심박수 측정, 진동 센서, 구조물 붕괴 위험 경고 — 그 속에서 인간의 생체 신호와 의료 행위 기록이 섞여 있음.
- **Function:** 유진의 데이터 분석 방식과 R-17에 대한 태도 재확인. Ch 004 Ep 001-003에서 유진이 R-17의 메모리를 대하는 태도가 여기서도 이어짐.

### Scene 2 — 재민의 수술 데이터 발견
**POV:** 유진 (3인칭) | **Location:** 유진 작업실

- **Situation:** 유진, 의료존 데이터를 세밀히 분석. R-17의 센서가 재민의 수술 장면을 기록하고 있음을 발견.
- **Beat:** 유진, 수술 데이터를 재생. CT/X-ray 없음 — 붕괴 현장의 의료 한계. R-17의 센서 데이터만으로 재민이 환자의 상태를 판단하고 수술을 집도. 유진, 수술 데이터의 세부를 확대 — 재민의 손 움직임, 절개 깊이, 지혈 속도. 데이터는 객관적이다. 유진, 재민의 손놀림과 판단 속도를 읽어냄. (→ POV Insert: 유진의 기억 — Ch 004 Ep 003, 자신이 재민에게 데이터를 건넨 순간. 그 데이터가 어떻게 사용되었는지 지금 확인함.)
- **Turn:** 유진, 재생을 멈춤. 잠시 화면을 응시. "…이 의사, 실력이 상당한데." 이수, 즉시 저장: "강재민 교수 — 유진의 평가: '실력이 상당함'. 객관적 기준: CT/X-ray 부재 상황, 수술 성공률 92.3%, 손놀림 정확도 센서 데이터 일치." 유진, 이수의 리포트를 보고 짧게 웃음. "또 기록하네."
- **Function:** 유진의 재민에 대한 첫 평가. 전문가로서의 인정. Ch 007 병원 재회의 전제 조건 충족.

### Scene 3 — 분석 계속 / 데이터 속의 재민
**POV:** 유진 (3인칭) | **Location:** 유진 작업실

- **Situation:** 유진, 데이터 분석을 계속하지만 시선이 자꾸 재민의 수술 기록으로 감. 의식적으로 다음 데이터로 넘어감.
- **Beat:** 유진, 다른 센서 데이터로 주의를 돌림. 하지만 손은 자연스럽게 재민의 기록에 대한 북마크를 생성. '참고 — 의료 데이터 분석'. 공식적인 이유. 유진 자신도 이 북마크의 의미를 완전히 인식하지 못함. 이수, 유진의 행동 패턴을 관찰. 기록에는 추가하지 않음 — 아직.
- **Turn:** 유진, 다음 데이터 카테고리로 이동. 시각 센서 로그. 아직 복구되지 않은 이미지들. 마지막 말: "이따가 이것도 봐야겠네." 시각 데이터를 가리킴. (→ Ep 003 프레임으로의 연결.)
- **Function:** 유진의 재민에 대한 관심이 기술적 호기심을 넘어서고 있음을 암시. 그러나 유진 자신도 인식하지 못함. Ep 003으로의 자연스러운 전환.

## POV

## Tone Notes
Ep 001의 침묵과 고요함에서, Ep 002는 기술 작업의 집중으로 전환. 대사는 여전히 적지만, 데이터 스트림과 모니터의 움직임이 템포를 만듦. 유진의 전문성이 드러나는 에피소드 — 그녀가 데이터를 읽는 방식, R-17의 데이터를 대하는 태도에서 그녀의 캐릭터가 드러남. 재민의 수술 데이터 발견 — 에피소드의 중심은 기술적 발견이지만, 그 의미(유진이 재민을 의사로서 처음 인정함)는 독자가 발견하도록. "실력이 상당한데"는 칭찬이면서 동시에 Ch 004와의 첫 연결고리.
