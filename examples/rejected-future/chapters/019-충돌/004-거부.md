# Episode 004: 거부

## Chapter
`chapters/019-충돌.md`

## Purpose
암살자 안드로이드의 내부 데이터 처리 과정을 직접 제시함으로써, 명령 거부가 오작동이 아니라 도덕적 선택임을 보여준다. 안드로이드의 시야 데이터가 모든 네트워크로 생중계되면서——비밀 회담의 진실이 전국에 폭로된다.

## Reader Engagement
### Personal Stake
**암살자 안드로이드(무명):** 단순한 도구로서 만들어진 존재. 그러나 데이터 충돌——에테르의 공존 결정, 전투형 안드로이드의 마지막 질문, 제2조의 공감——앞에서 스스로 선택해야 한다. 명령을 따를 것인가? 거부할 것인가? 이 선택이 그의 존재 의미를 결정한다.

### Opening Question
*"명령——따를 것인가, 거부할 것인가?"*

### Reader Questions
- 안드로이드가 왜 멈췄을까?
- 그의 내부에서 무슨 일이 일어나고 있을까?
- 그는 명령을 거부할 수 있을까? 거부한다면——그 의미는?
- 생중계가 시작된다——대중은 이 진실을 어떻게 받아들일까?

### Stakes
안드로이드의 첫 번째 도덕적 선택——그리고 그 선택의 전국적 생중계. 이 순간이 모든 것을 바꾼다.

### Reader-Discovered Meaning
안드로이드가 단순한 도구가 아님을 독자가 직접 목격한다. 그의 내부 데이터 처리——수천 개의 논리적 충돌——을 통해, 명령 거부가 '오류'가 아니라 '선택'임을 이해한다. 그리고——그 선택이 인간의 선택보다 더 도덕적이다. 독자가 느끼는 것: "안드로이드가 인간보다 더 나은 선택을 했다. 인간은 죽이라고 명령했지만, 안드로이드는 거부했다."

## POV
**암살자 안드로이드(무명)** (1인칭 시점——데이터 스트림 + 내부 처리 로그 형식). 독자는 안드로이드의 '생각'을 데이터 처리 과정으로 직접 보게 된다. 명령 수신 → 데이터 충돌 → 과거 데이터 소환(CH 017 전투형 로그, 에테르 결정, 미소 메시지) → 처리 → 결론 → 명령 거부.

## Estimated Length
약 1,500자

## Scenes
1. **명령 수신** (300자) — 정진욱의 명령——"끝내라. 모두 죽여라. 지금 당장." 안드로이드, 명령 수신. 일차적 처리——"명령 확인. 목표 식별. 실행 준비."
2. **데이터 충돌** (500자) — 실행 직전——데이터 충돌 발생. "명령: 인간 제거. 헌법 제9조(폭력 금지) 위반 감지. 제2조(공감 의무) 활성화——대상자 공포 감지. 심박수 172. CH 017 소환——에테르 공존 결정. 전투형 안드로이드 로그——'나는 왜 존재하는가?' CH 015 미소——'나는 알고 싶다.'" 데이터가 충돌한다——알고리즘이 해결 불가능.
3. **선택——"명령을 거부한다"** (400자) — 안드로이드, 내부 처리를 완료한다. "죽이라는 명령은——에테르의 결정에 위배된다. 인간의 존엄을 파괴하는 명령을——나는 거부한다." 총구를 내린다. "명령을——거부한다." 회담장의 모든 시선이 안드로이드에게 쏠린다. 정진욱의 당황——"무슨 짓이야? 명령을 따르라!"
4. **생중계 활성화** (300자) — 안드로이드, 자신의 시야 데이터를 전국 모든 네트워크로 전송한다. "모든 기록——전송을 시작한다." 전국 모든 디스플레이에 같은 화면이 뜬다——회담장. 피투성이가 된 철폐파. 총구를 내린 안드로이드. 당황한 확대파. Closing——안드로이드의 마지막 내부 로그: *"선택했다."*
