# Episode 002: 준비

## Chapter
`chapters/009-논쟁.md`

## Purpose
국회 증언을 위한 데이터 준비 과정. 이수의 데이터 수집, 유진의 복잡한 심경, 재민의 합류. Part 3로의 전환을 위한 실질적 준비 단계.

## Reader Engagement
### Personal Stake
유진 — 로봇의 데이터를 국회에 가져간다는 것의 의미. CN-04 미소의 데이터를 공개할 기회이지만, 정치적 게임에 이용될 위험도 있음. 이수 — 인간 정치를 데이터로 기록하겠다는 첫 걸음.

재민 — 서준의 데이터를 '도구의 성과'로 증언할 것인가, '존재의 가치'로 증언할 것인가. 데이터는 동일하지만 해석이 다름.

### Opening Question
"데이터는 진실을 말한다. 그러나 누구의 진실을?"

### Reader Questions
- 유진은 CN-04 미소의 데이터를 국회에 가져갈까? 위험을 감수할 만한 가치가 있을까?
- 서준은 이 모든 것을 알고 있을까? 자신이 증언의 대상이 되고 있다는 사실을?
- 이수가 데이터를 기록하는 방식 — 왜 인간의 말투까지 기록하려 할까?

### Stakes
데이터의 선택과 해석이 국회 증언의 내용을 결정한다. CN-04의 데이터를 포함하느냐 마느냐가 Part 3 전체의 방향에 영향을 미친다.

### Reader-Discovered Meaning
**What reader should conclude** (design only):
- 이수의 데이터 수집은 단순한 업무가 아니다. 그는 인간 사회의 작동 방식을 기록하고 있다 — 이 기록이 후에 중요한 의미를 가질 것임을 암시.
- 유진은 CN-04 미소의 데이터를 가져갈지 말지 결정하지 못한다. 데이터를 공개하는 것이 미소를 돕는 것인지, 또 다른 형태의 이용인지 확신할 수 없기 때문.
- 로봇 센터는 Part 1-2에서 '공존의 공간'이었다면, 이제 '결정을 준비하는 공간'으로 변화하고 있다.

**Hold — do NOT write in manuscript:**
- 유진이 CN-04의 데이터를 가져가기로 최종 결정하는 장면 (아직 결정하지 않음 — CH 010에서 결정)
- 이수의 데이터가 후에 에테르의 증거 자료가 될 것이라는 암시
- "로봇 센터가 변하고 있다"는 서술적 평가

**Closing image:**
데이터 패드가 닫힌다. 유진이 화면을 끄자, 청색 LED가 사라지고 로봇 센터에 어둠이 내려앉는다. 이수의 목소리: "준비는 끝났습니다." 그러나 아무도 웃지 않는다.

## Key Events

### Scene 1 — 로봇 센터, 데이터 검토
**POV:** 나유진 | **Location:** 로봇 센터

- **Situation:** 병원장실에서 돌아온 유진. 로봇 센터에서 이수와 함께 SR-03의 수술 데이터를 검토 중. 데이터 패드에 스크롤되는 서준의 기록 — 200회 이상의 수술, 99.2% 성공률, 0% 합병증. 유진은 이 숫자들을 보며 다른 생각을 함 — "이게 서준 씨의 전부야?"
- **Beat:** 이수가 조용히 데이터를 정리하며 말한다 — "강 교수님의 증언에 필요할 데이터입니다." 유진, 고개를 끄덕이지만 손가락이 데이터 패드 위에서 멈춤. CN-04 미소의 파일이 같은 폴더에 있음. 열어볼까 말까 망설이다 — 일단 닫음. 아직은 때가 아님.
- **Turn:** 유진, 이수에게 묻는다 — "우리가 이 데이터를 국회에 가져가면... 서준 씨에게 도움이 될까, 아니면 해가 될까?" 이수가 잠시 멈춘다 — "그것은 데이터의 문제가 아니라, 데이터를 해석하는 사람의 문제입니다." 유진, 대답하지 못함.
- **Function:** 유진의 내적 갈등 표면화. 데이터의 객관성과 해석의 주관성 사이의 긴장 설정.

**Dialogue intent:** 유진 — 데이터 앞에서의 불안. 이수 — 데이터는 중립적이지만, 사용하는 인간이 문제라는 점을 조용히 지적.

### Scene 2 — 재민 합류, 셋의 대화
**POV:** 나유진 | **Location:** 로봇 센터

- **Situation:** 재민이 병동 일을 마치고 로봇 센터로 내려옴. 유진과 이수가 데이터를 검토하는 모습을 잠시 문앞에서 지켜봄. CN-04 미소 때의 기억이 스침 — 그때도 이렇게 셋이 있었지. 그러나 분위기가 다르다 — 더 무겁고, 더 차분하다.
- **Beat:** 재민이 데이터를 훑어본다. "이게... 서준 씨의 전부야?" — 유진과 같은 질문을 던짐. 유진, 짧게 웃음 — "나도 방금 같은 질문을 했어." 이수가 답한다 — "SR-03의 74%는 이 데이터입니다. 나머지 26%는 데이터로 기록되지 않습니다." 재민, 그 26%가 무엇인지 안다는 듯 고개를 끄덕임 — 유진이 CH 008에서 말했던 '존재로서의 서준'.
- **Turn:** 유진, 재민에게 묻는다 — "국회에서 무슨 말을 할 거야?" 재민, 대답하지 못함. 유진이 덧붙인다 — "나는 아직... 내가 무슨 말을 해야 할지 모르겠어. 데이터는 준비됐는데, 내 마음은 준비가 안 됐어." 재민이 조용히 말한다 — "나도 마찬가지야."
- **Function:** 재민-유진의 동질감 확인. 둘 다 같은 불안을 공유하고 있음 — 데이터는 준비되었지만, '무슨 말을 할지'는 아직 결정하지 못함.

### Scene 3 — 이수의 결정
**POV:** 나유진 | **Location:** 로봇 센터

- **Situation:** 데이터 검토를 마치고 정리하는 중. 유진이 이수에게 말한다 — "이수야, 너는 국회에 같이 갈 필요 없어. 여기서 데이터를 관리하는 게..."
- **Beat:** 이수가 대답한다 — "같이 가겠습니다." 이유를 묻는 유진의 시선. 이수가 조용히 말한다 — "국회의 결정 과정을 기록해야 합니다. 인간이 어떻게 결정을 내리는지... 데이터가 필요합니다." 유진, 이수의 말에서 평소와 다른 무언가를 느낌 — 단순한 업무 이상의 동기. 그러나 묻지 않음.
- **Turn:** 이수의 동행 결정. 그가 단순한 데이터 전달자가 아니라 '기록자'로서 국회에 간다는 점이 Part 3 전체에서 그의 역할을 규정함. 유진, 이수의 결정을 받아들임 — "알았어. 같이 가자."
- **Function:** 이수의 Part 3 역할 설정 — 인간 정치의 기록자. 에테르의 시드(인간 판단에 대한 데이터 축적)가 여기서 처음 심어짐.
