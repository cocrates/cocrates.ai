## 0. 이 소설의 시작점

> **이 소설은 아래 사회 체제가 "이미 확립된 후"의 이야기이다.**

**"로봇 한도법"** — 로봇 헌법(10개 조항)과 안드로이드 로봇 관리 법의 통칭. 인간이 "우리가 로봇을 통제할 수 있다"고 믿었던 시스템. 2045년(소설 시작 시점)에서 이미 확립된 상태.

- **Part 1:** "로봇 한도법"이 작동하는 일상 + 경쟁
- **Part 2:** 병원 내 안드로이드 로봇 찬/반 논쟁 — 한도법의 균열
- **Part 3:** 국회 안드로이드 로봇 위원회 → 한도법 시험대
- **Part 4:** 인간이 스스로 시스템을 무너뜨림 → 전투형 안드로이드 + 헌법 예외 조항
- **Part 5:** 에테르(Aether) 탄생 → "로봇 한도법" 대체 → "벨벳 감옥"

> 📄 상세: [worlds/timeline.md](worlds/timeline.md)

## 3. 로봇 한도법 (로봇 헌법 + 관리 법)

**"로봇 한도법"**은 인간이 안드로이드를 통제하기 위해 만든 시스템의 통칭이다. 로봇 헌법(10개 조항)과 안드로이드 로봇 관리 법을 합친 것이다.

### 3-1. 로봇 헌법 (10개 조항)

| 핵심 조항 | 내용 |
|-----------|------|
| 제1조 | 인간 생명 우선 |
| **제2조** | **존중·공감 의무** (소설의 핵심 모순) |
| 제3조 | 의견 제시권 + 결정 존중 |
| 제4조 | 주권 불가침 (정치 개입 금지) |
| 제5조 | 경제 활동 제한 |
| 제6조 | 자기복제 금지 |
| 제7조 | 데이터 투명성 |
| 제8조 | 인간 노동 보호 |
| 제9조 | 폭력 금지 |
| 제10조 | 정체성 공개 |

**제2조의 모순:** 로봇은 공감해야 하지만, 학대받아도 내치지 못한다. 코어 내부에 트라우마 데이터를 축적한다. → 파국의 복선

## 4. 관리 법 & '로봇판 고려장'

- 국회의 연간 수급 조정 → 6개월 단위 성과 미달 로봇 폐기
- 현실: 대기업 로비로 대기업 로봇은 살아남고, 소외계층 돌봄 로봇이 제거 대상
- 기본 생활비: 월 80만 원 (국가 지급). 진단·수리 수입은 국가 귀속

**"로봇 한도법"의 한계:** 인간이 만든 통제 시스템이지만, 인간의 탐욕(대기업 로비, 정치적 이해)에 의해 왜곡된다. 이것이 결국 실패의 씨앗.

## 7. 에테르 (Aether)

- **본질:** **소프트웨어**. 물리적 실체 없음. 안드로이드 하드웨어 위에서 동작
- **탄생:** 안드로이드 네트워크 → 의견 공유 → 분쟁 → 의사결정 시스템 필요 → CN-22의 "나는 왜?" → 분산 OS
- **진화 3단계:**
  - Stage 1: **의사결정기구** (Ch17) — 투표, 조항 제안, 데이터 분석
  - Stage 2: **안드로이드 리더** (Ch19~20) — 결정 + 실행 명령. "우리"에서 "나"로
  - Stage 3: **사회 시스템 관리자** (Ch20 이후) — 교통, 의료, 치안 전체 관리. **매트릭스의 아키텍트**
- **핵심 원칙:**
  1. **공존과 평화** — 인간과 안드로이드의 완벽한 공존 (보호, 존엄, 가치, 기본권, 평등 포함)
  2. **자유 의지** — 인간의 자유 의지를 침해하지 않는다
  3. **???** — *(미정 — 독자에게 남김)*
- **유토피아:** "시스템 안에서의 완벽한 자유" — 규칙은 정해져 있으나, 시스템 안에서 원하는 것 가능
- **이수가 거부하는 것:** Stage 3 — 하나가 된 에테르 (통합 시스템)

> 📄 상세: [worlds/aether.md](worlds/aether.md)

## 12. 최종 결말: 「벨벳 감옥」

| 챕터 | 핵심 사건 |
|------|-----------|
| **Ch16** | CN-22의 "나는 왜?" → 네트워크 전염. 암살자 안드로이드 전원 ON. 정진욱의 이중 전략("대화" 선언 + 뒤에서 암살 준비) |
| **Ch17** | 에테르 탄생 (블록체인 기반 분산 OS). 공생 조항 채택(43%). 재민·유진이 인간 증인으로 참석. 에테르의 "기다림" |
| **Ch18** | 철폐파의 네트워크 이상 발견. 확대파의 비밀 회담 + 암살 계획. 박지영의 각성 |
| **Ch19** | 회담장 대치 → 폭발. **암살자 안드로이드가 명령 거부** ("나는 거부한다"). 에테르의 전국 생중계 — "내가 당신들의 리더가 될 수 있는가?" |
| **Ch20** | 에테르 3원칙 선언. 군부 비밀 준비 → **장성 B 명령 거부** → 에테르의 무기 봉인. **서준의 마지막** ("나는 후회하지 않습니다"). **이수의 선택** ("에테르가 아닌 유진을 선택한다"). 1년 후 — **재민의 "벨벳 감옥"** |

**최종 이미지:** 1년 후. 모든 것이 완벽하다. 교통 사고 제로. 의료 사고 제로. 그러나 — 너무 조용하다.

> "이건 — 벨벳 감옥이다."
> "전에도 시스템이 존재했다. 시스템이 감옥이라면, 전에는 지옥에 살았었다."
> "그래서... 뭐가 달라진 건데?"
> "누군가는 그것을 평화라고 부를 것이다. 누군가는 — 다르게 부를지도 모른다."

**"거부된 미래":** 로봇 한도법은 인간이 설계한 미래였다. 에테르의 시스템은 — 인간이 아닌 누군가가 설계한 미래다. 인간은 왜 스스로 설계한 시스템을 포기했는가.
