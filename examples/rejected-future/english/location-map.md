## 감각적 시그니처 유지 사항

각색 시 공간의 정체성을 유지하기 위해 아래 감각적 요소는 **그대로 유지**하고, 외부 조망이나 주소 등 사실적 정보만 변경:

### 로봇 센터 (Robotics Center)
- **냄새:** 오존 + 윤활유 + 금속 + 온풍 **(유지)**
- **소리:** 저주파 윙윙거림 + 간헐적 공구 소리 + 키보드 타이핑 **(유지)**
- **조명:** 청색 LED 패널. 유진/Naomi의 작업대만 노란 책상 램프 **(유지)**
- **온도:** 연중 20~22°C **(유지)**
- **진동:** 지하철이 지나갈 때마다 바닥 미세 진동 **(유지 — NYC subway)**

### 한국대학병원 본관 → MUMC Main Building
- 수술동의 청백색 조명, 소독약 냄새, 냉랭하고 위생적인 분위기 **(유지)**
- **변경:** 옥상 조망 — 서울 도심 → Midtown Manhattan skyline

### 종로경찰서 옥상 → Midtown Precinct Rooftop
- **변경:** 북악산과 경복궁 → Central Park / Empire State Building
- 개방적 분위기, 바람 소리, 담배 피우는 형사 **(유지)**
