## 1. Validation Criteria 검증

### VC①: 독자가 블룸 분류학의 **2차원적 구조**(지식 차원 × 인지 과정 차원)를 설명할 수 있는가

| 평가 | 근거 |
|------|------|
| **✅ 충족** | |

- **02장(블룸 분류학 — 2차원 프레임워크)** 에서 두 차원을 명확히 정의하고 있습니다.
  - 02-01: 2차원 구조 개괄 — "4 × 6 = 24개의 셀" 매트릭스, 학습 목표의 동사+목적어 구조
  - 02-02: 지식 차원(4가지)과 관계 — 유형별 질문, 본질, 난이도, 구체→추상 연속선
  - 02-03: 인지 과정 차원(6단계)과 관계 — 하위→상위, 상위는 하위를 포함
- 각 지식 차원 장(03~06)에서 Taxonomy Table로 두 차원의 교차를 반복 시각화
- 08장에서 Taxonomy Table 요약 포스터로 재제시

**강점:** 02장에서 프레임워크를 개괄하고 03~06장에서 각각 깊이 탐구한 후, 08장(부록)에서 다시 요약하는 '나선형 학습' 구조가 효과적입니다.

**약점:** 02-02절의 "관계: 구체→추상의 연속선" 설명이 한글 창제 사례 없이 추상적으로 제시되어, 처음 접하는 독자가 이해하기 어려울 수 있습니다.

---

### VC②: 독자가 **한글 창제 사례**의 각 요소가 분류 체계의 어느 위치에 해당하는지 말할 수 있는가

| 평가 | 근거 |
|------|------|
| **✅ 충족** | |

- 03~06장 각 장에서 지식 차원별로 한글 창제 사례를 인지 과정 6단계(기억→창안)에 매핑하여 제시
- 각 장 말미에 Taxonomy Table 요약 표로 시각화
- 부록의 Taxonomy Table 요약 포스터에서 한글 창제 사례 24셀 완성

**강점:** 단순히 "한글 창제"라는 하나의 사례를 24개 셀에 걸쳐 체계적으로 매핑한 점이 뛰어납니다. 특히 03장(사실적 지식)의 역사적 사실들에서 시작해 06장(메타인지)의 자기 인식까지 이어지는 흐름이 자연스럽습니다.

**약점:** '구조 명세 언어' 사례가 개입되면서 한글 창제 사례의 집중도가 일부 흐려집니다. 03장과 04장의 인지 과정별 사례 표에 한글 + 구조 명세 언어가 함께 제시되어, 처음 읽는 독자가 정보 과부하를 겪을 수 있습니다.

---

### VC③: 독자가 **자신의 현재 지식 수준**을 분류 체계 위에서 진단할 수 있는가

| 평가 | 근거 |
|------|------|
| **✅ 충족** | |

- 06장(메타인지 지식)에서 "연습해보기" 섹션 — "나는 03장의 사실적 지식을 어느 정도 알고 있는가?" 질문
- 06-03절의 "Taxonomy Table 위에서 자신의 지식 상태를 정확히 평가할 수 있다" 학습 목표
- 07장에서 "당신은 지금 어디에 있는가?" 자기 진단표와 "수요와 공급의 법칙" 연습을 통한 진단 방법 학습
- 07장 학습 목표 워크시트 Step 2: "현재 위치 찾기"

**강점:** 07장의 "수요와 공급의 법칙" 연습은 완전히 새로운 주제로 진단 방법을 연습하게 함으로써, 독자가 한글이 아닌 자신의 관심 분야에도 동일한 방법을 적용할 수 있도록 돕습니다.

**약점:** 06장의 "연습해보기" 섹션이 3개의 질문으로만 구성되어 있어, 독자가 충분히 연습하기에는 부족합니다. 06-03절의 내용(메타인지 학습 목표)이 이론적으로 풍부한 반면, 실제로 독자가 자신의 지식을 진단해볼 구체적인 연습이 더 필요합니다.

### VC④: 독자가 분류 체계를 활용해 **구체적인 학습 목표**를 설정할 수 있는가

| 평가 | 근거 |
|------|------|
| **✅ 충족** | |

- 07장 전체가 학습 목표 설계에 할애됨
- "수요와 공급의 법칙" A열~D열까지의 인지 과정별 질문 예시
- 4단계 워크시트: 주제 선정 → 현재 위치 → 목표 설정 → 격차 분석
- 예시 답안 제시로 독자의 이해 보조

**강점:** 워크시트가 단순한 양식이 아니라 "동사(인지 과정) + 목적어(지식 유형)" 구조를 따르도록 설계되어, 블룸 분류학의 핵심 원리를 자연스럽게 적용하게 합니다.

**약점:** 워크시트 이후 "다음 단계"에 대한 안내가 없습니다. 학습 목표를 설정한 후 이를 실행하고 점검하는 방법이 제시되지 않아, 독자가 워크시트 작성에서 멈출 위험이 있습니다(Q&A S5).

---

### VC⑤: "I know that I know nothing" 철학이 문서에 자연스럽게 녹아 있는가

| 평가 | 근거 |
|------|------|
| **✅ 충족** | |

- **01장(들어가며):** 소크라테스 인용과 "당신은 한글 창제에 대해 얼마나 알고 있나요?"라는 질문으로 도입
- **06장(메타인지 지식):** 06-01절에서 "I know that I know nothing"의 두 번의 '안다' 의미 분석 — 1차 지식 vs 2차 지식
- **08장(마치며):** 처음과 끝을 연결하며 철학적 완결 — "이 말은 결론이 아니라 시작입니다"
- 문서 전반에 걸쳐 "모르는 것을 아는 것이 힘"이라는 메시지가 반복됨

**강점:** 단순히 처음과 끝에 인용하는 것을 넘어, 06장에서 이 문장을 메타인지의 핵심 예시로 분석한 점이 특히 효과적입니다. 독자가 01장에서는 단순한 격언으로 읽었을 문장이, 06장을 지나 08장에서는 완전히 다른 의미로 다가오게 하는 구조가 잘 설계되었습니다.

**약점:** 없음. 이 기준은 완전히 충족되었습니다.
