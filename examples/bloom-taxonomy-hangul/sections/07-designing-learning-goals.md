# 07. 나의 학습 목표 설계하기

> *"당신은 지금 어디에 있고, 어디로 가고 싶은가?"*

지금까지 03~06장에서 블룸 분류학의 2차원 프레임워크를 한글 창제 사례를 통해 배웠습니다. 이제 이 프레임워크를 **당신 자신의 관심 주제**에 적용해보는 시간입니다.

**이 장을读完고 나면 당신은:**
- Taxonomy Table에서 자신의 현재 위치를 찾을 수 있습니다
- "수요와 공급의 법칙"을 예시로 24개 셀을 직접 채울 수 있습니다
- 자신의 관심 주제로 구체적인 학습 목표를 설정할 수 있습니다

## 당신은 지금 어디에 있는가?

먼저 잠시 이 문서를 다시 돌아보세요. 각 장을 읽으면서 당신은:

| 장 | 내용 | 당신의 수준은? |
|---|------|---------------|
| 03 | 사실적 지식 — 한글 창제의 기본 사실 | 대부분 알고 있었나요, 처음 듣는 내용이 많았나요? |

| 05 | 절차적 지식 — 읽고 쓰는 방법 | 직접 해볼 수 있었나요, 이론으로만 이해했나요? |
| 06 | 메타인지 지식 — 자신의 지식 진단 | "나는 어디쯤 있을까?" 생각해보게 되었나요? |

이제 당신은 블룸 분류학이라는 **거울**을 얻었습니다. 이 거울로 당신의 지식을 비춰보세요.

## 연습: "수요와 공급의 법칙"에 적용하기

한글이 아닌 전혀 다른 주제로 연습해봅시다. **"수요와 공급의 법칙"** 이라는 경제학 개념을 생각해보세요. 당신은 이 주제에 대해 얼마나 알고 있나요?

### A열: 사실적 지식 — 기본 개념

> *나는 수요와 공급의 기본 용어를 알고 있는가?*

| 질문 | 인지 과정 |
|------|-----------|
| "수요(Demand)와 공급(Supply)의 정의를 말할 수 있는가?" | A1. 기억 |
| "수요의 법칙(Law of Demand)이 의미하는 바를 설명할 수 있는가?" | A2. 이해 |
| "수요·공급 곡선을 그래프로 그릴 수 있는가?" | A3. 적용 |
| "수요 곡선의 이동(Shift)과 곡선 상의 이동(Movement)의 차이를 분석할 수 있는가?" | A4. 분석 |
| "수요·공급 모델이 현실 경제를 설명하는 데 얼마나 유효한지 평가할 수 있는가?" | A5. 평가 |
| "수요·공급 개념을 사용해 가상 시장의 시나리오를 직접 만들어낼 수 있는가?" | A6. 창안 |

### B열: 개념적 지식 — 원리와 관계

> *나는 수요와 공급이 어떻게 상호작용하는지 이해하고 있는가?*

| 질문 | 인지 과정 |
|------|-----------|
| "균형 가격(Equilibrium Price)의 정의를 말할 수 있는가?" | B1. 기억 |
| "수요와 공급이 만나 균형 가격을 형성하는 원리를 설명할 수 있는가?" | B2. 이해 |
| "균형 가격을 그래프상에서 찾을 수 있는가?" | B3. 적용 |
| "가격 통제(최고 가격제, 최저 가격제)가 시장에 미치는 영향을 분석할 수 있는가?" | B4. 분석 |
| "서로 다른 시장 구조(완전 경쟁, 독점)에서 수요·공급 모델의 적용 가능성을 평가할 수 있는가?" | B5. 평가 |
| "전혀 새로운 시장 상황에서 수요·공급 모델을 확장한 새 이론을 설계할 수 있는가?" | B6. 창안 |

### C열: 절차적 지식 — 방법과 적용

> *나는 실제 경제 문제에 수요·공급 모델을 적용할 수 있는가?*

| 질문 | 인지 과정 |
|------|-----------|
| "수요·공급 곡선을 그리는 순서를 말할 수 있는가?" | C1. 기억 |
| "곡선 이동과 곡선 상 이동의 차이점을 설명할 수 있는가?" | C2. 이해 |
| "주어진 데이터로 수요·공급 곡선을 그리고 균형점을 찾을 수 있는가?" | C3. 적용 |
| "두 재화(대체재, 보완재)의 수요 변화가 서로에게 미치는 영향을 분석할 수 있는가?" | C4. 분석 |
| "어떤 경제 상황에서 어떤 분석 방법(부분 균형 vs 일반 균형)이 적절한지 평가할 수 있는가?" | C5. 평가 |
| "기존 경제 모델의 한계를 극복하는 새로운 분석 방법론을 창안할 수 있는가?" | C6. 창안 |

### D열: 메타인지 지식 — 자신의 지식 진단

> *나는 이 주제에 대해 어디까지 알고 있고, 무엇을 모르는지 아는가?*

| 질문 | 인지 과정 |
|------|-----------|
| "수요·공급 학습에 효과적인 학습 전략(사례 연구, 그래프 연습 등)을 말할 수 있는가?" | D1. 기억 |
| "수요·공급 과제 중 어떤 것이 가장 높은 인지적 노력을 요구하는지 설명할 수 있는가?" | D2. 이해 |
| "자신의 약점(예: 그래프 해석)에 맞는 전략을 선택해 적용할 수 있는가?" | D3. 적용 |
| "이 주제에 대한 자신의 강점과 약점을 분석할 수 있는가?" | D4. 분석 |
| "자신의 현재 지식 수준을 Taxonomy Table 위에서 정확히 평가할 수 있는가?" | D5. 평가 |
| "자신의 지식 프로필에 맞춘 6개월 학습 계획을 설계할 수 있는가?" | D6. 창안 |

## 학습 목표 설정 워크시트

이제 당신의 관심 주제로 직접 해보세요. 아래 워크시트를 따라 작성해보세요.

### Step 1: 주제 선정

```
내가 배우고 싶은 주제: ________________________________
```

### Step 2: 현재 위치 찾기

```
지금 나는 이 주제에 대해:

사실적(A): □ 기억  □ 이해  □ 적용  □ 분석  □ 평가  □ 창안
개념적(B): □ 기억  □ 이해  □ 적용  □ 분석  □ 평가  □ 창안
절차적(C): □ 기억  □ 이해  □ 적용  □ 분석  □ 평가  □ 창안
메타인지(D): □ 기억  □ 이해  □ 적용  □ 분석  □ 평가  □ 창안
```

### Step 3: 목표 설정

```
나는 "______________________________"을/를
"____________________________________" 할 수 있게 되고 싶다.
                                      (인지 과정 동사)

이 목표는 Taxonomy Table의 (   )열 (   )행에 해당한다.
                   (A/B/C/D)  (1~6)
```

### Step 4: 격차 분석

```
현재 위치와 목표 위치의 차이: ________________________________
이 차이를 극복하기 위한 계획:
1. ________________________________________________________
2. ________________________________________________________
3. ________________________________________________________
```

### 워크시트 예시 답안

> **주제:** 수요와 공급의 법칙
>
> **현재 위치:** 나는 경제학을 전공하지 않았다. 기본 용어는 기억 수준(A1)이고, 곡선을 그릴 수는 있다(C3). 하지만 원리 설명(B2)이나 시장 분석(B4)은 어렵다.
>
> **목표:** 나는 **"수요와 공급의 균형 가격 형성 원리"** 를 **설명(B2)할 수 있게** 되고 싶다. 이 목표는 **B열 2행**에 해당한다. 지금 나는 대략 A1 수준이므로, 먼저 핵심 개념을 B2 수준으로 올리는 것부터 시작하겠다.
>
> **계획:** 1. 교과서 3장 읽고 개념 정리 (B1로) → 2. 그래프를 직접 그리고 원리를 글로 설명 (B2로) → 3. 스스로에게 설명해보고 막히는 부분 확인 (D2 전략)
