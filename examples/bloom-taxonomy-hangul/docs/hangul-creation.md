# Research: 한글 창제 (Hangul Creation)

## 개요

한글(훈민정음)은 조선 제4대 왕 세종대왕이 1443년(세종 25년)에 창제하고 1446년(세종 28년)에 반포한 문자.

---

## 주요 연표

| 연도 | 사건 |
|------|------|
| 1443년(세종 25) 12월 | 세종, 언문 28자 창제. 《세종실록》에 "이번 달에 왕이 언문 28자를 만들었다"는 기록. |
| 1445년(세종 27) | 《용비어천가》 편찬 — 훈민정음으로 기록된 최초의 문헌. |
| 1446년(세종 28) 9월 | 《훈민정음 해례본》 완성·반포. 정인지·신숙주·성삼문·최항·박팽년·강희안·이개·이선로 등 8명의 집현전 학사들이 저술. |
| 1940년 | 《훈민정음 해례본》 원본 발견 (간송미술관 소장, 국보 제70호, 유네스코 세계기록유산) |

---

## 창제 동기

**세종의 어제 서문** (훈민정음 해례본):

> "나라의 말이 중국과 달라 문자와 서로 통하지 않으므로, 이런 까닭으로 어리석은 백성이 말하고자 하는 바가 있어도 끝내 뜻을 펴지 못하는 자가 많다. 내가 이를 가엾게 여겨 새로 스물여덟 글자를 만드니, 사람마다 하여금 쉬워서 편안히 쓰게 하고자 할 따름이다."

즉:
- 백성들이 한자(중국 문자)를 몰라서 자신의 뜻을 표현하지 못하는 문제 해결
- 누구나 쉽게 배우고 쓸 수 있는 실용적 문자 창제가 목적

---

## 제자 원리 (글자를 만든 원리)

### 1. 상형(象形) — 소리 기관을 본뜸
- **초성(자음):** 발음 기관의 모양을 본떠 글자 모양 결정
  - `ㄱ` — 혀뿌리가 목구멍을 막는 모양
  - `ㄴ` — 혀가 윗잇몸에 닿는 모양
  - `ㅁ` — 입 모양
  - `ㅅ` — 이 모양
  - `ㅇ` — 목구멍 모양
- **중성(모음):** 천·지·인(天·地·人)의 형상
  - `•` (아래아) — 하늘(천)의 둥근 모양
  - `ㅡ` — 땅(지)의 평평한 모양
  - `ㅣ` — 사람(인)의 서 있는 모양

### 2. 가획(加劃) — 기본 글자에 획을 더해 새로운 글자 생성
- 기본자에 획을 더하면 소리가 더 거세짐
- 예: `ㄱ` → `ㅋ` (획 추가), `ㄴ` → `ㄷ` → `ㅌ`

### 3. 합자(合字) — 초성·중성·종성의 결합
- 음절 단위로 초성 + 중성 + (종성)이 모여 하나의 글자 구성
- 글자 모양이 네모꼴(방괘형)로 모아짐

### 4. 고전(古篆) 모방
- 글자의 전체적인 외형(방괘형, 모아쓰기 방식)은 고전(옛 전자)을 모방

---

## 이론적 기초

- **성리학(주자학)** — 태극·음양·오행의 우주론적 원리를 문자 창제에 적용
- **음성학** — 당시 동아시아의 음운학(운서) 지식을 바탕으로 한글의 음소 체계 설계
- 발성 기관의 과학적 관찰과 분석

---

## 훈민정음의 특징

| 특징 | 설명 |
|------|------|
| **표음문자** | 소리 나는 대로 적을 수 있는 문자 |
| **과학성** | 발음 기관의 형태를 본뜬 자형, 음양오행 이론에 기반 |
| **간편성** | 28자의 기본자로 모든 한국어 소리 표현 가능 |
| **확장성** | 한자음, 외래어, 자연음까지 표기 가능 |
| **독창성** | 기존 문자를 단순 모방한 것이 아니라 체계적으로 창안 |

---

## 관련 인물

| 인물 | 역할 |
|------|------|
| **세종대왕** | 문자 창제, 예의(例義) 집필 |
| **정인지** | 서문 작성, 해례본 총괄 |
| **신숙주, 성삼문, 최항, 박팽년, 강희안, 이개, 이선로** | 해례본 집필, 이론 정립, 《동국정운》 편찬 등 |
| **최만리** | 훈민정음 창제 반대 상소 — "언문" 창제에 반대했으나 세종의 의지가 더 강함 |
