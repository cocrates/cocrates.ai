# Cocrates에 오신 것을 환영합니다

Cocrates는 생각을 정리하고 기록하는 문서 사이트입니다.

## 시작하기

이 사이트는 [Docusaurus](https://docusaurus.io/)로 만들어졌으며, GitHub Pages에 배포됩니다.

문서는 `docs/` 폴더에 Markdown 또는 MDX 파일로 추가할 수 있습니다.

```bash
# 로컬 개발 서버 실행
npm start

# 프로덕션 빌드
npm run build
```

## 다음 단계

- `docs/`에 새 문서 파일을 추가하세요.
- `docusaurus.config.ts`에서 사이트 제목, 네비게이션 등을 수정하세요.
- `main` 브랜치에 push하면 GitHub Actions가 자동으로 배포합니다.
