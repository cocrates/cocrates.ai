# EP3. Menginstal Cocrates Harness

## 🛠️ 'Pelatih AI' Anda dalam 5 Menit

Selama dua episode kita telah memahami bahwa AI yang sama menghasilkan hasil berbeda tergantung sikap—dan menjaga kendali berarti menghormati **"Kode yang tidak diperiksa tidak layak dihasilkan."**

Jadi bagaimana Anda benar-benar menjalankan **Cocrates Harness**, alat yang menempatkan filsafat Sokratik itu di mesin Anda?

Kedengarannya rumit? Tidak. Investasikan lima menit—kami akan memandu Anda melalui instalasi sederhana. Mata terbuka, ikuti! 🚀

---

## 🧩 Cocrates Harness Adalah Plugin

Pertama: Cocrates Harness bukan aplikasi mandiri. Ia hidup di **opencode** sebagai **plugin**.

* **opencode:** Platform yang menjalankan dan mengelola AI agent
* **@cocrates/cocrates-harness:** Plugin yang membawa Agent dan Skills Cocrates ke panggung itu

Arsitektur yang jelas. Instalasi terbagi menjadi tiga langkah: **instal opencode → tambahkan plugin → salin file skill**. Selesai.

---

## 🛠️ Langkah 1. Instal opencode (Pilih Gaya Anda)

opencode menawarkan tiga cara menjalankan. Pilih yang cocok untuk Anda.

| Metode instalasi | Terbaik untuk | Catatan |
| --- | --- | --- |
| **opencode terminal** | Penggemar TUI Claude/GPT di terminal | UI teks berbasis keyboard |
| **opencode desktop** | Pengguna nyaman Cursor / VS Code | Editor GUI bersih (saat ini beta) |
| **opencode extension** | Tidak bisa meninggalkan IDE yang ada | Tambahkan ekstensi ke Cursor atau VS Code |

👉 Unduhan khusus OS: [opencode.ai/download](https://opencode.ai/download)

> **💡 Tips dari senior!**
>
> Jika lingkungan terminal (TUI) terasa asing, jangan ragu—gunakan **ekstensi di VS Code atau Cursor**. Terutama dengan **ekstensi OpenChamber**, Anda mendapat GUI cantik seperti Copilot di VS Code atau Cursor—cara termudah untuk memulai.
>
> 1. Instal [opencode terminal](https://opencode.ai/download).
> 2. Instal [VS Code](https://code.visualstudio.com).
> 3. Instal ekstensi OpenChamber (lihat gambar di bawah).
>
> ![OpenChamber Extension](/img/openchamber-extension.png)
>

---

## ⚙️ Langkah 2. Tambahkan Plugin cocrates-harness

Panggung siap—saatnya menginstal Cocrates. Satu edit kecil pada konfigurasi opencode.

Cari di `~/.config/opencode/` file `opencode.jsonc`. (Jika Anda mengatur `OPENCODE_CONFIG_DIR`, gunakan path itu.)

Buka file dan tambahkan cocrates di `plugin`:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "plugin": ["@cocrates/cocrates-harness"]
}

```

Dua langkah selesai. Plugin terhubung.

---

## 📚 Langkah 3. Salin File 'Skill'

Plugin saja tidak cukup—kekuatan sebenarnya Cocrates ada di **Skills**: definisi alur kerja untuk pendidikan, generasi berbasis arsitektur, dan lainnya.

File skill ada di folder `skills/` dari [repositori GitHub Cocrates](https://github.com/cocrates/cocrates.ai/tree/main/skills). Salin semuanya ke `~/.config/opencode/skills/` di mesin Anda. Setup selesai.

### 😎 "Terlalu merepotkan—bisa minta AI yang melakukannya?"

**Tentu saja. Anda adalah direktur.** Setelah menginstal opencode, ucapkan satu kalimat tegas ke jendela eksekusi:

> 👤 **"Instal Cocrates Harness menggunakan [https://cocrates.ai/install.md](pathname:///install.md) sebagai panduan."**
> 
> ![Cocrates Harness Installed](/img/cocrates-harness-installed.png)
>

Agent yang mampu dapat mengikuti panduan—konfigurasi plugin, unduhan skill, salin—dan menanganinya untuk Anda. Tidak perlu menjelajahi GitHub secara manual. Zaman yang baik.
