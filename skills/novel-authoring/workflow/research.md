# Research — Independent Activity

**Not a fixed stage.** No gate required.

Can be initiated at any point — planning, architecture, chapter design, or revision.

| When | Example |
|------|---------|
| Plan | *"Research 14th-century Korean court attire."* |
| Architecture | *"Research arsenic poisoning for mystery plot."* |
| Chapter design | *"What does a Mongolian steppe look like in winter?"* |
| Revision | *"Check if this sword technique is realistic."* |

Save under `novels/{title-slug}/docs/{topic-slug}.md`:

```markdown
# Research: {Topic}

## Summary
{Key findings}

## Sources
{Attribution}

## Application Notes
{How this informs the novel}
```

Research is informational — feed findings into the appropriate stage artifact when relevant.
